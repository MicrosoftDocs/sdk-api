# IDWriteFontFace4::ReleaseGlyphImageData


## -description


Releases the table data obtained from ReadGlyphData.


## -parameters




### -param glyphDataContext

Type: <b>void*</b>

Opaque context from ReadGlyphData.


## -returns



This method does not return a value.
