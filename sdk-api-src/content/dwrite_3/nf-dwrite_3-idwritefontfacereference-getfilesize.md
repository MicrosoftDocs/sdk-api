# IDWriteFontFaceReference::GetFileSize


## -description


Get the total size of the font face in bytes.  


## -parameters






## -returns



Type: <b>UINT64</b>

Returns the total size of the font face in bytes. If the locality is remote, this value is unknown and will be zero.
