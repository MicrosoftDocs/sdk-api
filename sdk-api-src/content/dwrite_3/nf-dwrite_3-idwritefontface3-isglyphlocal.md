# IDWriteFontFace3::IsGlyphLocal


## -description


Determines whether the glyph is locally downloaded from the font.


## -parameters




### -param glyphId [in]

Type: <b>UINT16</b>

Glyph identifier.


## -returns



Type: <b>BOOL</b>

Returns TRUE if the font has the specified glyph locally available.
