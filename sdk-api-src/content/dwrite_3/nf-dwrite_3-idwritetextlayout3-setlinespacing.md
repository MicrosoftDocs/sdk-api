# IDWriteTextLayout3::SetLineSpacing


## -description

Set line spacing.

## -parameters

### -param lineSpacingOptions [in]

How to manage space between lines.

## -returns

If this method succeeds, it returns <b>S_OK</b>. Otherwise, it returns an <b>HRESULT</b> error code.
