## -description

Represents an absolute reference to a font face. This interface contains font face type, appropriate file references, and face identification data.

This interface extends [IDWriteFontFace3](./nn-dwrite_3-idwritefontface3.md). Various font data such as metrics, names, and glyph outlines are obtained from **IDWriteFontFace**.
