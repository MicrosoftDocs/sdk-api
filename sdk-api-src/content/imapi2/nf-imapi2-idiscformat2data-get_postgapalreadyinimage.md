# IDiscFormat2Data::get_PostgapAlreadyInImage


## -description


Determines if the data stream contains post-writing gaps.


## -parameters




### -param value [out]

Is VARIANT_TRUE if the data stream contains post-writing gaps; otherwise, VARIANT_FALSE.
