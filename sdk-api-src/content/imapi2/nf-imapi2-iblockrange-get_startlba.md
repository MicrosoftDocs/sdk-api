# IBlockRange::get_StartLba


## -description


Retrieves the start sector of the range described by <a href="https://docs.microsoft.com/windows/desktop/api/imapi2/nn-imapi2-iblockrange">IBlockRange</a>.


## -parameters




### -param value [out]

The start sector of the range.


## -returns



S_OK is returned on success, but other success codes may be returned as a result of implementation. The following error codes are commonly returned on operation failure, but do not represent the only possible error values:

<th>Description</th>

<dt>Value: 0x80004003</dt>
</dl>
</td>
<td width="60%">

</td>
</tr>
</table>
