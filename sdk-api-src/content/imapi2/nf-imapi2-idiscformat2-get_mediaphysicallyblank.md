# IDiscFormat2::get_MediaPhysicallyBlank


## -description


Determines if the current media is reported as physically blank by the drive.


## -parameters




### -param value [out]

Is VARIANT_TRUE if the disc is physically blank; otherwise, VARIANT_FALSE.
