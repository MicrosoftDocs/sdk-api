# UiaRect structure


## -description


Contains the position and size of a rectangle, in screen coordinates.


## -struct-fields




### -field left

Type: <b>double</b>

Position of the left side.


### -field top

Type: <b>double</b>

Position of the top side.


### -field width

Type: <b>double</b>

Width.


### -field height

Type: <b>double</b>

Height.
