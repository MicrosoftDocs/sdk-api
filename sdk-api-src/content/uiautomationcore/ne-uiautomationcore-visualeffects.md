# VisualEffects enumeration


## -description

Shadow effect


### -field VisualEffects_Reflection

Reflection effect


### -field VisualEffects_Glow

Glow effect


### -field VisualEffects_SoftEdges

Soft edges effect


### -field VisualEffects_Bevel
