# DrtClose function


## -description


The <b>DrtClose</b> function closes the local instance of the DRT. 


## -parameters




### -param hDrt [in]

Handle to the DRT instance.


## -returns



This function returns S_OK on success. Other possible values include:

<table>
<tr>
<th>Return code</th>
<th>Description</th>
</tr>
<tr>
<td width="40%">
<dl>
<dt><b>E_HANDLE</b></dt>
</dl>
</td>
<td width="60%">
The <i>hDrt</i> handle is invalid.

</td>
</tr>
</table>
