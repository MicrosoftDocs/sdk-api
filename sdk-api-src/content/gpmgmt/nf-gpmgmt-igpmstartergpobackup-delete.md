# IGPMStarterGPOBackup::Delete


## -description


Removes the Starter GPO backup from the backup directory, and from the file system.


## -parameters






## -returns



<h3>JScript</h3>
Returns <b>S_OK</b> if successful. Returns a failure code if an error occurs.

<h3>VB</h3>
Returns <b>S_OK</b> if successful. Returns a failure code if an error occurs.
