# IGPMGPOCollection::get_Item


## -description

Given an index, returns a pointer to an <a href="/previous-versions/windows/desktop/api/gpmgmt/nn-gpmgmt-igpmgpo">GPMGPO</a> object from the collection.

This property is read-only.
