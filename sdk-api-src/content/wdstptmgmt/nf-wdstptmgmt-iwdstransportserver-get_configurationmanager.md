# IWdsTransportServer::get_ConfigurationManager


## -description

Returns a pointer to the object of an <a href="/windows/desktop/api/wdstptmgmt/nn-wdstptmgmt-iwdstransportconfigurationmanager">IWdsTransportConfigurationManager</a> interface used to manage the configuration of this server.

This property is read-only.
