# IVssFileShareSnapshotProvider::BeginPrepareSnapshot


## -description


VSS calls this method for each shadow copy that is added to the shadow copy set.


## -parameters




### -param SnapshotSetId [in]

Shadow copy set identifier.


### -param SnapshotId [in]

Identifier of the shadow copy to be created.


### -param pwszSharePath [in]

The file share path.


### -param lNewContext [in]

The context for the shadow copy set. This context consists of a bitmask of <a href="https://docs.microsoft.com/windows/desktop/api/vss/ne-vss-vss_volume_snapshot_attributes">_VSS_VOLUME_SNAPSHOT_ATTRIBUTES</a> values.


### -param ProviderId [in]

The provider ID.
