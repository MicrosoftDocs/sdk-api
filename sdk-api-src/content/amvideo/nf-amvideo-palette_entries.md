# PALETTE_ENTRIES macro


## -description


The <b>PALETTE_ENTRIES</b> macro retrieves the maximum number of colors in the palette of a specified bitmap.


## -parameters




### -param pbmi

Pointer to a <a href="https://docs.microsoft.com/windows/desktop/api/amvideo/ns-amvideo-tagvideoinfoheader">VIDEOINFOHEADER</a> structure.
