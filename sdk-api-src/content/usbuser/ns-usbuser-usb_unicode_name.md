# USB_UNICODE_NAME structure


## -description

The <b>USB_UNICODE_NAME</b> structure contains a Unicode string that specifies a symbolic link name.

## -struct-fields

### -field Length

The length, in bytes, of the string.

### -field String

A pointer to the string.
