# IFsrmFileConditionProperty::get_Operator


## -description


<p class="CCE_Message">[This property is supported for compatibility but it's recommended to use the 
    <a href="https://msdn.microsoft.com/1CE772FA-CE33-4900-A499-058175A7C37E">FSRM WMI Classes</a> to manage FSRM. Please see the 
    <a href="https://msdn.microsoft.com/deb5be80-4255-4d56-8db5-31239e2872ef">MSFT_FSRMFMJCondition</a> class.]

Specifies the comparison operator, as enumerated by the 
    <a href="https://msdn.microsoft.com/b3b5caab-4d70-4f85-bf53-0344192d3674">FsrmPropertyConditionType</a> 
    enumeration.

This property is read/write.
