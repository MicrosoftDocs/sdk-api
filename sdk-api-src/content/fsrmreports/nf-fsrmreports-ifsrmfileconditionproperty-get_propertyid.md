# IFsrmFileConditionProperty::get_PropertyId


## -description

<p class="CCE_Message">[This property is supported for compatibility but it's recommended to use the 
    <a href="/previous-versions/windows/desktop/fsrm/fsrm-wmi-classes">FSRM WMI Classes</a> to manage FSRM. Please see the 
    <a href="/previous-versions/windows/desktop/fsrm/msft-fsrmfmjcondition">MSFT_FSRMFMJCondition</a> class.]

Specifies the predefined file property, as enumerated by the 
    <a href="/windows/desktop/api/fsrmenums/ne-fsrmenums-fsrmfilesystempropertyid">FsrmFileSystemPropertyId</a> enumeration.

This property is read/write.
