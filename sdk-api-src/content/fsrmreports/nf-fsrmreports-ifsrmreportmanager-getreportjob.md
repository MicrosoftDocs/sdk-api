# IFsrmReportManager::GetReportJob


## -description


Retrieves the specified report job.


## -parameters




### -param taskName [in]

The task name that identifies the report job to retrieve. The string is limited to 230 characters.


### -param reportJob [out]

An <a href="https://docs.microsoft.com/previous-versions/windows/desktop/api/fsrmreports/nn-fsrmreports-ifsrmreportjob">IFsrmReportJob</a> interface to the retrieved job.


## -returns



The method returns the following return values.
