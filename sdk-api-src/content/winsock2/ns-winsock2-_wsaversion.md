# _WSAVersion structure


## -description


The 
<b>WSAVERSION</b> structure provides version comparison in Windows Sockets.


## -struct-fields




### -field dwVersion

Version of Windows Sockets.


### -field ecHow


<a href="https://msdn.microsoft.com/a1de171e-42d7-4d57-b241-1db9989dbd8e">WSAECOMPARATOR</a> enumeration, used in the comparison.
