# IEnroll2::SetHStoreROOT


## -description


<p class="CCE_Message">[This method is no longer available for use as of Windows Server 2008 and Windows Vista.]

The <b>SetHStoreROOT</b> method specifies the handle to use for the Root store. This method was first defined in the <a href="https://docs.microsoft.com/windows/desktop/api/xenroll/nn-xenroll-ienroll2">IEnroll2</a> interface.


## -parameters




### -param hStore [in]

Certificate store handle to use for the Root store.


## -returns



The return value is an <b>HRESULT</b>. A value of S_OK indicates success.
