# DXGI_OVERLAY_SUPPORT_FLAG enumeration


## -description

Specifies overlay support to check for in a call to <a href="/windows/desktop/api/dxgi1_3/nf-dxgi1_3-idxgioutput3-checkoverlaysupport">IDXGIOutput3::CheckOverlaySupport</a>.

## -enum-fields

### -field DXGI_OVERLAY_SUPPORT_FLAG_DIRECT:0x1

Direct overlay support.

### -field DXGI_OVERLAY_SUPPORT_FLAG_SCALING:0x2

Scaling overlay support.
