# ID3D10Device1 interface


## -description

The device interface represents a virtual adapter for Direct3D 10.1; it is used to perform rendering and create Direct3D resources.

## -inheritance

The <b>ID3D10Device1</b> interface inherits from <a href="/windows/desktop/api/d3d10/nn-d3d10-id3d10device">ID3D10Device</a>. <b>ID3D10Device1</b> also has these types of members:

## -remarks

A device is created using <a href="/windows/desktop/api/d3d10_1/nf-d3d10_1-d3d10createdevice1">D3D10CreateDevice1</a>.

This method requires Windows Vista Service Pack 1.
