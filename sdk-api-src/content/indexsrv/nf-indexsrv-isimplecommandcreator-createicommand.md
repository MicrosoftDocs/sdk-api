# ISimpleCommandCreator::CreateICommand


## -description


Creates an ICommand.


## -parameters




### -param ppIUnknown

Returns the IUnknown for the command.


### -param pOuterUnk

Optional outer unknown pointer.


## -returns



If this method succeeds, it returns <b>S_OK</b>. Otherwise, it returns an <b>HRESULT</b> error code.
