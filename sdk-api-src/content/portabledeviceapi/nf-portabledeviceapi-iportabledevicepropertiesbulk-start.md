# IPortableDevicePropertiesBulk::Start


## -description



The <b>Start</b> method starts a queued operation.




## -parameters




### -param pContext [in]

A pointer to a GUID that identifies the operation to start. This value is generated by a <b>Queue...</b> method of this interface.


## -returns



The method returns an <b>HRESULT</b>. Possible values include, but are not limited to, those in the following table.

<table>
<tr>
<th>Return code</th>
<th>Description</th>
</tr>
<tr>
<td width="40%">
<dl>
<dt><b>S_OK</b></dt>
</dl>
</td>
<td width="60%">
The method succeeded.

</td>
</tr>
<tr>
<td width="40%">
<dl>
<dt><b>HRESULT_FROM_WIN32(ERROR_NOT_FOUND)</b></dt>
</dl>
</td>
<td width="60%">
The GUID passed to <i>pContext</i> does not match a queued operation.

</td>
</tr>
</table>
