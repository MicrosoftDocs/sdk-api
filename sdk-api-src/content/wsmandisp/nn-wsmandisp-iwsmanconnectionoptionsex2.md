# IWSManConnectionOptionsEx2 interface


## -description

The <b>IWSManConnectionOptionsEx2</b> object is passed to the <a href="/windows/desktop/api/wsmandisp/nf-wsmandisp-iwsman-createsession">IWSMan::CreateSession</a> method to provide the authentication mechanism, access type, and credentials to connect to a proxy server.
