# IkeextSaDeleteById0 function


## -description


The <b>IkeextSaDeleteById0</b> function removes a security association (SA) from the database.


## -parameters




### -param engineHandle [in]

Type: <b>HANDLE</b>

Handle for an open session to the filter engine generated by a previous  call to  <a href="https://msdn.microsoft.com/5165f219-f3e0-4e84-915b-75912aab02b7">FwpmEngineOpen0</a>.


### -param id [in]

Type: <b>UINT64</b>

The SA identifier.
