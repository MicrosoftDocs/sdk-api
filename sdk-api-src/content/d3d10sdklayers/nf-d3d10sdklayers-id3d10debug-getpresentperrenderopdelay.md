# ID3D10Debug::GetPresentPerRenderOpDelay


## -description


Get the number of milliseconds to sleep after <a href="https://msdn.microsoft.com/library/Bb174576(v=VS.85).aspx">Present</a> is called.


## -parameters






## -returns



Type: <b><a href="https://msdn.microsoft.com/4553cafc-450e-4493-a4d4-cb6e2f274d46">UINT</a></b>

Number of milliseconds to sleep after Present is called.




## -remarks



Value is set with <a href="https://msdn.microsoft.com/library/Bb173521(v=VS.85).aspx">ID3D10Debug::SetPresentPerRenderOpDelay</a>.
