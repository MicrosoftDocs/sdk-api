# IMediaRenderer::streaming


## -description


Instructs the DMR asynchronously to either mute or unmute the audio.




## -parameters




### -param mute [in]

A reference to a boolean value that specifies the mute setting. A value of True specifies that mute is turned on, and a value of False specifies that it is turned off.


### -param value [out]

Receives a reference to a <a href="https://docs.microsoft.com/windows/desktop/mediastreaming/playbackoperation">PlaybackOperation</a> object that is used to get results from the asynchronous operation.
