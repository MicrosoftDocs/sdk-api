# D2D1_HUEROTATION_PROP enumeration


## -description

Identifiers for properties of the <a href="/windows/desktop/Direct2D/hue-rotate">Hue rotate effect</a>.

## -enum-fields

### -field D2D1_HUEROTATION_PROP_ANGLE:0

The angle to rotate the hue, in degrees.
          

The type is FLOAT.

The default is 0.0f.

### -field D2D1_HUEROTATION_PROP_FORCE_DWORD:0xffffffff
