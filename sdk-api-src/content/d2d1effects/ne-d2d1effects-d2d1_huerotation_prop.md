# D2D1_HUEROTATION_PROP enumeration


## -description


Identifiers for properties of the <a href="https://msdn.microsoft.com/en-us/library/Hh706342(v=VS.85).aspx">Hue rotate effect</a>.


## -enum-fields




### -field D2D1_HUEROTATION_PROP_ANGLE

The angle to rotate the hue, in degrees.
          

The type is FLOAT.

The default is 0.0f.
