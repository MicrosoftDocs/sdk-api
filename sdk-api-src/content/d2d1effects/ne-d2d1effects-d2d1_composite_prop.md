# D2D1_COMPOSITE_PROP enumeration


## -description

Identifiers for properties of the <a href="/windows/desktop/Direct2D/composite">Composite effect</a>.

## -enum-fields

### -field D2D1_COMPOSITE_PROP_MODE:0

The mode used for the effect.
            

Type is D2D1_COMPOSITE_MODE.

Default value is D2D1_COMPOSITE_MODE_SOURCE_OVER

### -field D2D1_COMPOSITE_PROP_FORCE_DWORD:0xffffffff
