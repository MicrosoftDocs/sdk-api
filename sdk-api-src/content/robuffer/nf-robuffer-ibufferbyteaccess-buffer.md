# IBufferByteAccess::Buffer


## -description

Gets the array of bytes in the buffer.

## -parameters

### -param value [out]

The byte array.

## -returns

If this method succeeds, it returns <b>S_OK</b>. Otherwise, it returns an <b>HRESULT</b> error code.
