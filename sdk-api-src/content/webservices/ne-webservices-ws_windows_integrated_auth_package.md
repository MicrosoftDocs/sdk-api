# WS_WINDOWS_INTEGRATED_AUTH_PACKAGE enumeration


## -description

Defines the specific SSP package to be used for Windows Integrated
Authentication.

## -enum-fields

### -field WS_WINDOWS_INTEGRATED_AUTH_PACKAGE_KERBEROS:1

The Kerberos package.

### -field WS_WINDOWS_INTEGRATED_AUTH_PACKAGE_NTLM:2

The NTLM package.

### -field WS_WINDOWS_INTEGRATED_AUTH_PACKAGE_SPNEGO:3

The SPNEGO package.
