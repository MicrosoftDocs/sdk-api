# WS_XML_DATETIME_TEXT structure


## -description

Represents a datetime formatted as an 
        xsd:dateTime.
      

Negative datetime values are not supported.

## -struct-fields

### -field text

The base type for all types that derive from <a href="/windows/desktop/api/webservices/ns-webservices-ws_xml_text">WS_XML_TEXT</a>.

### -field value
