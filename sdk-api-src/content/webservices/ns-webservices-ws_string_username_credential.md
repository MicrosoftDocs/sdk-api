# WS_STRING_USERNAME_CREDENTIAL structure


## -description


The type for supplying a username/password pair as strings.
            


## -struct-fields




### -field credential

The base type from which this type and all other username credential types derive.
                


### -field username

The username.
                


### -field password

The password.
