# WS_DOUBLE_DESCRIPTION structure


## -description


An optional type description used with <a href="https://docs.microsoft.com/windows/desktop/api/webservices/ne-webservices-ws_type">WS_DOUBLE_TYPE</a>.
                It is used to specify constraints on the set of values
                which can be deserialized.
            


## -struct-fields




### -field minValue

The minimum value.
                


### -field maxValue

The maximum value.
