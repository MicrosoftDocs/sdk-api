# WS_SERVICE_CANCEL_REASON enumeration


## -description


The reasons for a cancellation.
            


## -enum-fields




### -field WS_SERVICE_HOST_ABORT

A service host is going through an abort.
                


### -field WS_SERVICE_CHANNEL_FAULTED

The underlying channel faulted.
