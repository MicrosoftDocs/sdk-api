# WS_XML_TEXT_NODE structure


## -description

Represents an element, attribute, or CDATA content.

## -struct-fields

### -field node

The base type for all types that derive from <a href="/windows/desktop/api/webservices/ns-webservices-ws_xml_node">WS_XML_NODE</a>.

### -field text

The text content.
