# WS_XML_BASE64_TEXT structure


## -description


Represents base64 encoded data. (for example, the three bytes { 0, 0, 0 } represent the text "AAAA".)
      


## -struct-fields




### -field text

The base type for all types that derive from <a href="https://docs.microsoft.com/windows/desktop/api/webservices/ns-webservices-ws_xml_text">WS_XML_TEXT</a>.
        


### -field bytes

The bytes of data.
        


### -field length

The length of the bytes of data.
