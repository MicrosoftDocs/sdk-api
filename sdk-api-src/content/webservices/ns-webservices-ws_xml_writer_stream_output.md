# WS_XML_WRITER_STREAM_OUTPUT structure


## -description

Specifies that the generated bytes should be sent to callback.

## -struct-fields

### -field output

The base type for all types that derive from <a href="/windows/desktop/api/webservices/ns-webservices-ws_xml_writer_output">WS_XML_WRITER_OUTPUT</a>.

### -field writeCallback

A callback that will get invoked when <a href="/windows/desktop/api/webservices/nf-webservices-wsflushwriter">WsFlushWriter</a> is called.  This may not be <b>NULL</b>.

### -field writeCallbackState

User-defined data that will be passed to the writeCallback.
