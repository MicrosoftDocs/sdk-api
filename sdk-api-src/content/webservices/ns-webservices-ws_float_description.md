# WS_FLOAT_DESCRIPTION structure


## -description

An optional type description used with <a href="/windows/desktop/api/webservices/ne-webservices-ws_type">WS_FLOAT_TYPE</a> to specify constraints on the set of values
                which can be deserialized.

## -struct-fields

### -field minValue

The minimum value.

### -field maxValue

The maximum value.
