# WS_PROXY_PROPERTY structure


## -description

Specifies a proxy property.

## -struct-fields

### -field id

Identifies the <a href="/windows/desktop/api/webservices/ne-webservices-ws_proxy_property_id">WS_PROXY_PROPERTY_ID</a>.

### -field value

The buffer for the property value.

### -field valueSize

The size of buffer, in bytes.
