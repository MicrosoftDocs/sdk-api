# WS_XML_UNIQUE_ID_TEXT structure


## -description


Represents a GUID formatted as the text "urn:uuid:xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx".
      


## -struct-fields




### -field text

The base type for all types that derive from <a href="https://docs.microsoft.com/windows/desktop/api/webservices/ns-webservices-ws_xml_text">WS_XML_TEXT</a>.
        


### -field value

The GUID value.
