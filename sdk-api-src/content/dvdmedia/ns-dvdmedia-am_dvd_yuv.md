# AM_DVD_YUV structure


## -description



Contains DVD YUV subpicture data.




## -struct-fields




### -field Reserved

Reserved.


### -field Y

Y color data.


### -field U

U color data.


### -field V

V color data.


## -remarks



This structure is contained within the <a href="https://docs.microsoft.com/windows/desktop/api/dvdmedia/ns-dvdmedia-_am_property_sppal">AM_PROPERTY_SPPAL</a> structure.
