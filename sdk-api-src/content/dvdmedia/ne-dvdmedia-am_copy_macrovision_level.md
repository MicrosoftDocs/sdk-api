# AM_COPY_MACROVISION_LEVEL enumeration


## -description



Identifies the analog copy protection level.




## -enum-fields




### -field AM_MACROVISION_DISABLED

Disabled. 


### -field AM_MACROVISION_LEVEL1

Level 1.


### -field AM_MACROVISION_LEVEL2

Level 2.


### -field AM_MACROVISION_LEVEL3

Level 3.
