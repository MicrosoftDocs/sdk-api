# EventDescGetKeyword function


## -description

Retrieves
		
		
	
	the keyword from the event descriptor.

## -parameters

### -param EventDescriptor [in]

Event descriptor from which to retrieve the keyword. See <a href="/windows/desktop/api/evntprov/ns-evntprov-event_descriptor">EVENT_DESCRIPTOR</a>.

## -returns

Keyword that is a bitmask that further defines the category of events to which the event belongs.

## -remarks

This is a convenience macro for retrieving the member of the <a href="/windows/desktop/api/evntprov/ns-evntprov-event_descriptor">EVENT_DESCRIPTOR</a> structure.
