# EventDescGetLevel function


## -description


Retrieves
		
		
	
	the severity level from the event descriptor.


## -parameters




### -param EventDescriptor [in]

Event descriptor from which to retrieve the severity level. See <a href="https://docs.microsoft.com/windows/desktop/api/evntprov/ns-evntprov-event_descriptor">EVENT_DESCRIPTOR</a>.


## -returns



Severity level that indicates the verboseness with which to log the event.




## -remarks



This is a convenience macro for retrieving the member of the <a href="https://docs.microsoft.com/windows/desktop/api/evntprov/ns-evntprov-event_descriptor">EVENT_DESCRIPTOR</a> structure.
