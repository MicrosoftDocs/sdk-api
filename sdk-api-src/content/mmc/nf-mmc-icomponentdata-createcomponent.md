# IComponentData::CreateComponent


## -description


The <b>IComponentData::CreateComponent</b> method creates an instance of the 
<a href="https://msdn.microsoft.com/65eaa5ef-182b-4fec-bb3d-a308ac9dc660">IComponent</a> that will be associated with this 
<a href="https://msdn.microsoft.com/60900b8d-59cc-4c1d-86b7-b902ba89216d">IComponentData</a> interface.


## -parameters




### -param ppComponent [out]

A pointer to the location that stores the newly created pointer to 
<a href="https://msdn.microsoft.com/65eaa5ef-182b-4fec-bb3d-a308ac9dc660">IComponent</a>.


## -returns



This method can return one of these values.
