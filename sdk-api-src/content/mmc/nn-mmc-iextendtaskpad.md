# IExtendTaskPad interface


## -description

The 
<b>IExtendTaskPad</b> interface is introduced in MMC 1.1.

The 
<b>IExtendTaskPad</b> interface enables a snap-in component to set up a taskpad and receive notifications from the taskpad.
