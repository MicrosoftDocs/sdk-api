# GAMING_DEVICE_MODEL_INFORMATION structure


## -description

Contains information about the device that the game is running on.

## -struct-fields

### -field vendorId

The vendor of the device.

### -field deviceId

The type of device.

## -remarks

Supported in both Win32 and UWP apps.
