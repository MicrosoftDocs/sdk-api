# ACTCTX_COMPATIBILITY_ELEMENT_TYPE enumeration


## -description

The <b>ACTCTX_COMPATIBILITY_ELEMENT_TYPE</b> enumeration describes the compatibility element in the application manifest.

## -enum-fields

### -field ACTCTX_COMPATIBILITY_ELEMENT_TYPE_UNKNOWN:0

### -field ACTCTX_COMPATIBILITY_ELEMENT_TYPE_OS

### -field ACTCTX_COMPATIBILITY_ELEMENT_TYPE_MITIGATION

#### - ACTCX_COMPATIBILITY_ELEMENT_TYPE_OS

The compatibility element in the application manifest specifies a supported version of  Windows.


#### - ACTCX_COMPATIBILITY_ELEMENT_TYPE_UNKNOWN

The compatibility element is an unknown type.
