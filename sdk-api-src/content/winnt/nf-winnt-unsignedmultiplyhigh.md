# UnsignedMultiplyHigh function


## -description


Multiplies two 64-bit integers to produce a 128-bit integer and gets the high unsigned 64 bits.


## -parameters




### -param Multiplier [in]

The first integer.


### -param Multiplicand [in]

The second integer.


## -returns



The high 64 bits of the product.
