# IEnumNetworks::Clone


## -description


The <b>Clone</b> method creates an enumerator that contains the same enumeration state as the enumerator currently in use.


## -parameters




### -param ppEnumNetwork [out]

Pointer to a new <a href="https://msdn.microsoft.com/ce2b65e5-89fe-48c9-aa00-373406891d02">IEnumNetworks</a> interface.


## -returns



Returns S_OK if the method succeeds.
