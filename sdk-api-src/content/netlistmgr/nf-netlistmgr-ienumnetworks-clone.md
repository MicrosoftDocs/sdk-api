# IEnumNetworks::Clone


## -description


The <b>Clone</b> method creates an enumerator that contains the same enumeration state as the enumerator currently in use.


## -parameters




### -param ppEnumNetwork [out]

Pointer to a new <a href="https://docs.microsoft.com/windows/desktop/api/netlistmgr/nn-netlistmgr-ienumnetworks">IEnumNetworks</a> interface.


## -returns



Returns S_OK if the method succeeds.
