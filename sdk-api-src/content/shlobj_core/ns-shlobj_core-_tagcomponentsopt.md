# _tagCOMPONENTSOPT structure


## -description


Contains the desktop item options.


## -struct-fields




### -field dwSize

Type: <b>DWORD</b>

A <b>DWORD</b> value that contains the size of the structure.


### -field fEnableComponents

Type: <b>BOOL</b>

A <b>BOOL</b> value that determines if desktop items are enabled.


### -field fActiveDesktop

Type: <b>BOOL</b>

A <b>BOOL</b> value that determines if the Active Desktop is enabled.
