# IProgressDialog::StopProgressDialog


## -description

Stops the progress dialog box and removes it from the screen.



## -returns

Type: <b>HRESULT</b>

If this method succeeds, it returns <b>S_OK</b>. Otherwise, it returns an <b>HRESULT</b> error code.
