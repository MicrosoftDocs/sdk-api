# IInkStrokes::get_RecognitionResult


## -description

Gets the <a href="/windows/desktop/api/msinkaut/nn-msinkaut-iinkrecognitionresult">IInkRecognitionResult</a> object of the <a href="/previous-versions/windows/desktop/legacy/ms703293(v=vs.85)">InkStrokes</a> collection.



This property is read-only.

## -parameters

## -remarks

The value of this property is <b>NULL</b> if recognition has not been done or if the results were not persisted.
