# InkOverlayEditingMode enumeration


## -description



Specifies the behavior mode of the <a href="https://docs.microsoft.com/windows/desktop/tablet/inkoverlay-class">InkOverlay</a> object and the <a href="https://docs.microsoft.com/windows/desktop/tablet/inkpicture-control-reference">InkPicture</a> control.




## -enum-fields




### -field IOEM_Ink

The object or control is in collection mode.


### -field IOEM_Delete

The object or control is in deletion mode.


### -field IOEM_Select

The object or control is in selection and editing mode.
