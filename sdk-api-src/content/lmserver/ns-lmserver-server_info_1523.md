# SERVER_INFO_1523 structure


## -description


The
				<b>SERVER_INFO_1523</b> structure specifies the length of time the server retains information about incomplete search operations.


## -struct-fields




### -field sv1523_maxkeepsearch

Specifies the length of time the server retains information about incomplete search operations.
