# _DHCPCAPI_CLASSID structure


## -description


The <b>DHCPCAPI_CLASSID</b> structure defines a client Class ID.

Reserved. Must be set to zero.


### -field Data.size_is

 


### -field Data.size_is.nBytesData

 


### -field Data

Class ID binary data.


### -field nBytesData

Size of <b>Data</b>, in bytes.
