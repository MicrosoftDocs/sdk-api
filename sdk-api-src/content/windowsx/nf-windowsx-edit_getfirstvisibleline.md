# Edit_GetFirstVisibleLine macro


## -description


Gets the index of the uppermost visible line in a multiline edit or rich edit control. You can use this macro or send the <a href="https://docs.microsoft.com/windows/desktop/Controls/em-getfirstvisibleline">EM_GETFIRSTVISIBLELINE</a> message explicitly.


## -parameters




### -param hwndCtl

Type: <b><a href="https://docs.microsoft.com/windows/desktop/WinProg/windows-data-types">HWND</a></b>

A handle to the control.


## -remarks



For more information, see <a href="https://docs.microsoft.com/windows/desktop/Controls/em-getfirstvisibleline">EM_GETFIRSTVISIBLELINE</a>.
