# ListBox_GetHorizontalExtent macro


## -description


Gets the width that a list box can be scrolled horizontally (the scrollable width) if the list box has a horizontal scroll bar. You can use this macro or send the <a href="https://docs.microsoft.com/windows/desktop/Controls/lb-gethorizontalextent">LB_GETHORIZONTALEXTENT</a> message explicitly.


## -parameters




### -param hwndCtl

Type: <b><a href="https://docs.microsoft.com/windows/desktop/WinProg/windows-data-types">HWND</a></b>

A handle to the control.


## -remarks



For more information, see <a href="https://docs.microsoft.com/windows/desktop/Controls/lb-gethorizontalextent">LB_GETHORIZONTALEXTENT</a>.
