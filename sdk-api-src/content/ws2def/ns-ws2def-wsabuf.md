# WSABUF structure


## -description


The 
<b>WSABUF</b> structure enables the creation or manipulation of a data buffer used by some Winsock functions.


## -struct-fields




### -field len

The length of the buffer, in bytes.


### -field buf

A pointer to the buffer.
