# SCODE_FACILITY macro


## -description

Extracts the facility of the specified <b>SCODE</b>.

## -parameters

### -param sc

The status code.

## -remarks

This macro is defined as follows:


``` syntax
#define SCODE_FACILITY(sc)    (((sc) &gt;&gt; 16) &amp; 0x1fff)
```
