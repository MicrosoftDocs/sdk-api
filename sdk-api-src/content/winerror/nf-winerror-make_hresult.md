# MAKE_HRESULT macro


## -description

Creates an <b>HRESULT</b> value from its component pieces.

## -parameters

### -param sev

The severity.

### -param fac

The facility.

### -param code

The code.

## -remarks

This macro is defined as follows:


``` syntax
#define MAKE_HRESULT(sev,fac,code) \
    ((HRESULT) (((unsigned long)(sev)&lt;&lt;31) | ((unsigned long)(fac)&lt;&lt;16) | ((unsigned long)(code))) )
```
