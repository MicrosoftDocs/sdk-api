# HRESULT_CODE macro


## -description

Extracts the code portion of the specified <b>HRESULT</b>.

## -parameters

### -param hr

The <b>HRESULT</b> value.

## -remarks

This macro is defined as follows:


``` syntax
#define HRESULT_CODE(hr)    ((hr) &amp; 0xFFFF)
```
