# SCODE_CODE macro


## -description

Extracts the code portion of the specified <b>SCODE</b>.

## -parameters

### -param sc

The status code.

## -remarks

This macro is defined as follows:


``` syntax
#define SCODE_CODE(sc)      ((sc) &amp; 0xFFFF)
```
