# IMsmConfigurableItem interface


## -description

The <b>IMsmConfigurableItem</b> interface manages a single row from the 
<a href="/windows/desktop/Msi/moduleconfiguration-table">ModuleConfiguration table</a>. This is a single configurable "attribute" from the module. The interface consists of read-only properties, one for each column in the ModuleConfiguration table.
