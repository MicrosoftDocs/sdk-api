# XMMatrixTranslation function


## -description


Builds a translation matrix from the specified offsets.


## -parameters




### -param OffsetX [in]

Translation along the x-axis.


### -param OffsetY [in]

Translation along the y-axis.


### -param OffsetZ [in]

Translation along the z-axis.


## -returns



Returns the translation matrix.




## -remarks
