# XMVector3EqualInt function


## -description

Tests whether two 3D vectors are equal, treating each component as an unsigned integer.

## -parameters

### -param V1 [in]

3D vector.

### -param V2 [in]

3D vector.

## -returns

Returns true if the 3D vectors are equal and false otherwise.

## -remarks
