# XMVector3IsInfinite function


## -description

Tests whether any component of a 3D vector is positive or negative infinity.

## -parameters

### -param V [in]

3D vector.

## -returns

Returns true if any component of <i>V</i> is positive or negative infinity, and false otherwise.

## -remarks
