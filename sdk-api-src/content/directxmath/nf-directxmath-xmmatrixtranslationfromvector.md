# XMMatrixTranslationFromVector function


## -description


Builds a translation matrix from a vector.


## -parameters




### -param Offset [in]

3D vector describing the translations along the x-axis, y-axis, and z-axis.


## -returns



Returns the translation matrix.




## -remarks
