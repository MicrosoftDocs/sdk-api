# XMVectorSplatX function


## -description


Replicates the x component of a vector to all of the components.


## -parameters




### -param V [in]

Vector from which to select the x component.


## -returns



Returns a vector, all of whose components are equal to the x component of <i>V</i>.




## -remarks
