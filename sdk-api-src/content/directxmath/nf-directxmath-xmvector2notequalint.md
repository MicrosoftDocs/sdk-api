# XMVector2NotEqualInt function


## -description

Test whether two vectors are not equal, treating each component as an unsigned integer.

## -parameters

### -param V1 [in]

2D vector.

### -param V2 [in]

2D vector.

## -returns

Returns true if the 2D vectors are not equal and false otherwise.

## -remarks
