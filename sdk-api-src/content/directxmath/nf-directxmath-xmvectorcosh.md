# XMVectorCosH function


## -description


Computes the hyperbolic cosine of each component of an <a href="https://docs.microsoft.com/windows/desktop/dxmath/xmvector-data-type">XMVECTOR</a>.


## -parameters




### -param V [in]

Vector for which to compute the hyperbolic cosine.


## -returns



Returns a vector. Each component is the hyperbolic cosine of the corresponding component of <i>V</i>.




## -remarks

Microsoft Visual Studio 2010 or Microsoft Visual Studio 2012 with the Windows SDK for Windows 8. Supported for Win32 desktop apps, Windows Store apps, and Windows Phone 8 apps.
