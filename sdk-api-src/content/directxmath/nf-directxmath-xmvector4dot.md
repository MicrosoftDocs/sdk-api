# XMVector4Dot function


## -description


Computes the dot product between 4D vectors.


## -parameters




### -param V1 [in]

4D vector.


### -param V2 [in]

4D vector.


## -returns



Returns a vector. The dot product between <i>V1</i> and <i>V2</i> is replicated into each component. 




## -remarks
