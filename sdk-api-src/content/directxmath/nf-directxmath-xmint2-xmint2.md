# XMINT2::XMINT2


## -description

Default constructor for <code>XMINT2</code>.
    

Default constructor for <a href="/windows/desktop/api/directxmath/ns-directxmath-xmint2">XMINT2</a>.
<div class="alert"><b>Note</b>  This constructor is only available under C++.</div><div> </div>



## -remarks

<code>XMINT2()</code> constructs a new instance of the <code>XMINT2</code> structure without
	defining the value of any of its members.
