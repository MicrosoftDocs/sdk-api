# XMVector3ReciprocalLength function


## -description

Computes the reciprocal of the length of a 3D vector.

## -parameters

### -param V [in]

3D vector.

## -returns

Returns a vector. The reciprocal of the length of <i>V</i> is replicated into each of the returned vector's components.

## -remarks
