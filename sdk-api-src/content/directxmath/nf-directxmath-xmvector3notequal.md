# XMVector3NotEqual function


## -description


Tests whether two 3D vectors are not equal.


## -parameters




### -param V1 [in]

3D vector.


### -param V2 [in]

3D vector.


## -returns



Returns true if the 3D vectors are not equal and false otherwise.




## -remarks



The following pseudocode demonstrates the operation of the function:


```
return ( V1.x != V2.x || 
         V1.y != V2.y ||
         V1.z != V2.z;
```
