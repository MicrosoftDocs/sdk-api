# XMVectorSubtract function


## -description

Computes the difference of two vectors.

## -parameters

### -param V1 [in]

First vector.

### -param V2 [in]

Vector to subtract from <i>V1</i>

## -returns

Returns a vector that is the difference of <i>V1</i> and <i>V2</i>.

## -remarks
