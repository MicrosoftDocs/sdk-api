# XMVector4IsInfinite function


## -description


Tests whether any component of a 4D vector is positive or negative infinity.


## -parameters




### -param V [in]

4D vector.


## -returns



Returns true if any component of <i>V</i> is positive or negative infinity, and false otherwise.




## -remarks
