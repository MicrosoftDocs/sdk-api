# XMScalarACos function


## -description

Computes the arccosine of a floating-point number.

## -parameters

### -param Value [in]

<b>float</b> value between -1.0f and 1.0f.

## -returns

Returns the inverse cosine of <i>Value</i>.

## -remarks

This function uses a 7-degree minimax approximation.
