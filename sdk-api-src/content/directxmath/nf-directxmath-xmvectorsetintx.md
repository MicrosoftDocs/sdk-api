# XMVectorSetIntX function


## -description


Set the value of the <code>x</code> component of an <a href="https://docs.microsoft.com/windows/desktop/dxmath/xmvector-data-type">XMVECTOR Data Type</a>.


## -parameters




### -param V [in]

A valid 4D vector storing integer data.


### -param x [in]

An integer value to be assigned to <code>x</code> of <i>V</i>.


## -returns



None.




## -remarks
