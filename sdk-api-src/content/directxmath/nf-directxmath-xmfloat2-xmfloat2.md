# XMFLOAT2::XMFLOAT2


## -description

Default constructor for <code>XMFLOAT2</code>.
    

Default constructor for <a href="/windows/desktop/api/directxmath/ns-directxmath-xmfloat2">XMFLOAT2</a>.
<div class="alert"><b>Note</b>  This constructor is only available under C++.</div><div> </div>



## -remarks

<code>XMFLOAT2()</code> constructs a new instance of the <code>XMFLOAT2</code> structure without
	defining the value of any of its members.
