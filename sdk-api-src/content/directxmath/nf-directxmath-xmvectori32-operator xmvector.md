# operator XMVECTOR function


## -description


Returns the instance of <code>XMVECTOR</code> implementing the current instance of <code>XMVECTORF32</code>.

The <code>operator XMVECTOR</code> returns the instance of <a href="https://msdn.microsoft.com/1a044094-444d-e787-fa6a-76e88531aef1">XMVECTOR</a> implementing the current instance of
  <a href="https://msdn.microsoft.com/500547B8-73C1-4655-9405-1B3AD4E02B0E">XMVECTORF32</a>.
<div class="alert"><b>Note</b>  This operator is only available for C++ based development.</div><div> </div>

## -parameters






## -returns



The instance of <code>XMVECTOR</code> implementing the current instance of <code>XMVECTORF32</code>.
