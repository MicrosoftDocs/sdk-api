# operator __m128d function


## -description


Returns the object implementing the current instance of <a href="https://msdn.microsoft.com/500547B8-73C1-4655-9405-1B3AD4E02B0E">XMVECTORF32</a> cast to a <a href="http://msdn.microsoft.com/en-us/library/25537780.aspx">__m128d</a> instance
  [Windows systems supporting <a href="http://msdn.microsoft.com/en-us/library/t467de55.aspx">SSE</a> Only]
<div class="alert"><b>Note</b>  This operator is only available for Windows C++ based development.</div><div> </div>

## -parameters






## -returns



The underlying implementation object for an instance of <a href="https://msdn.microsoft.com/500547B8-73C1-4655-9405-1B3AD4E02B0E">XMVECTORF32</a> cast to a
	  <a href="http://msdn.microsoft.com/en-us/library/25537780.aspx">__m128d</a> instance.
