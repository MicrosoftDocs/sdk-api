# IWebApplicationNavigationEvents::DownloadComplete


## -description


Download of a page has completed.


## -parameters






## -returns



Type: <b>HRESULT</b>

Ignored by the host. If this method succeeds, it returns <b>S_OK</b>. Otherwise, it returns an <b>HRESULT</b> error code.
