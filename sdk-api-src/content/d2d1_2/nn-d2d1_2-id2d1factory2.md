# ID2D1Factory2 interface


## -description


Creates Direct2D resources.
      

This interface also enables the creation of <a href="https://docs.microsoft.com/windows/desktop/api/d2d1_2/nn-d2d1_2-id2d1device1">ID2D1Device1</a> objects.
