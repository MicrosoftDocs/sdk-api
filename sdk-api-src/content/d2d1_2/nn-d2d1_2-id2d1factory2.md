# ID2D1Factory2 interface


## -description


Creates Direct2D resources.
      

This interface also enables the creation of <a href="https://msdn.microsoft.com/D0CC0F2C-2BAA-4BD6-AE67-BF99458160F9">ID2D1Device1</a> objects.
