# ID2D1CommandSink1 interface


## -description

This interface performs all the same functions as the existing <a href="/windows/desktop/api/d2d1_1/nn-d2d1_1-id2d1commandsink">ID2D1CommandSink</a> interface. It also enables access to the new primitive blend modes, MIN and ADD, through its <a href="/windows/desktop/Direct2D/id2d1commandsink-setprimitiveblend1">SetPrimitiveBlend1</a> method.
