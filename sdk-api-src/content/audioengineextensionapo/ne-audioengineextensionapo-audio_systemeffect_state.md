## -description

Specifies the state of a System Effects Audio Processing Object (sAPO) audio effect.

## -enum-fields

### -field AUDIO_SYSTEMEFFECT_STATE_OFF

The audio effect is off.

### -field AUDIO_SYSTEMEFFECT_STATE_ON

The audio effect is on.
