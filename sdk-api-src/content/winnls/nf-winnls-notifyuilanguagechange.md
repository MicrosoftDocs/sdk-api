# NotifyUILanguageChange function


## -description


Unsupported. <b>NotifyUILanguageChange</b> may be altered or unavailable.


## -parameters




### -param dwFlags [in]

Reserved.


### -param pcwstrNewLanguage [in, optional]

The new language.


### -param pcwstrPreviousLanguage [in, optional]

The previous language.


### -param dwReserved [in]

Reserved.


### -param pdwStatusRtrn [out, optional]

A pointer to a <b>DWORD</b> return status.


## -returns



A <b>BOOL</b> datatype.




## -remarks



<b>NotifyUILanguageChange</b> is not supported and may be altered or unavailable in the future.
