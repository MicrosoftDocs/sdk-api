# FILEMUIINFO_GET_MAIN_TYPEIDS macro


## -description


Gets the main module types array associated with the type identifier offset information in the <b>dwTypeIDMainOffset</b> member of a <a href="https://msdn.microsoft.com/4c757d19-ac66-4ba4-a691-f575f61961be">FILEMUIINFO</a> structure.


## -parameters




### -param pInfo

Pointer to the <a href="https://msdn.microsoft.com/4c757d19-ac66-4ba4-a691-f575f61961be">FILEMUIINFO</a> structure.
