# FILEMUIINFO_GET_CULTURE macro


## -description


Gets the locale name associated with the language name offset information in the <b>dwLanguageNameOffset</b> member of a <a href="https://msdn.microsoft.com/4c757d19-ac66-4ba4-a691-f575f61961be">FILEMUIINFO</a> structure.


## -parameters




### -param pInfo

Pointer to the <a href="https://msdn.microsoft.com/4c757d19-ac66-4ba4-a691-f575f61961be">FILEMUIINFO</a> structure.
