# IFECommon::InvokeDictToolDialog


## -description

Invokes the Microsoft IME's Dictionary Tool from the app.

## -parameters

### -param pimedlg [in]

Pointer to an <a href="/windows/desktop/api/msime/ns-msime-imedlg">IMEDLG</a> structure.

## -returns

<b>S_OK</b> if successful, 
otherwise <b>E_FAIL</b>.
