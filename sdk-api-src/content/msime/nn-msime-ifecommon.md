# IFECommon interface


## -description

The <b>IFECommon</b> interface provides IME-related services that are common for different languages.

<b>IFECommon</b> allows the developer to control very basic functions of IMEs.

## -inheritance

The <b xmlns:loc="http://microsoft.com/wdcml/l10n">IFECommon</b> interface inherits from the <a href="/windows/desktop/api/unknwn/nn-unknwn-iunknown">IUnknown</a> interface. <b>IFECommon</b> also has these types of members:
<ul>
<li><a href="https://docs.microsoft.com/">Methods</a></li>
</ul>
