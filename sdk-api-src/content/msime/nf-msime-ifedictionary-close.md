# IFEDictionary::Close


## -description

Closes a dictionary file.

This method closes the file associated to this <a href="/windows/desktop/api/msime/nn-msime-ifedictionary">IFEDictionary</a> object.

## -parameters

## -returns

<b>S_OK</b> if successful, otherwise <b>E_FAIL</b>.
