# IPropertyValue::GetPoint (windows.foundation.h)


## -description

Gets the <a href="/windows/desktop/api/windows.foundation/ns-windows-foundation-point">Point</a> value that is stored in the current <a href="/windows/desktop/api/windows.foundation/nn-windows-foundation-ipropertyvalue">IPropertyValue</a> object.

## -parameters

### -param value [out, retval]

Type: <b><a href="/windows/desktop/api/windows.foundation/ns-windows-foundation-point">Point</a>*</b>

The value.

## -returns

Type: <b>HRESULT</b>

If this method succeeds, it returns <b>S_OK</b>. Otherwise, it returns an <b>HRESULT</b> error code.
