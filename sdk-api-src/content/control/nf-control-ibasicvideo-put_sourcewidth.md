# IBasicVideo::put_SourceWidth


## -description

The <code>put_SourceWidth</code> method sets the width of the source rectangle.

## -parameters

### -param SourceWidth [in]

Specifies the width, in pixels.

## -returns

Returns an <b>HRESULT</b> value.
