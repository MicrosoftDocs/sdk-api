# MF_FILE_ACCESSMODE enumeration


## -description

Specifies the requested access mode for opening a file.

## -enum-fields

### -field MF_ACCESSMODE_READ

Read mode.

### -field MF_ACCESSMODE_WRITE

Write mode.

### -field MF_ACCESSMODE_READWRITE

Read and write mode.
