# ITraceRelogger::Cancel


## -description

The <b>Cancel</b> method terminates the relogging process.



## -returns

Type: <b>HRESULT</b>

If this method succeeds, it returns <b>S_OK</b>. Otherwise, it returns an <b>HRESULT</b> error code.
