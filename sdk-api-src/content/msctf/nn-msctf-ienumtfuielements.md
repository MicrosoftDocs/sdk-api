# IEnumTfUIElements interface


## -description

The <b>IEnumTfUIElements</b> interface is implemented by TSF manager and used by applications or textservices. This interface can be retrieved by <a href="/windows/desktop/api/msctf/nf-msctf-itfuielementmgr-enumuielements">ITfUIElementMgr::EnumUIElements</a>  and enumerates the registered UI elements.
