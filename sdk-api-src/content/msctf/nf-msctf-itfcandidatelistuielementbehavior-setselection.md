# ITfCandidateListUIElementBehavior::SetSelection


## -description


The <b>ITfCandidateListUIElementBehavior::SetSelection</b> method set the selection of the candidate list.


## -parameters




### -param nIndex [in]

[in] An index for the candidate string to be selected.
