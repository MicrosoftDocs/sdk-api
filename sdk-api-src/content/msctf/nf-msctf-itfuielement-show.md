# ITfUIElement::Show


## -description


The <b>ITfUIElement::Show</b> method shows the text service's UI of this UI element.


## -parameters




### -param bShow [in]

[in] <b>TRUE</b> to show the original UI of the element. <b>FALSE</b> to hide the original UI of the element.
