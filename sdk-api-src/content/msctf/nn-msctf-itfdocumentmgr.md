# ITfDocumentMgr interface


## -description

The <b>ITfDocumentMgr</b> interface is implemented by the TSF manager and used by an application or text service to create and manage text contexts. To obtain an instance of this interface call <a href="/windows/desktop/api/msctf/nf-msctf-itfthreadmgr-createdocumentmgr">ITfThreadMgr::CreateDocumentMgr</a>.

The <b xmlns:loc="http://microsoft.com/wdcml/l10n">ITfDocumentMgr</b> interface inherits from the <a href="/windows/desktop/api/unknwn/nn-unknwn-iunknown">IUnknown</a> interface. <b>ITfDocumentMgr</b> also has these types of members:
