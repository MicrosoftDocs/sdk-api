# __MIDL___MIDL_itf_msctf_0000_0070_0001 enumeration


## -description


Elements of the <b>TF_DA_LINESTYLE</b> enumeration specify the underline style of a display attribute in the <a href="https://msdn.microsoft.com/0ce8f941-c187-437f-8bad-f882e63b8421">TF_DA_COLOR</a> structure.


## -enum-fields




### -field TF_LS_NONE

The text is not underlined.


### -field TF_LS_SOLID

The text is underlined with a solid line.


### -field TF_LS_DOT

The text is underlined with a dotted line.


### -field TF_LS_DASH

The text is underlined with a dashed line.


### -field TF_LS_SQUIGGLE

The text is underlined with a solid wavy line.
