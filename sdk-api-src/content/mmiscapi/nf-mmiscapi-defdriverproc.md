# DefDriverProc function


## -description



Provides default processing for any messages not processed by an installable driver. This function is intended to be used only within the <a href="https://docs.microsoft.com/previous-versions/dd797918(v=vs.85)">DriverProc</a> function of an installable driver.




## -parameters




### -param dwDriverIdentifier

Identifier of the installable driver.


### -param hdrvr

Handle of the installable driver instance.


### -param uMsg

Driver message value.


### -param lParam1

32-bit message-dependent information.


### -param lParam2

32-bit message-dependent information.


## -returns



Returns nonzero if successful or zero otherwise.
