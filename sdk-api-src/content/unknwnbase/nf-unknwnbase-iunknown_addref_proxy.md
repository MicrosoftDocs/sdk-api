# IUnknown_AddRef_Proxy function


## -description

<p class="CCE_Message">[IUnknown_AddRef_Proxy is not supported and may be altered or unavailable in the future.]

The <b>IUnknown_AddRef_Proxy</b> function implements the <a href="/windows/desktop/api/unknwn/nf-unknwn-iunknown-addref">AddRef</a> method for all interface proxies.

## -parameters

### -param This

Pointer to  the proxy object.

## -returns

Returns an integer from 1 to <i>n</i>, indicating the value of the new reference count.

## -remarks

 The return value is for diagnostic and testing purposes only; the return value may be unstable.

This function is used internally by proxies that are generated by MIDL.
