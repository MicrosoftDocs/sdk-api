# IDVBTLocator::get_Mode


## -description



The <b>get_Mode</b> method receives the transmission mode.




## -parameters




### -param mode [out]

Receives a member of the <a href="https://docs.microsoft.com/previous-versions/windows/desktop/mstv/transmissionmode">TransmissionMode</a> enumeration.


## -returns



Returns S_OK if successful. If the method fails, error information can be retrieved using the standard COM <b>IErrorInfo</b> interface.
