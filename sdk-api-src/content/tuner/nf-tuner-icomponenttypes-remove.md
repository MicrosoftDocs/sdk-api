# IComponentTypes::Remove


## -description



The <b>Remove</b> method removes the <a href="https://docs.microsoft.com/previous-versions/windows/desktop/legacy/dd693036(v=vs.85)">ComponentType</a> object at the specified index number.




## -parameters




### -param Index [in]

Index of the item to remove.


## -returns



Returns S_OK if successful. If the method fails, error information can be retrieved using the standard COM <b>IErrorInfo</b> interface.
