# IDigitalCableTuningSpace::put_MinMajorChannel


## -description



The <b>put_MinMajorChannel</b> method sets the lowest major channel number for this tuning space.




## -parameters




### -param NewMinMajorChannelVal [in]

Specifies the lowest major channel number.


## -returns



If the method succeeds, it returns S_OK. If it fails, it returns an error code.
