# IAnalogRadioTuningSpace::get_MinFrequency


## -description

\[The feature associated with this page, [Microsoft TV Technologies](/previous-versions/windows/desktop/mstv/microsoft-tv-technologies-portal), is a legacy feature. Microsoft strongly recommends that new code does not use this feature.\]

The <b>get_MinFrequency</b> method retrieves the minimum frequency for this tuning space.

## -parameters

### -param MinFrequencyVal [out]

Pointer to a variable that receives the minimum frequency, in kilohertz (kHz).

## -returns

Returns E_NOTIMPL.
