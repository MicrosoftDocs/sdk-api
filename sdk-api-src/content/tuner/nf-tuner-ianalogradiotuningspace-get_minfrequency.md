# IAnalogRadioTuningSpace::get_MinFrequency


## -description


The <b>get_MinFrequency</b> method retrieves the minimum frequency for this tuning space.


## -parameters




### -param MinFrequencyVal [out]

Pointer to a variable that receives the minimum frequency, in kilohertz (kHz).


## -returns



Returns E_NOTIMPL.
