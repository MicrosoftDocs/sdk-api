# IAnalogRadioTuningSpace::put_MaxFrequency


## -description


The <b>put_MaxFrequency</b> method sets the maximum frequency for this tuning space.


## -parameters




### -param NewMaxFrequencyVal [in]

Specifies the maximum frequency, in kilohertz (kHz).


## -returns



Returns E_NOTIMPL.
