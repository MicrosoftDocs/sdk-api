# IAnalogRadioTuningSpace::put_Step


## -description


The <b>put_Step</b> method sets the step value to the next frequency.


## -parameters




### -param NewStepVal [in]

Specifies the step value to the next frequency, in kilohertz (kHz).


## -returns



Returns E_NOTIMPL.
