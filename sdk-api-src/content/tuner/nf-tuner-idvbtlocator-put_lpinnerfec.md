# IDVBTLocator::put_LPInnerFEC


## -description



The <b>put_LPInnerFEC</b> method sets the inner FEC type of the low-priority stream.




## -parameters




### -param FEC [in]

Variable of type <a href="https://docs.microsoft.com/previous-versions/windows/desktop/mstv/fecmethod">FECMethod</a> that specifies the FEC type.


## -returns



Returns S_OK if successful. If the method fails, error information can be retrieved using the standard COM <b>IErrorInfo</b> interface.
