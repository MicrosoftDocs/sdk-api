# IAnalogRadioTuningSpace::put_MinFrequency


## -description


The <b>put_MinFrequency</b> method sets the minimum frequency for this tuning space.


## -parameters




### -param NewMinFrequencyVal [in]

Specifies the minimum frequency, in kilohertz (kHz).


## -returns



Returns E_NOTIMPL.
