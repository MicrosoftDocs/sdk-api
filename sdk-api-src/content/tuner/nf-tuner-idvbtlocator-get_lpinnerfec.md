# IDVBTLocator::get_LPInnerFEC


## -description



The <b>get_LPInnerFEC</b> method retrieves the inner FEC type of the low-priority stream.




## -parameters




### -param FEC [out]

Receives a member of the <a href="https://docs.microsoft.com/previous-versions/windows/desktop/mstv/fecmethod">FECMethod</a> enumeration.


## -returns



Returns S_OK if successful. If the method fails, error information can be retrieved using the standard COM <b>IErrorInfo</b> interface.
