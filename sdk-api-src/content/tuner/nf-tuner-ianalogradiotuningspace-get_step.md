# IAnalogRadioTuningSpace::get_Step


## -description


The <b>get_Step</b> method retrieves the step value to the next frequency.


## -parameters




### -param StepVal [out]

Pointer to a variable receives the step value to the next frequency, in kilohertz (kHz).


## -returns



Returns E_NOTIMPL.
