# XMSHORT4::XMSHORT4


## -description

Default constructor for <code>XMSHORT4</code>.
    

Default constructor for <a href="/windows/desktop/api/directxpackedvector/ns-directxpackedvector-xmshort4">XMSHORT4</a>.
<div class="alert"><b>Note</b>  This constructor is only available under C++.
    </div><div> </div>



## -remarks

<code>XMSHORT4()</code> constructs a new instance of the <code>XMSHORT4</code> structure without
	defining the value of any of its members.
