# XMHALF4::XMHALF4


## -description

Default constructor for <code>XMHALF4</code>.
    

Default constructor for <a href="/windows/desktop/api/directxpackedvector/ns-directxpackedvector-xmhalf4">XMHALF4</a>.
<div class="alert"><b>Note</b>  This constructor is only available under C++.
    </div><div> </div>



## -remarks

<code>XMHALF4()</code> constructs a new instance of the <code>XMHALF4</code> structure without
	defining the value of any of its members.
