# XMUSHORTN2::XMUSHORTN2


## -description

Default constructor for <code>XMUSHORTN2</code>.

Default constructor for <a href="/windows/desktop/api/directxpackedvector/ns-directxpackedvector-xmushortn2">XMUSHORTN2</a>.
<div class="alert"><b>Note</b>  This constructor is only available under C++.</div><div> </div>



## -remarks

<code>XMUSHORTN2()</code> constructs a new instance of the <code>XMUSHORTN2</code> structure without
	  defining the value of any of its members.
