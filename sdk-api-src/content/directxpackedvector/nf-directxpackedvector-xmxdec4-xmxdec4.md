# XMXDEC4::XMXDEC4


## -description

Default constructor for <code>XMXDEC4</code>.
    

Default constructor for <a href="/windows/desktop/api/directxpackedvector/ns-directxpackedvector-xmxdec4">XMXDEC4</a> .
<div class="alert"><b>Note</b>  This constructor is only available under C++.</div><div> </div>



## -remarks

<code>XMXDEC4()</code> constructs a new instance of the <code>XMXDEC4</code> structure without
