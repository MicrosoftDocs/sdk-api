# XMUSHORTN4::XMUSHORTN4


## -description

Default constructor for <code>XMUSHORTN4</code>.
    

Default constructor for <a href="/windows/desktop/api/directxpackedvector/ns-directxpackedvector-xmushortn4">XMUSHORTN4</a>.
<div class="alert"><b>Note</b>  This constructor is only available under C++.
    </div><div> </div>



## -remarks

<code>XMUSHORTN4()</code> constructs a new instance of the <code>XMUSHORTN4</code> structure without
	defining the value of any of its members.
