# XMLoadUDecN4 function


## -description


Loads an <a href="https://docs.microsoft.com/windows/desktop/api/directxpackedvector/ns-directxpackedvector-xmudecn4">XMUDECN4</a> into an <a href="https://docs.microsoft.com/windows/desktop/dxmath/xmvector-data-type">XMVECTOR</a>.


## -parameters




### -param pSource [in]

Address of the <a href="https://docs.microsoft.com/windows/desktop/api/directxpackedvector/ns-directxpackedvector-xmudecn4">XMUDECN4</a> structure to load. 


## -returns



Returns an <a href="https://docs.microsoft.com/windows/desktop/dxmath/xmvector-data-type">XMVECTOR</a> loaded with the data from the <i>pSource</i> parameter.




## -remarks



The following pseudocode demonstrates the operation of the function.


```
XMVECTOR vectorOut;

uint32_t Element;

Element = pSource->v & 0x3FF;
vectorOut.x = (float)Element / 1023.f;
Element = (pSource->v >> 10) & 0x3FF;
vectorOut.y = (float)Element / 1023.f;
Element = (pSource->v >> 20) & 0x3FF;
vectorOut.z = (float)Element / 1023.f;
vectorOut.w = (float)(pSource->v >> 30) / 3.f;

return vectorOut;
```
