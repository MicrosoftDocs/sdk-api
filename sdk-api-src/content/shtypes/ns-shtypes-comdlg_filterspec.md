# COMDLG_FILTERSPEC structure


## -description


Used generically to filter elements.


## -struct-fields




### -field pszName

Type: <b>LPCWSTR</b>

A pointer to a buffer that contains the friendly name of the filter.


### -field pszSpec

Type: <b>LPCWSTR</b>

A pointer to a buffer that contains the filter pattern.
