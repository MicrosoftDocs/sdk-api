# Pen::GetDashPatternCount


## -description

The <b>Pen::GetDashPatternCount</b> method gets the number of elements in a dash pattern array.



## -returns

Type: <b>INT</b>

This method returns an integer that indicates the number of elements in the dash pattern array.
