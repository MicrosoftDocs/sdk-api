# APPX_PACKAGE_EDITOR_UPDATE_PACKAGE_MANIFEST_OPTIONS enumeration


## -description

Options for app manifest validation when updating the manifest.

## -enum-fields

### -field APPX_PACKAGE_EDITOR_UPDATE_PACKAGE_MANIFEST_OPTION_NONE

No options.

### -field APPX_PACKAGE_EDITOR_UPDATE_PACKAGE_MANIFEST_OPTION_SKIP_VALIDATION

Skip app package manifest validation.

### -field APPX_PACKAGE_EDITOR_UPDATE_PACKAGE_MANIFEST_OPTION_LOCALIZED

The app package manifest is localized.
