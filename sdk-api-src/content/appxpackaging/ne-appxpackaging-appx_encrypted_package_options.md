# APPX_ENCRYPTED_PACKAGE_OPTIONS enumeration


## -description


Encrypted app package options.


## -enum-fields




### -field APPX_ENCRYPTED_PACKAGE_OPTION_NONE

No options.


### -field APPX_ENCRYPTED_PACKAGE_OPTION_DIFFUSION

Option to use diffusion.


### -field APPX_ENCRYPTED_PACKAGE_OPTION_PAGE_HASHING

Option to add page hashes.
