# tag_WBEM_GET_KEY_FLAGS enumeration


## -description

Contains flags which control the format of the text.

## -enum-fields

### -field WBEMPATH_TEXT:0x1

Not used.

### -field WBEMPATH_QUOTEDTEXT:0x2

Places quotes around the string key values.
