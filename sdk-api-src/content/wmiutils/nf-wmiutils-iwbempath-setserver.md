# IWbemPath::SetServer


## -description


The 
<b>IWbemPath::SetServer</b> method sets the server portion of the path.


## -parameters




### -param Name [in]

New server name. <b>NULL</b> is an acceptable value.


## -returns



This method returns an <b>HRESULT</b> indicating the status of the method call.
