# IWbemPath::RemoveScope


## -description

The <b>IWbemPath::RemoveScope</b> method removes a scope 
    based on the index.

## -parameters

### -param uIndex [in]

Index of the scope to be removed.

## -returns

This method returns an <b>HRESULT</b> with one of the following values.
