# IWbemPathKeyList::GetCount


## -description

The 
<b>IWbemPathKeyList::GetCount</b> method retrieves the number of keys in the path.

## -parameters

### -param puKeyCount [in, out]

Number of keys.

## -returns

This method returns an <b>HRESULT</b> indicating the status of the method call.
