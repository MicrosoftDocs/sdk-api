# IWbemPathKeyList::RemoveAllKeys


## -description

The 
<b>IWbemPathKeyList::RemoveAllKeys</b> method removes all keys.

## -parameters

### -param uFlags [in]

Reserved. Must be 0 (zero).

## -returns

This method returns an <b>HRESULT</b> indicating the status of the method call.
