# IWbemPath::SetClassName


## -description


The 
<b>IWbemPath::SetClassName</b> method sets the class name portion of the path.


## -parameters




### -param Name [in]

Class name.


## -returns



This method returns an <b>HRESULT</b> indicating the status of the method call.
