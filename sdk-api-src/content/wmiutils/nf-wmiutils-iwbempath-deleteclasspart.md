# IWbemPath::DeleteClassPart


## -description

The 
<b>IWbemPath::DeleteClassPart</b> method deletes the class portion of the path.

## -parameters

### -param lFlags [in]

Reserved. Must be 0 (zero).

## -returns

This method returns an <b>HRESULT</b> indicating the status of the method call.
