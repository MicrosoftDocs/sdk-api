# IRealTimeStylus2::get_FlicksEnabled


## -description

Returns a value indicating whether flick gestures are enabled for the RTS.

## -parameters

### -param pfEnable [out, retval]

<b>TRUE</b> if flick gestures have been enabled; otherwise, <b>FALSE</b>.

## -returns

This method can return one of these values.
