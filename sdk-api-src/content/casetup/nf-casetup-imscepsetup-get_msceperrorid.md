# IMSCEPSetup::get_MSCEPErrorId


## -description


The <b>MSCEPErrorId</b> property gets the ID for additional error information related to a failed Network Device Enrollment Service (NDES) specification. Any method call on the parent object resets this property.

This property is read-only.
