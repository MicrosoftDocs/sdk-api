# WdsTransportProviderCloseContent function


## -description


Closes a content stream specified by a handle. 


## -parameters




### -param hContent [in]

Handle to the content stream to be closed.


## -returns



If the function succeeds, the return is <b>S_OK</b>.




## -remarks



This callback is required.
