# _HTTP_LOG_DATA_TYPE enumeration


## -description


The <b>HTTP_LOG_DATA_TYPE</b> enumeration identifies the type of log data.


## -enum-fields




### -field HttpLogDataTypeFields

The <a href="https://msdn.microsoft.com/5d1b86fe-161d-4182-b3fe-9a03a843e62e">HTTP_LOG_FIELDS_DATA</a> structure is used for logging a request. This structure is passed to an <a href="https://msdn.microsoft.com/0183584f-105e-4fa3-8991-d3f2dfca1d62">HttpSendHttpResponse</a> or <a href="https://msdn.microsoft.com/f2ff2e40-ef1f-4c35-a615-f31ac63ab738">HttpSendResponseEntityBody</a> call.
