## -description

Checks whether a particular feature is supported.

## -parameters

### -param FeatureId

Type: \_In\_ **[HTTP_FEATURE_ID](./ne-http-http_feature_id.md)**

The identifier of the feature.

## -returns

`TRUE` if the feature is supported, otherwise `FALSE`.
