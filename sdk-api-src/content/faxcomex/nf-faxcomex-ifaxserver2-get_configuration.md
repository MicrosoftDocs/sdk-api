# IFaxServer2::get_Configuration


## -description

The <b>IFaxServer2::Configuration</b> property holds a <a href="/previous-versions/windows/desktop/api/faxcomex/nn-faxcomex-ifaxconfiguration">IFaxConfiguration</a> object. The object permits a fax client application to access information about the configuration of the connected fax server.

This property is read-only.
