# IFaxReceiptOptions::put_SMTPPort


## -description

The <b>IFaxReceiptOptions::get_SMTPPort</b> property is a value that specifies the Simple Mail Transport Protocol (SMTP) port number.

This property is read/write.

## -parameters

## -remarks

To read or to write to this property, a user must have the <a href="/previous-versions/windows/desktop/api/faxcomex/ne-faxcomex-fax_access_rights_enum">farQUERY_CONFIG</a> access right.
