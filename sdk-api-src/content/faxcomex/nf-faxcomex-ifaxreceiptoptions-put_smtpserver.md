# IFaxReceiptOptions::put_SMTPServer


## -description

The <b>IFaxReceiptOptions::get_SMTPServer</b> property is a null-terminated string that contains the name of the Simple Mail Transport Protocol (SMTP) server.

This property is read/write.

## -parameters

## -remarks

To read or to write to this property, a user must have the <a href="/previous-versions/windows/desktop/api/faxcomex/ne-faxcomex-fax_access_rights_enum">farQUERY_CONFIG</a> access right.
