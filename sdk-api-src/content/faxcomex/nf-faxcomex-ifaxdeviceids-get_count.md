# IFaxDeviceIds::get_Count


## -description

The <b>IFaxDeviceIds::get_Count</b> property represents the number of objects in the <a href="/previous-versions/windows/desktop/fax/-mfax-faxdeviceids">FaxDeviceIds</a> collection. This is the total number of device IDs associated with the fax server.

This property is read-only.
