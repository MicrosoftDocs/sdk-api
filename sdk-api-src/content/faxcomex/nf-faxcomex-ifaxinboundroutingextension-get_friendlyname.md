# IFaxInboundRoutingExtension::get_FriendlyName


## -description

The <b>IFaxInboundRoutingExtension::get_FriendlyName</b> property is a null-terminated string that contains the user-friendly name for the fax routing extension. The string is suitable for display to users.

This property is read-only.
