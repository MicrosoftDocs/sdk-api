# IFaxReceiptOptions::get_SMTPSender


## -description


The <b>IFaxReceiptOptions::get_SMTPSender</b> property is a null-terminated string that contains the Simple Mail Transport Protocol (SMTP) email address for the sender of the mail message receipt.

This property is read/write.


## -parameters


## -remarks



To read or to write to this property, a user must have the <a href="https://msdn.microsoft.com/70d729c6-8299-47d7-8dea-f7c754a25531">farQUERY_CONFIG</a> access right.
