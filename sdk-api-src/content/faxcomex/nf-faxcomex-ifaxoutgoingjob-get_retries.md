# IFaxOutgoingJob::get_Retries


## -description


The <b>IFaxOutgoingJob::get_Retries</b> property is a value that indicates the number of times that the fax service attempted to transmit an outgoing fax after the initial transmission attempt failed.

This property is read-only.
