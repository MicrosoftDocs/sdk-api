# IFaxReceiptOptions::get_SMTPServer


## -description


The <b>IFaxReceiptOptions::get_SMTPServer</b> property is a null-terminated string that contains the name of the Simple Mail Transport Protocol (SMTP) server.

This property is read/write.


## -parameters


## -remarks



To read or to write to this property, a user must have the <a href="https://msdn.microsoft.com/70d729c6-8299-47d7-8dea-f7c754a25531">farQUERY_CONFIG</a> access right.
