# IFaxAccounts interface


## -description

Represents the collection of fax accounts on the fax server. It provides methods and properties for enumerating the accounts, retrieving a particular account, and reporting the total number of accounts.

## -inheritance

The <b>IFaxAccounts</b> interface inherits from the <a href="/previous-versions/windows/desktop/api/oaidl/nn-oaidl-idispatch">IDispatch</a> interface. <b>IFaxAccounts</b> also has these types of members:

## -remarks

A default implementation of <b>IFaxAccounts</b> is provided as the <a href="/previous-versions/windows/desktop/fax/-mfax-faxaccounts">FaxAccounts</a> object. The interface and the object are supported only on Windows Vista or later.
