# IFaxInboundRoutingExtension::get_MajorVersion


## -description

The <b>IFaxInboundRoutingExtension::get_MajorVersion</b> property is a value that specifies the major part of the version number for the fax routing extension's DLL.

This property is read-only.

## -parameters

## -remarks

The standard format for build numbers is MajorVersion.MinorVersion.MajorBuild.MinorBuild.
