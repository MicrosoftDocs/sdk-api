# IFaxServer2::get_FaxAccountSet


## -description

The <b>IFaxServer2::FaxAccountSet</b> property returns a <a href="/previous-versions/windows/desktop/api/faxcomex/nn-faxcomex-ifaxaccountset">IFaxAccountSet</a> object used to manage the fax accounts on the fax server. 

This property is read-only.
