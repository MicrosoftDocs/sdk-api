# IFaxInboundRoutingMethod::get_FunctionName


## -description


The <b>IFaxInboundRoutingMethod::get_FunctionName</b> property is a null-terminated string that contains the name of the function that executes a specific fax routing procedure.

This property is read-only.


## -parameters


## -remarks



The fax routing extension DLL identified by the <a href="https://msdn.microsoft.com/33be2539-5477-4e56-8347-a80c5cfa77e7">IFaxInboundRoutingMethod::get_ExtensionImageName</a> property defines and exports the function.
