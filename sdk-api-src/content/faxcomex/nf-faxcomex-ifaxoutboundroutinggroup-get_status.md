# IFaxOutboundRoutingGroup::get_Status


## -description

The <b>Status</b> property indicates the collective status of the fax devices in the outbound routing group.

This property is read-only.

## -parameters

## -remarks

When devices are added to or removed from a group, the group's status does not change.
