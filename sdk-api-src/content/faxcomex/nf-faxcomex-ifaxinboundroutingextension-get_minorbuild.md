# IFaxInboundRoutingExtension::get_MinorBuild


## -description


The <b>IFaxInboundRoutingExtension::get_MinorBuild</b> property is a value that specifies the minor part of the build number for the fax routing extension's DLL.

This property is read-only.


## -parameters


## -remarks



The standard format for build numbers is MajorVersion.MinorVersion.MajorBuild.MinorBuild.
