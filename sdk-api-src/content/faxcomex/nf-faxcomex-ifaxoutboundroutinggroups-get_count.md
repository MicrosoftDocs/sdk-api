# IFaxOutboundRoutingGroups::get_Count


## -description


The <b>Count</b> property represents the number of objects in the <a href="https://docs.microsoft.com/previous-versions/windows/desktop/fax/-mfax-faxoutboundroutinggroups">FaxOutboundRoutingGroups</a> collection. This is the total number of outbound routing groups associated with the fax server.

This property is read-only.
