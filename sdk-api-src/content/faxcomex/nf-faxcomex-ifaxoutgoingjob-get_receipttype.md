# IFaxOutgoingJob::get_ReceiptType


## -description

The <b>IFaxOutgoingJob::get_ReceiptType</b> property is a value that specifies the type of delivery receipt to deliver when the fax message reaches a final state. The receipt type can be Simple Mail Transport Protocol (SMTP) mail, a message box, or no receipt.

This property is read-only.
