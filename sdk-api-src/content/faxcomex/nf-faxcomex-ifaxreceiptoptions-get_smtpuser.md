# IFaxReceiptOptions::get_SMTPUser


## -description


The <b>IFaxReceiptOptions::get_SMTPUser</b> property is a null-terminated string that contains the Simple Mail Transport Protocol (SMTP) user name used for authenticated connections. 

This property is read/write.


## -parameters


## -remarks



To read or to write to this property, a user must have the <a href="https://msdn.microsoft.com/en-us/library/ms689205(v=VS.85).aspx">farQUERY_CONFIG</a> access right.
