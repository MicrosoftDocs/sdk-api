# IFaxOutgoingMessage::get_SubmissionId


## -description

The <b>IFaxOutgoingMessage::get_SubmissionId</b> property is a null-terminated string that contains the unique identifier assigned to the fax message during the submission process. All fax jobs created by the same submission process share the same unique submission ID.

This property is read-only.
