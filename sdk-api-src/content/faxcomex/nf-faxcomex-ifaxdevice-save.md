# IFaxDevice::Save


## -description

The <b>IFaxDevice::Save</b> method saves the <a href="/previous-versions/windows/desktop/fax/-mfax-faxdevice">FaxDevice</a> object's data.



## -returns

Type: <b>HRESULT</b>

If this method succeeds, it returns <b>S_OK</b>. Otherwise, it returns an <b>HRESULT</b> error code.

## -remarks

To use this method, a user must have the <a href="/previous-versions/windows/desktop/api/faxcomex/ne-faxcomex-fax_access_rights_enum">farMANAGE_CONFIG</a> access right.
