# IFaxReceiptOptions interface


## -description

The <b>IFaxReceiptOptions</b> interface defines a <a href="/previous-versions/windows/desktop/fax/-mfax-faxreceiptoptions">FaxReceiptOptions</a> configuration object used by a fax client application to set and retrieve the receipt configuration that the fax service uses to send delivery receipts for fax transmissions.
