# IFaxOutboundRoutingRule::get_UseDevice


## -description

The <b>IFaxOutboundRoutingRule::get_UseDevice</b> property is a Boolean value that indicates whether the outbound routing rule points to a single fax device.

This property is read/write.

## -parameters

## -remarks

To read or to write to this property, a user must have the <a href="/previous-versions/windows/desktop/api/faxcomex/ne-faxcomex-fax_access_rights_enum">farQUERY_CONFIG</a> access right.
