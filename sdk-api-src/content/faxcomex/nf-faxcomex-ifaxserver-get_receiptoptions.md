# IFaxServer::get_ReceiptOptions


## -description

The <b>IFaxServer::get_ReceiptOptions</b> property creates a <a href="/previous-versions/windows/desktop/api/faxcomex/nn-faxcomex-ifaxreceiptoptions">IFaxReceiptOptions</a> configuration interface. The object permits a fax client application to set and retrieve the receipt configuration that the fax service uses to send fax receipts.

This property is read-only.
