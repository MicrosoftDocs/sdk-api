# IFaxInboundRoutingExtension::get_ImageName


## -description

The <b>IFaxInboundRoutingExtension::get_ImageName</b> property is a null-terminated string that contains the executable image name (DLL path and file name) of the fax routing extension.

This property is read-only.

## -parameters

## -remarks

The path can include valid environment variables, for example, <code>%SYSTEMDRIVE%</code> and <code>%SYSTEMROOT%</code>.
