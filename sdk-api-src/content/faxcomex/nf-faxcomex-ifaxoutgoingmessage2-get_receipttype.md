# IFaxOutgoingMessage2::get_ReceiptType


## -description

Specifies the type of delivery report that is sent following an attempted transmission. 


<div class="alert"><b>Note</b>  This property is supported only on Windows Vista and later.</div><div> </div>This property is read-only.
