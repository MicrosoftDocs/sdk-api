# IFaxOutgoingJob::get_GroupBroadcastReceipts


## -description


The <b>IFaxOutgoingJob::get_GroupBroadcastReceipts</b> property is a Boolean value that indicates whether to send an individual delivery receipt for each recipient of the broadcast or to send a summary receipt for all recipients.

This property is read-only.


## -parameters


## -remarks



Set the <b>IFaxOutgoingJob::get_GroupBroadcastReceipts</b> property for a document using the <a href="https://msdn.microsoft.com/en-us/library/ms685135(v=VS.85).aspx">IFaxDocument::put_GroupBroadcastReceipts</a> method.
