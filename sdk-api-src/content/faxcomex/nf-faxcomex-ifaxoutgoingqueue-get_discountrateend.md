# IFaxOutgoingQueue::get_DiscountRateEnd


## -description


The <b>IFaxOutgoingQueue::get_DiscountRateEnd</b> property is a value that indicates the time at which the discount period for transmitting faxes ends. The discount period applies to outgoing faxes.

This property is read/write.


## -parameters


## -remarks



To read or to write to this property, a user must have the <a href="https://msdn.microsoft.com/70d729c6-8299-47d7-8dea-f7c754a25531">farQUERY_CONFIG</a> access right.
