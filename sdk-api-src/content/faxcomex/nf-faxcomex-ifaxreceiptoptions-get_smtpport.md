# IFaxReceiptOptions::get_SMTPPort


## -description


The <b>IFaxReceiptOptions::get_SMTPPort</b> property is a value that specifies the Simple Mail Transport Protocol (SMTP) port number.

This property is read/write.


## -parameters


## -remarks



To read or to write to this property, a user must have the <a href="https://msdn.microsoft.com/70d729c6-8299-47d7-8dea-f7c754a25531">farQUERY_CONFIG</a> access right.
