# IFaxOutboundRoutingRules::get_Count


## -description


The <b>IFaxOutboundRoutingRules::get_Count</b> property represents the number of objects in the <a href="https://msdn.microsoft.com/971819e1-a3cb-4e58-b533-fee15ccb4352">FaxOutboundRoutingRules</a> collection. This is the total number of outbound routing rules associated with the fax server.

This property is read-only.
