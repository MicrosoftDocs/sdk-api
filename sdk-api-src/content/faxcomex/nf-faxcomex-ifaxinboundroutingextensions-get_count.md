# IFaxInboundRoutingExtensions::get_Count


## -description

The <b>IFaxInboundRoutingExtensions::get_Count</b> property represents the number of objects in the <a href="/previous-versions/windows/desktop/api/faxcomex/nn-faxcomex-ifaxinboundroutingextensions">IFaxInboundRoutingExtensions</a> collection. This is the total number of inbound routing extensions associated with the fax server.

This property is read-only.
