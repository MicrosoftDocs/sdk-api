# IFaxServer2::get_Security2


## -description


The <b>IFaxServer2::Security2</b> property returns a <a href="https://docs.microsoft.com/previous-versions/windows/desktop/api/faxcomex/nn-faxcomex-ifaxsecurity2">IFaxSecurity2</a> object used to configure security on the fax server. 

This property is read-only.
