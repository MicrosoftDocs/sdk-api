# IFaxDeviceProvider interface


## -description


The <b>IFaxDeviceProvider</b> interface defines a configuration object used by a fax client application to retrieve information about a fax service provider (FSP) registered with the fax service.


## -remarks



A default implementation of <b>IFaxDeviceProvider</b> is provided as the <a href="https://docs.microsoft.com/previous-versions/windows/desktop/fax/-mfax-faxdeviceprovider">FaxDeviceProvider</a> object.
