# IPicture::put_KeepOriginalFormat


## -description

Sets the value of the picture's KeepOriginalFormat property.

## -parameters

### -param keep [in]

Specifies the new value to assign to the property.

## -returns

This method returns S_OK on success and E_FAIL otherwise.
