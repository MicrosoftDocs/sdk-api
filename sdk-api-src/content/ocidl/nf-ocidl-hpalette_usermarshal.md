## -description

Marshals an HPALETTE object into the RPC buffer.

## -parameters

### -param unnamedParam1

The data used by RPC.

### -param unnamedParam2

The current buffer. This pointer may or may not be aligned on entry.

### -param unnamedParam3

The object.

## -returns

A pointer to an unsigned character array.
