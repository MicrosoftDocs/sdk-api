# _MC_POSITION_TYPE enumeration


## -description


Specifies whether to get or set the vertical or horizontal position of a monitor's display area.


## -enum-fields




### -field MC_HORIZONTAL_POSITION

Horizontal position.
          


### -field MC_VERTICAL_POSITION

Vertical position.
