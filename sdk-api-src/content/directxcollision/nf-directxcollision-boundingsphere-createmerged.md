# BoundingSphere::CreateMerged


## -description

Creates a BoundingSphere that contains the two specified BoundingSphere objects.

## -parameters

### -param Out [out, ref]

A new BoundingSphere containing the two specified BoundingSphere objects.

### -param S1 [in, ref]

A BoundingSphere that the new BoundingSphere should contain.

### -param S2 [in, ref]

A BoundingSphere that the new BoundingSphere should contain.

## -remarks

Microsoft Visual Studio 2010 or Microsoft Visual Studio 2012 with the Windows SDK for Windows 8. Supported for Win32 desktop apps, Windows Store apps, and Windows Phone 8 apps.
