# BoundingOrientedBox.Contains(XMVECTOR) method


## -description

Tests whether the **BoundingOrientedBox** contains a specified point.

## -parameters

### -param Point

The point to test against.

## -returns

A **ContainmentType** indicating whether point is contained in the **BoundingOrientedBox**.

## -remarks

### Platform Requirements

Microsoft Visual Studio 2010 or Microsoft Visual Studio 2012 with the Windows SDK for Windows 8.
Supported for Win32 desktop apps, Windows Store apps, and Windows Phone 8 apps.
