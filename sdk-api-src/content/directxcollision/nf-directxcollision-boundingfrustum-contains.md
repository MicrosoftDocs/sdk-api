# BoundingFrustum::Contains


## -description

Tests whether the [BoundingFrustum](./ns-directxcollision-boundingfrustum.md) contains the specified point.

## -parameters

### -param Point [in]

The point to test against.

## -returns

A <a href="/windows/win32/api/directxcollision/ne-directxcollision-containmenttype">ContainmentType</a> value indicating whether the point is contained in the [BoundingFrustum](./ns-directxcollision-boundingfrustum.md).

## -remarks

<h3><a id="Platform_Requirements"></a><a id="platform_requirements"></a><a id="PLATFORM_REQUIREMENTS"></a>Platform Requirements</h3>
Microsoft Visual Studio 2010 or Microsoft Visual Studio 2012 with the Windows SDK for Windows 8. Supported for Win32 desktop apps, Windows Store apps, and Windows Phone 8 apps.
