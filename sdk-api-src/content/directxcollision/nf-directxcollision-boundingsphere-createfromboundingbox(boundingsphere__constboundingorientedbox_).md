# BoundingSphere::CreateFromBoundingBox(BoundingSphere &,const BoundingOrientedBox &)


## -description

Creates a BoundingSphere containing the specified BoundingBox.

## -parameters

### -param Out [out, ref]

The new BoundingSphere.

### -param box [in, ref]

The BoundingBox the new BoundingSphere should contain.

## -remarks
