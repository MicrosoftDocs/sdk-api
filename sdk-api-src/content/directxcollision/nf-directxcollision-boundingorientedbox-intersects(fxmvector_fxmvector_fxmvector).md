# BoundingOrientedBox::Intersects(FXMVECTOR,FXMVECTOR,FXMVECTOR)


## -description

Tests the <a href="/windows/win32/api/directxcollision/ns-directxcollision-boundingorientedbox">BoundingOrientedBox</a> for intersection with a triangle.

## -parameters

### -param V0 [in]

A vector describing the triangle.

### -param V1 [in]

A vector describing the triangle.

### -param V2 [in]

A vector describing the triangle.

## -returns

A boolean value indicating whether the <a href="/windows/win32/api/directxcollision/ns-directxcollision-boundingorientedbox">BoundingOrientedBox</a> intersects the triangle.

## -remarks
