# BoundingSphere::operator-assign(BoundingSphere &&)


## -description

Initializes the BoundingSphere with values from a specified BoundingSphere.

## -parameters

### -param unnamedParam1 [in, ref]

The BoundingSphere to copy values from.

## -returns

The BoundingSphere.

## -remarks
