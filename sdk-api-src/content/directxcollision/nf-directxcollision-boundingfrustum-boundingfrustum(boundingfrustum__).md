# BoundingFrustum::BoundingFrustum(BoundingFrustum &&)


## -description

Creates an instance of [BoundingFrustum](./ns-directxcollision-boundingfrustum.md).

## -parameters

### -param unnamedParam1

A **BoundingFrustum** to use to initialize the new **BoundingFrustum**W.
