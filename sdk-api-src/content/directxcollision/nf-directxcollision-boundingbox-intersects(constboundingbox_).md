# BoundingBox::Intersects(const BoundingBox &)


## -description


Tests the BoundingBox for intersection with another BoundingBox.


## -parameters




### -param box [in, ref]

The BoundingBox to test for intersection.


## -returns



A bool value indicating whether the BoundingBox intersects the specified BoundingBox.




## -remarks
