# IConfigAsfWriter2::GetParam


## -description



The <code>GetParam</code> method retrieves the current value of the specified filter configuration parameter.




## -parameters




### -param dwParam [in]

Specifies the parameter to retrieve, as a member of the <a href="https://docs.microsoft.com/previous-versions/windows/desktop/legacy/dd758054(v=vs.85)">_AM_ASFWRITERCONFIG_PARAM</a> enumeration.


### -param pdwParam1 [out]

Receives the value of the parameter specified in <i>dwParam</i>.


### -param pdwParam2 [out]

Reserved. Must be zero.


## -returns



If the method succeeds, it returns S_OK. If it fails, it returns an <b>HRESULT</b> error code.
