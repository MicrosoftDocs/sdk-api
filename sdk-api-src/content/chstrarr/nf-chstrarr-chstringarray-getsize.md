# CHStringArray::GetSize


## -description


<p class="CCE_Message">[The <a href="https://msdn.microsoft.com/en-us/library/Aa385304(v=VS.85).aspx">CHStringArray</a> class 
    is part of the WMI Provider Framework which is now considered in final state, and no further development, 
    enhancements, or updates will be available for non-security related issues affecting these libraries. The 
    <a href="https://msdn.microsoft.com/en-us/library/JJ152383(v=VS.85).aspx">MI APIs</a> should be used for all new 
    development.]

The <b>GetSize</b> method gets the size of the array. Because indexes are zero-based, the size is one greater than the largest index.


## -parameters






## -returns



If the <b>GetSize</b> method is successful, it returns the number of elements in the array.
