# DXVA2_AYUVSample16 structure


## -description



Defines a 16-bit AYUV pixel value.




## -struct-fields




### -field Cr

Contains the Cr chroma value (also called V).


### -field Cb

Contains the Cb chroma value (also called U).


### -field Y

Contains the luma value.


### -field Alpha

Contains the alpha value.
