# Rect::GetRight


## -description


The <b>Rect::GetRight</b> method gets the x-coordinate of the right edge of the rectangle.


## -parameters






## -returns



Type: <strong>Type: <b>INT</b>
</strong>

This method returns the x-coordinate of the right edge of the rectangle.
