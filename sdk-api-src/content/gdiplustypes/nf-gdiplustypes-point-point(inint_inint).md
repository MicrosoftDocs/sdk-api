# Point::Point(IN INT,IN INT)


## -description


Creates a <a href="https://msdn.microsoft.com/8bf4d566-b061-4102-8307-218431e286f8">Point</a> object using two integers to initialize the 
			<b>X</b> and 
			<b>Y</b> data members.


## -parameters




### -param x [in]

Type: <b>INT</b>

Integer that specifies the x-coordinate of the <a href="https://msdn.microsoft.com/8bf4d566-b061-4102-8307-218431e286f8">Point</a> object. 


### -param y [in]

Type: <b>INT</b>

Integer that specifies the y-coordinate of the <a href="https://msdn.microsoft.com/8bf4d566-b061-4102-8307-218431e286f8">Point</a> object.
