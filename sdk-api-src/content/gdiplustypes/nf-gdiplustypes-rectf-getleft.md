# RectF::GetLeft


## -description


The <b>RectF::GetLeft</b> method gets the x-coordinate of the left edge of the rectangle.


## -parameters






## -returns



Type: <b>REAL</b>

This method returns the x-coordinate of the left edge of the rectangle.
