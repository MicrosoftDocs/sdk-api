# RectF::GetBottom


## -description

The <b>RectF::GetBottom</b> method gets the y-coordinate of the bottom edge of the rectangle.



## -returns

Type: <b>REAL</b>

This method returns the y-coordinate of the bottom edge of the rectangle.
