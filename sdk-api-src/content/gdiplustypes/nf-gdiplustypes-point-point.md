# Point::Point


## -description


Creates a <a href="https://docs.microsoft.com/windows/desktop/api/gdiplustypes/nl-gdiplustypes-point">Point</a> object and initializes the 
			<b>X</b> and 
			<b>Y</b> data members to zero. This is the default constructor.
