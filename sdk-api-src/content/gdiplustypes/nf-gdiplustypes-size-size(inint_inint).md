# Size::Size(IN INT,IN INT)


## -description


Creates a <a href="https://docs.microsoft.com/windows/desktop/api/gdiplustypes/nl-gdiplustypes-size">Size</a> object and initializes its 
			<b>Width</b> and 
			<b>Height</b> data members.


## -parameters




### -param width [in]

Type: <b>INT</b>

Integer that specifies the initial value of the 
					<b>Width</b> data member. 


### -param height [in]

Type: <b>INT</b>

Integer that specifies the initial value of the 
					<b>Height</b> data member.
