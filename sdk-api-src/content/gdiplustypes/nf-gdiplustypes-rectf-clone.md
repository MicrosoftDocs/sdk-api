# RectF::Clone


## -description


The <b>RectF::Clone</b> method creates a new 
			<a href="https://msdn.microsoft.com/6821442b-d352-48cb-a48a-839105a8c36a">RectF</a> object and initializes it with the contents of this 
			<b>RectF</b> object.


## -parameters






## -returns



Type: <strong>Type: <b><a href="https://msdn.microsoft.com/6821442b-d352-48cb-a48a-839105a8c36a">RectF</a>*</b>
</strong>

This method returns a pointer to the new 
						<a href="https://msdn.microsoft.com/6821442b-d352-48cb-a48a-839105a8c36a">RectF</a> object.
