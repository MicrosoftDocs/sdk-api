# Rect::GetBottom


## -description

The <b>Rect::GetBottom</b> method gets the y-coordinate of the bottom edge of the rectangle.

## -parameters

## -returns

Type: <b>INT</b>

This method returns the y-coordinate of the bottom edge of the rectangle.
