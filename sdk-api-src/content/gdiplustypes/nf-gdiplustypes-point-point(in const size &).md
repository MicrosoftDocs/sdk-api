# Point::Point(IN const Size &)


## -description


Creates a <a href="https://msdn.microsoft.com/en-us/library/ms534487(v=VS.85).aspx">Point</a> object using a 
			<a href="https://msdn.microsoft.com/d5be390d-11c7-47e3-8cd0-335fb6b031fd">Size</a> object to initialize the 
			<b>X</b> and 
			<b>Y</b> data members.


## -parameters




### -param size [in, ref]

Type: <b>const <a href="https://msdn.microsoft.com/d5be390d-11c7-47e3-8cd0-335fb6b031fd">Size</a></b>

Reference to a 
					<a href="https://msdn.microsoft.com/d5be390d-11c7-47e3-8cd0-335fb6b031fd">Size</a> object whose 
					<b>Width</b> data member specifies the 
					<b>X</b> data member of this <a href="https://msdn.microsoft.com/en-us/library/ms534487(v=VS.85).aspx">Point</a> object, and whose 
					<b>Height</b> data member specifies the 
					<b>Y</b> data member of this <b>Point</b> object.
