# SizeF::SizeF(IN REAL,IN REAL)


## -description

Creates a <a href="/windows/desktop/api/gdiplustypes/nl-gdiplustypes-sizef">SizeF</a> object and initializes its 
			<b>Width</b> and 
			<b>Height</b> data members.

## -parameters

### -param width [in]

Type: <b>REAL</b>

Real number that specifies the initial value of the 
					<b>Width</b> data member.

### -param height [in]

Type: <b>REAL</b>

Real number that specifies the initial value of the 
					<b>Height</b> data member.
