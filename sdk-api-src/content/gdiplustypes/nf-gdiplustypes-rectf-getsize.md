# RectF::GetSize


## -description

The <b>RectF::GetSize</b> method gets the width and height of this rectangle.

## -parameters

### -param size [out]

Type: <b>SizeF*</b>

Pointer to a <a href="/windows/desktop/api/gdiplustypes/nl-gdiplustypes-sizef">SizeF</a> object that receives the width and height of this rectangle.
