# RectF::GetSize


## -description


The <b>RectF::GetSize</b> method gets the width and height of this rectangle.


## -parameters




### -param size [out]

Type: <b>SizeF*</b>

Pointer to a <a href="https://msdn.microsoft.com/en-us/library/ms534506(v=VS.85).aspx">SizeF</a> object that receives the width and height of this rectangle. 


## -returns



This method does not return a value.
