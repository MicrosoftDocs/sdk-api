# Rect::GetLocation


## -description


The <b>Rect::GetLocation</b> method gets the coordinates of the upper-left corner of the rectangle.


## -parameters




### -param point [out]

Type: <b>Point*</b>

Pointer to a <a href="https://msdn.microsoft.com/library/windows/hardware/ff569161">Point</a> object that receives the coordinates of the upper-left corner. 


## -returns



This method does not return a value.
