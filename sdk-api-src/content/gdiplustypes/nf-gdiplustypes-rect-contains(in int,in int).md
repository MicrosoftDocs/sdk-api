# Rect::Contains(IN INT,IN INT)


## -description


The <b>Rect::Contains</b> method determines whether the point (
			<i>x</i>, 
			<i>y</i>) is inside this rectangle.


## -parameters




### -param x [in]

Type: <b>INT</b>

Integer that specifies the x-coordinate of the point to be tested. 


### -param y [in]

Type: <b>INT</b>

Integer that specifies the y-coordinate of the point to be tested. 


## -returns



Type: <strong>Type: <b>BOOL</b>
</strong>

If the point (
						<i>x</i>, 
						<i>y</i>) is inside this rectangle, this method returns <b>TRUE</b>; otherwise, it returns <b>FALSE</b>.
