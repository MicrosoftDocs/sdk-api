# RectF::GetBounds


## -description

The <b>RectF::GetBounds</b> method makes a copy of this rectangle.

## -parameters

### -param rect [out]

Type: <b>RectF*</b>

Pointer to a 
					<a href="/windows/desktop/api/gdiplustypes/nl-gdiplustypes-rectf">RectF</a> object that receives the copy of this rectangle.
