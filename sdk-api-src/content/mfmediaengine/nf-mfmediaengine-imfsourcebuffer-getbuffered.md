# IMFSourceBuffer::GetBuffered


## -description

Gets the buffered time range.

## -parameters

### -param ppBuffered [out]

The buffered time range.

## -returns

If this method succeeds, it returns <b>S_OK</b>. Otherwise, it returns an <b>HRESULT</b> error code.
