# IMFTimedText::IsInBandEnabled


## -description

Determines whether inband mode is enabled.



## -returns

Type: <b>BOOL</b>

Returns whether inband mode is enabled. If <b>TRUE</b>, inband mode is enabled. If <b>FALSE</b>, inband mode is disabled.
