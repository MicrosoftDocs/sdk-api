# IMFTimedText::SetCueTimeOffset


## -description

Sets the offset to the cue time.

## -parameters

### -param offset [in]

Type: <b>double</b>

The offset to the cue time.

## -returns

Type: <b>HRESULT</b>

If this method succeeds, it returns <b>S_OK</b>. Otherwise, it returns an <b>HRESULT</b> error code.
