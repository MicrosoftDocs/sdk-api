# IMFMediaEngineSupportsSourceTransfer::ShouldTransferSource


## -description


Specifies wether or not the source should be transferred.


## -parameters




### -param pfShouldTransfer [out]

<b>true</b> if the source should be transferred; otherwise, <b>false</b>.


## -returns



If this method succeeds, it returns <b xmlns:loc="http://microsoft.com/wdcml/l10n">S_OK</b>. Otherwise, it returns an <b xmlns:loc="http://microsoft.com/wdcml/l10n">HRESULT</b> error code.
