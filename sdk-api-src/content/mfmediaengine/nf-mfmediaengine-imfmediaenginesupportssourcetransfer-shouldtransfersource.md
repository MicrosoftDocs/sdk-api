# IMFMediaEngineSupportsSourceTransfer::ShouldTransferSource


## -description

Specifies whether or not the source should be transferred.

## -parameters

### -param pfShouldTransfer [out]

<b>true</b> if the source should be transferred; otherwise, <b>false</b>.

## -returns

If this method succeeds, it returns <b>S_OK</b>. Otherwise, it returns an <b>HRESULT</b> error code.
