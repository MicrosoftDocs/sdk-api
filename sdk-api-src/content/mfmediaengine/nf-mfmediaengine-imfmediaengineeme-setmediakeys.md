# IMFMediaEngineEME::SetMediaKeys


## -description

Sets the media keys object to use with the media engine.

## -parameters

### -param keys

The media keys.

## -returns

If this method succeeds, it returns <b>S_OK</b>. Otherwise, it returns an <b>HRESULT</b> error code.
