# IMFTimedTextCue::GetId


## -description


Gets the identifier of a timed-text cue.


## -parameters






## -returns



Type: <b>
DWORD
</b>

The identifier of a timed-text cue.




## -remarks



The identifier is retrieved by this method is dynamically generated by the system and is guaranteed to uniquely identify a cue within a single timed-text track. It is not guaranteed to be unique across tracks. If a cue already has an identifier that is provided in the text-track data format, this ID can be retrieved by calling <a href="https://msdn.microsoft.com/D5B94171-AEB0-4A7D-B596-F888B69A436D">GetOriginalId</a>.
