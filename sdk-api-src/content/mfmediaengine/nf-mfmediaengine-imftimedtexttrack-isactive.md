# IMFTimedTextTrack::IsActive


## -description

Determines whether the timed-text track is active.



## -returns

Type: <b>BOOL</b>

Returns whether the timed-text track is active. If <b>TRUE</b>, the timed-text track is active; otherwise, <b>FALSE</b>.
