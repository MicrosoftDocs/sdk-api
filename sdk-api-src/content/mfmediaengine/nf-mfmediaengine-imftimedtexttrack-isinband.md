# IMFTimedTextTrack::IsInBand


## -description


Determines whether the timed-text track is inband.


## -parameters






## -returns



Type: <b>BOOL</b>

Returns whether the timed-text track is inband. If <b>TRUE</b>, the timed-text track is inband; otherwise, <b>FALSE</b>.
