# IMFMediaEngineEx::EnableTimeUpdateTimer


## -description

Enables or disables the time update timer.

## -parameters

### -param fEnableTimer [in]

If <b>TRUE</b>, the update timer is enabled. Otherwise, the timer is disabled.

## -returns

If this method succeeds, it returns <b>S_OK</b>. Otherwise, it returns an <b>HRESULT</b> error code.
