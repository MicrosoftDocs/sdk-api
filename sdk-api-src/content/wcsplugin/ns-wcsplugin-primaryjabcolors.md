# PrimaryJabColors structure


## -description

This structure contains eight primary colors in Jab coordinates.

Yellow primary.

Green primary.

### -field cyan

Cyan primary.

### -field blue

Blue primary.

### -field magenta

Magenta primary.

### -field black

Black primary.

### -field white

White primary.
