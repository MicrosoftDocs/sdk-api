# PrimaryXYZColors structure


## -description

This structure contains eight primary colors in XYZ coordinates.

## -struct-fields

### -field red

Red primary.

### -field yellow

Yellow primary.

### -field green

Green primary.

### -field cyan

Cyan primary.

### -field blue

Blue primary.

### -field magenta

Magenta primary.

### -field black

Black primary.

### -field white

White primary.
