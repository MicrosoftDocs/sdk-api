# HrtfEnvironment enumeration


## -description

Indicates one of several stock environment types.

## -enum-fields

### -field Small:0

A small room.

### -field Medium

A medium-sized room.

### -field Large

A large, enclosed space.

### -field Outdoors

An outdoor space.
