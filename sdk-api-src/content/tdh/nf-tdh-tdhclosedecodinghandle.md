# TdhCloseDecodingHandle function


## -description


Frees any resources associated with the input decoding handle.


## -parameters




### -param Handle [in]

Type: <b>TDH_HANDLE</b>

The decoding handle to be closed.


## -returns



Type: <b>ULONG</b>

This function returns ERROR_SUCCESS on completion.
