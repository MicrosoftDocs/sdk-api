# CLSIDFromString function


## -description


Converts a string generated by the <a href="https://docs.microsoft.com/windows/desktop/api/combaseapi/nf-combaseapi-stringfromclsid">StringFromCLSID</a> function back into the original CLSID.




## -parameters




### -param lpsz [in]

The string representation of the CLSID.


### -param pclsid [out]

A pointer to the CLSID.


## -returns



This function can return the standard return value E_INVALIDARG, as well as the following values.

<tr>
<th>Return code</th>
<th>Description</th>
</tr>
<tr>
<td width="40%">
<dl>
<dt><b>NOERROR</b></dt>
</dl>
</td>
<td width="60%">
The CLSID was obtained successfully.

</td>
</tr>
<tr>
<td width="40%">
<dl>

</dl>
</td>
<td width="60%">

</td>

<tr>
<td width="40%">
<dl>
<dt><b>REGDB_E_CLASSNOTREG</b></dt>
</dl>
</td>
<td width="60%">
