# CLSIDFromString function


## -description


Converts a string generated by the <a href="https://msdn.microsoft.com/61210ebd-cbf3-4e78-b077-53d2779053eb">StringFromCLSID</a> function back into the original CLSID.




## -parameters




### -param lpsz [in]

The string representation of the CLSID.


### -param pclsid [out]

A pointer to the CLSID.


## -returns



This function can return the standard return value E_INVALIDARG, as well as the following values.

<th>Description</th>

<dl>
<dt><b>NOERROR</b></dt>
</dl>
</td>

The CLSID was obtained successfully.
