# PROPVAR_COMPARE_UNIT enumeration


## -description


These flags are associated with certain <a href="https://msdn.microsoft.com/e86cc279-826d-4767-8d96-fc8280060ea1">PROPVARIANT</a> structure comparisons.


## -enum-fields




### -field PVCU_DEFAULT

The default unit.


### -field PVCU_SECOND

The second comparison unit.


### -field PVCU_MINUTE

The minute comparison unit.


### -field PVCU_HOUR

The hour comparison unit.


### -field PVCU_DAY

The day comparison unit.


### -field PVCU_MONTH

The month comparison unit.


### -field PVCU_YEAR

The year comparison unit.
