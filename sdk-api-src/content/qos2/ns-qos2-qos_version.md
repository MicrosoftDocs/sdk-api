# QOS_VERSION structure


## -description

The <b>QOS_VERSION</b> structure indicates the version of the QOS protocol.

## -struct-fields

### -field MajorVersion

Major version of the QOS protocol.

### -field MinorVersion

Minor version of the QOS protocol.
