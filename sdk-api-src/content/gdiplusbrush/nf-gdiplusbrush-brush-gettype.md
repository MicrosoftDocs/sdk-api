# Brush::GetType


## -description


The <b>Brush::GetType</b> method gets the type of this brush.


## -parameters






## -returns



Type: <strong>Type: <b><a href="https://msdn.microsoft.com/a5d2e252-b7b9-4f8b-b796-1cba8bcc58dc">BrushType</a></b>
</strong>

This method returns the type of this brush. The value returned is one of the elements of the 
						<a href="https://msdn.microsoft.com/a5d2e252-b7b9-4f8b-b796-1cba8bcc58dc">BrushType</a> enumeration.
