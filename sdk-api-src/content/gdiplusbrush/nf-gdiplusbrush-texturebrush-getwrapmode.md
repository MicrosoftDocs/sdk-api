# TextureBrush::GetWrapMode


## -description


The <b>TextureBrush::GetWrapMode</b> method gets the wrap mode currently set for this texture brush.


## -parameters






## -returns



Type: <strong>Type: <b><a href="https://msdn.microsoft.com/library/ms534407(v=VS.85).aspx">WrapMode</a></b>
</strong>

This method returns the wrap mode currently set for this texture brush. The value returned is one of the elements of the <a href="https://msdn.microsoft.com/library/ms534407(v=VS.85).aspx">WrapMode</a> enumeration.
