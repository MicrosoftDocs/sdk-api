# TextureBrush::GetImage


## -description


The <b>TextureBrush::GetImage</b> method gets a pointer to the <a href="https://msdn.microsoft.com/en-us/library/ms534462(v=VS.85).aspx">Image</a> object that is defined by this texture brush.


## -parameters






## -returns



Type: <strong>Type: <b><a href="https://msdn.microsoft.com/en-us/library/ms534462(v=VS.85).aspx">Image</a>*</b>
</strong>

This method returns a pointer to the <a href="https://msdn.microsoft.com/en-us/library/ms534462(v=VS.85).aspx">Image</a> object that is defined by this texture brush.
