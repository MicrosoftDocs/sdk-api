# DISK_GROW_PARTITION structure


## -description


Contains information used to increase the size of a partition.This structure is used by the <a href="https://docs.microsoft.com/windows/desktop/api/winioctl/ni-winioctl-ioctl_disk_grow_partition">IOCTL_DISK_GROW_PARTITION</a> control code.


## -struct-fields




### -field PartitionNumber

The identifier of the partition to be enlarged.


### -field BytesToGrow

The number of bytes by which the partition is to be enlarged (positive value) or reduced (negative value). Note that this value is not the new size of the partition.
