# FILE_STORAGE_TIER_REGION structure


## -description


Describes a single storage tier region.


## -struct-fields




### -field TierId

Tier ID.


### -field Offset

Offset from the beginning of the volume of this region, in bytes.


### -field Length

Length of region in bytes.
