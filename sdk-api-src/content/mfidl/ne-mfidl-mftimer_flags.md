# MFTIMER_FLAGS enumeration


## -description



Contains flags for the <a href="https://msdn.microsoft.com/3b583541-6480-490d-883f-376ea95f7a98">IMFTimer::SetTimer</a> method.




## -enum-fields




### -field MFTIMER_RELATIVE

The time passed to the timer is relative to the current time. If this flag is absent, the time is expressed as an absolute clock time.
