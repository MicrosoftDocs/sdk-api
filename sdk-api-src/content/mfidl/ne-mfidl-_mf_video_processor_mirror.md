# _MF_VIDEO_PROCESSOR_MIRROR enumeration


## -description


Specifies how to flip a video image.


## -enum-fields




### -field MIRROR_NONE

Do not flip the image.


### -field MIRROR_HORIZONTAL

Flip the image horizontally.


### -field MIRROR_VERTICAL

Flip the image vertically.
