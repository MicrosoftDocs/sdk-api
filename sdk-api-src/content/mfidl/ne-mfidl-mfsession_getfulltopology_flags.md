# MFSESSION_GETFULLTOPOLOGY_FLAGS enumeration


## -description


Defines flags for the <a href="https://docs.microsoft.com/windows/desktop/api/mfidl/nf-mfidl-imfmediasession-getfulltopology">IMFMediaSession::GetFullTopology</a> method.
        


## -enum-fields




### -field MFSESSION_GETFULLTOPOLOGY_CURRENT

Get the full topology for the presentation that the Media Session is currently playing.
