# IMFTopology::GetNodeCount


## -description

Gets the number of nodes in the topology.

## -parameters

### -param pwNodes [out]

Receives the number of nodes.

## -returns

If this method succeeds, it returns <b>S_OK</b>. Otherwise, it returns an <b>HRESULT</b> error code.
