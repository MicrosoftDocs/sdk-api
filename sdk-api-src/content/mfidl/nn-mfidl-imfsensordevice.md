# IMFSensorDevice interface


## -description

Represents a sensor device that can belong to a sensor group, which is represented by the <a href="/windows/desktop/api/mfidl/nn-mfidl-imfsensorgroup">IMFSensorGroup</a> interface. The term "device" in this context could refer to a physical device, a custom media source, or a frame provider.
