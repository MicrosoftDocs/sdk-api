# IMFVideoSampleAllocator::UninitializeSampleAllocator


## -description

Releases all of the video samples that have been allocated.

<th>Description</th>

<td width="40%">
<dl>
<dt><b>S_OK</b></dt>
</dl>
</td>
<td width="60%">
The method succeeded.

</td>
</tr>
</table>
