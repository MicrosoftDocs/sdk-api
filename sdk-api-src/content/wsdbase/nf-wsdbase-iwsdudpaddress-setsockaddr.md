# IWSDUdpAddress::SetSockaddr


## -description


Sets the socket address information.


## -parameters




### -param pSockAddr [in]

Pointer to a <a href="https://docs.microsoft.com/previous-versions/windows/desktop/legacy/ms740504(v=vs.85)">SOCKADDR_STORAGE</a> structure.
