# ID3D12ShaderReflection::GetBitwiseInstructionCount


## -description


Gets the number of bitwise instructions.
        


## -parameters






## -returns



Type: <b><a href="https://docs.microsoft.com/windows/desktop/WinProg/windows-data-types">UINT</a></b>
