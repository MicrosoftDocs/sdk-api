# IActiveDesktopP::SetSafeMode


## -description


Sets or updates the Microsoft Active Desktop to safe mode.


## -parameters




### -param dwFlags

Type: <b>DWORD</b>

One of the following flags.



#### SSM_SET

Set to safe mode.



#### SSM_UPDATE

Update to safe mode.


## -returns



Type: <b>HRESULT</b>

Returns S_OK if successful, or an error code otherwise.
