# IThumbnailCapture interface


## -description


Exposes a method that obtains a thumbnail representation of an HTML wallpaper.
        
            
<div class="alert"><b>Note</b>  This interface is deprecated as of Windows 7. The feature it supported is no longer present in Windows.</div><div> </div>
