## -description

Specifies the input operations for which visual feedback should be provided. This enumeration is used by the [DwmShowContact](nf-dwmapi-dwmshowcontact.md) function.

## -enum-fields

### -field DWMSC_DOWN

Down.

### -field DWMSC_UP

Up.

### -field DWMSC_DRAG

Drag.

### -field DWMSC_HOLD

Hold.

### -field DWMSC_PENBARREL

Pen barrel.

### -field DWMSC_NONE

None.

### -field DWMSC_ALL

All.
