# AdjustableArrowCap::IsFilled


## -description

The <b>AdjustableArrowCap::IsFilled</b> method determines whether the arrow cap is filled.



## -returns

Type: <b>BOOL</b>

If the arrow cap is filled, this method returns <b>TRUE</b>; otherwise, it returns <b>FALSE</b>.
