# IMSVidCtl::get_Window


## -description


The <b>get_Window</b> method retrieves the window associated with the Video Control.


## -parameters




### -param phwnd [out]

Receives a handle to the window.


## -returns



Returns an <b>HRESULT</b> value. Possible values include the following.
