# Image::GetFlags


## -description


The <b>Image::GetFlags</b> method gets a set of flags that indicate certain attributes of this 
			<a href="https://msdn.microsoft.com/library/ms534462(v=VS.85).aspx">Image</a> object.


## -parameters






## -returns



Type: <strong>Type: <b>UINT</b>
</strong>

This method returns a value that holds a set of single-bit flags. The flags are defined in the <a href="https://msdn.microsoft.com/library/ms534136(v=VS.85).aspx">ImageFlags</a> enumeration.
