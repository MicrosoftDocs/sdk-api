# Region::Region(IN HRGN)


## -description

Creates a region that is identical to the region that is specified by a handle to a Windows Graphics Device Interface (GDI) region.

## -parameters

### -param hRgn [in]

Type: <b>HRGN</b>

Handle to an existing GDI region.
