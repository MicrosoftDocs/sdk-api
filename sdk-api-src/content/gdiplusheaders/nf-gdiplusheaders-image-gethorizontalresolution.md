# Image::GetHorizontalResolution


## -description


The <b>Image::GetHorizontalResolution</b> method gets the horizontal resolution, in dots per inch, of this image.


## -parameters






## -returns



Type: <b>REAL</b>

This method returns the horizontal resolution, in dots per inch, of this image.
