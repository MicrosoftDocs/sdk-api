# Image::GetPixelFormat


## -description


The <b>Image::GetPixelFormat</b> method gets the pixel format of this 
			<a href="https://msdn.microsoft.com/library/ms534462(v=VS.85).aspx">Image</a> object.


## -parameters






## -returns



Type: <strong>Type: <b>PixelFormat</b>
</strong>

This method returns an integer that indicates the pixel format of this 
						<a href="https://msdn.microsoft.com/library/ms534462(v=VS.85).aspx">Image</a> object. The 
						<b>PixelFormat</b> data type and constants that represent various pixel formats are defined in Gdipluspixelformats.h. For more information about pixel format constants, see <a href="https://msdn.microsoft.com/library/ms534412(v=VS.85).aspx">Image Pixel Format Constants</a>.
