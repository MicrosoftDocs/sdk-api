# ITTerminalManager interface


## -description

The 
<b>ITTerminalManager</b> interface is used by the MSP to create dynamic terminals.

The 
<a href="/windows/desktop/api/termmgr/nn-termmgr-itterminalmanager2">ITTerminalManager2</a> interface exposes methods that retrieve information about pluggable terminal classes registered in the current system. 
<b>ITTerminalManager2</b> is derived from the 
<b>ITTerminalManager</b> interface.
