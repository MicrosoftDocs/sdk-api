# ITPluggableTerminalSuperclassRegistration::get_Name


## -description


The 
<b>get_Name</b> method gets the friendly name for the terminal superclass.


## -parameters




### -param pName [out]

 Pointer to a <b>BSTR</b> representation of the friendly name. The <b>BSTR</b> is allocated using 
<a href="https://docs.microsoft.com/previous-versions/windows/desktop/api/oleauto/nf-oleauto-sysallocstring">SysAllocString</a>. The <b>BSTR</b> argument should be deallocated by the client.
