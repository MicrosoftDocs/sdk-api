# IBitmapData interface


## -description

Represents an image associated with a node in the visual tree.

## -inheritance

The <b xmlns:loc="http://microsoft.com/wdcml/l10n">IBitmapData</b> interface inherits from the <a href="/windows/desktop/api/unknwn/nn-unknwn-iunknown">IUnknown</a> interface. <b>IBitmapData</b> also has these types of members:
<ul>
<li><a href="https://docs.microsoft.com/">Methods</a></li>
</ul>
