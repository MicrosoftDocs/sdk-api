# IEnumGuideDataProperties::Skip


## -description



Not implemented for this release.
        



The <b>Skip</b> method skips over a specified number of items in the collection.


## -parameters




### -param celt [in]

Specifies the number of items to skip.


## -returns



Returns E_NOTIMPL.
