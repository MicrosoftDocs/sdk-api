# ID3D10Asynchronous::GetDataSize


## -description

## -parameters






## -returns



Type: <b><a href="https://msdn.microsoft.com/4553cafc-450e-4493-a4d4-cb6e2f274d46">UINT</a></b>

Size of the data (in bytes) that is output when calling GetData.
