# ID3D10RasterizerState::GetDesc


## -description


Get the properties of a rasterizer-state object.


## -parameters




### -param pDesc [out]

Type: <b><a href="https://msdn.microsoft.com/library/Bb172408(v=VS.85).aspx">D3D10_RASTERIZER_DESC</a>*</b>

Pointer to a rasterizer-state description (see <a href="https://msdn.microsoft.com/library/Bb172408(v=VS.85).aspx">D3D10_RASTERIZER_DESC</a>).


## -returns



Returns nothing.
