# ID3D10DepthStencilState::GetDesc


## -description

Get the depth-stencil state.

## -parameters

### -param pDesc [out]

Type: <b><a href="/windows/desktop/api/d3d10/ns-d3d10-d3d10_depth_stencil_desc">D3D10_DEPTH_STENCIL_DESC</a>*</b>

A pointer to the depth-stencil state (see <a href="/windows/desktop/api/d3d10/ns-d3d10-d3d10_depth_stencil_desc">D3D10_DEPTH_STENCIL_DESC</a>).
