# IDWriteFont2::IsColorFont


## -description

Enables determining whether a color rendering path is potentially necessary.



## -returns

Type: <b>BOOL</b>

Returns <b>TRUE</b> if the font has color information (COLR and CPAL tables); otherwise <b>FALSE</b>.
