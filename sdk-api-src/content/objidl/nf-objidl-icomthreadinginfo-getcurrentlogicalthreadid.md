# IComThreadingInfo::GetCurrentLogicalThreadId


## -description


Retrieves the GUID of the thread in which the caller is executing.


## -parameters




### -param pguidLogicalThreadId [out]

A pointer to the GUID of the caller's thread.


## -returns
