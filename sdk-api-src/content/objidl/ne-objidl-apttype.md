# APTTYPE enumeration


## -description


Specifies different types of apartments.


## -enum-fields




### -field APTTYPE_CURRENT

The current thread.


### -field APTTYPE_STA

A single-threaded apartment.


### -field APTTYPE_MTA

A multithreaded apartment.


### -field APTTYPE_NA

A neutral apartment.


### -field APTTYPE_MAINSTA

The main single-threaded apartment.
