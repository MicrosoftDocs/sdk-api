# IForegroundTransfer::AllowForegroundTransfer


## -description


Yields the foreground window to the COM server process.


## -parameters




### -param lpvReserved [in]

This parameter is reserved and must be <b>NULL</b>.


## -returns

</dl>
</td>
<td width="60%">
