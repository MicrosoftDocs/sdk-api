# MIB_OPAQUE_QUERY structure


## -description


The 
<b>MIB_OPAQUE_QUERY</b> structure contains information for a MIB opaque query.


## -struct-fields




### -field dwVarId

The identifier of the MIB object to query.


### -field rgdwVarIndex

The index of the MIB object to query.
