# APPLY_SNAPSHOT_VHDSET_VERSION enumeration


## -description

Enumerates the possible versions for parameters for the 
    <a href="/windows/win32/api/virtdisk/nf-virtdisk-applysnapshotvhdset">ApplySnapshotVhdSet</a> function.

## -enum-fields

### -field APPLY_SNAPSHOT_VHDSET_VERSION_UNSPECIFIED:0

Not Supported.

### -field APPLY_SNAPSHOT_VHDSET_VERSION_1:1

The Version1 member structure will be used.
