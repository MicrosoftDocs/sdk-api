# IWICDevelopRaw::GetRotation


## -description

Gets the current rotation angle.

## -parameters

### -param pRotation [out]

Type: <b>double*</b>

A pointer that receives the current rotation angle.

## -returns

Type: <b>HRESULT</b>

If this method succeeds, it returns <b>S_OK</b>. Otherwise, it returns an <b>HRESULT</b> error code.
