# WICRect structure


## -description


Represents a rectangle for Windows Imaging Component (WIC) API.


## -struct-fields




### -field X

Type: <b>INT</b>

The horizontal coordinate of the rectangle.


### -field Y

Type: <b>INT</b>

The vertical coordinate of the rectangle.


### -field Width

Type: <b>INT</b>

The width of the rectangle.


### -field Height

Type: <b>INT</b>

The height of the rectangle.
