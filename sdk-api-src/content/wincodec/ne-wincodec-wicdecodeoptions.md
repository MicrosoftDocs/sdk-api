# WICDecodeOptions enumeration


## -description

Specifies decode options.

## -enum-fields

### -field WICDecodeMetadataCacheOnDemand:0

Cache metadata when needed.

### -field WICDecodeMetadataCacheOnLoad:0x1

Cache metadata when decoder is loaded.
