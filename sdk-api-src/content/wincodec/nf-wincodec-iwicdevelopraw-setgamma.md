# IWICDevelopRaw::SetGamma


## -description

Sets the desired gamma value.

## -parameters

### -param Gamma [in]

Type: <b>double</b>

The desired gamma value.

## -returns

Type: <b>HRESULT</b>

If this method succeeds, it returns <b>S_OK</b>. Otherwise, it returns an <b>HRESULT</b> error code.
