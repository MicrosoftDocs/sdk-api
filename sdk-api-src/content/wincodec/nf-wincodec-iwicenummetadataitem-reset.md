# IWICEnumMetadataItem::Reset


## -description

Resets the current position to the beginning of the enumeration.



## -returns

Type: <b>HRESULT</b>

If this method succeeds, it returns <b>S_OK</b>. Otherwise, it returns an <b>HRESULT</b> error code.
