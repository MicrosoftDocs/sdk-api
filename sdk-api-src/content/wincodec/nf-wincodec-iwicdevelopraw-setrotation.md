# IWICDevelopRaw::SetRotation


## -description

Sets the desired rotation angle.

## -parameters

### -param Rotation [in]

Type: <b>double</b>

The desired rotation angle.

## -returns

Type: <b>HRESULT</b>

If this method succeeds, it returns <b>S_OK</b>. Otherwise, it returns an <b>HRESULT</b> error code.
