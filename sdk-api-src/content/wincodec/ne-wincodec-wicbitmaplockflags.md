# WICBitmapLockFlags enumeration


## -description


Specifies access to an <a href="https://docs.microsoft.com/windows/desktop/api/wincodec/nn-wincodec-iwicbitmap">IWICBitmap</a>.


## -enum-fields




### -field WICBitmapLockRead

A read access lock.


### -field WICBitmapLockWrite

A write access lock.
