# NET_FW_ACTION enumeration


## -description


The <b>NET_FW_ACTION</b> enumerated type specifies the action for a rule or default setting.


## -enum-fields




### -field NET_FW_ACTION_BLOCK

Block traffic.


### -field NET_FW_ACTION_ALLOW

Allow traffic.


### -field NET_FW_ACTION_MAX

Maximum traffic.
