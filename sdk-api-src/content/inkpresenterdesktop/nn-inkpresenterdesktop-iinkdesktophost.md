# IInkDesktopHost interface


## -description


An <b>IInkDesktopHost</b> object enables ink input, processing, and rendering through the creation of an app thread to host  an <a href="https://msdn.microsoft.com/6d175981-6379-4a61-84b0-8b08274bc3a3">IInkPresenterDesktop</a> object and  insert it into the app's  <a href="https://msdn.microsoft.com/40e2d02b-77e8-425f-ac5e-3dcddef08173">DirectComposition</a> visual tree.
