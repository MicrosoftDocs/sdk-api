# ISettingsNamespace::Save


## -description


Updates the settings namespace to persistent and visible.


## -parameters




### -param PushSettings [in]

 Not used. A flag that controls whether to transfer settings to the registry or to an initialization file.


### -param Result [out]

 A pointer to an <a href="https://docs.microsoft.com/previous-versions/windows/desktop/api/wcmconfig/nn-wcmconfig-isettingsresult">ISettingsResult</a> object that contains any error that may have occurred while saving the namespace.


## -returns



This method returns an HRESULT value. <b>S_OK</b> indicates success.
