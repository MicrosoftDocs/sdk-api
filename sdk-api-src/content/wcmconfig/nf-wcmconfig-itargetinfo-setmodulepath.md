# ITargetInfo::SetModulePath


## -description


Sets the module path for the offline installation location.


## -parameters




### -param Module [in]

The name of the module.


### -param Path [in]

The module path.


## -returns



This method returns an HRESULT value. <b>S_OK</b> indicates success.
