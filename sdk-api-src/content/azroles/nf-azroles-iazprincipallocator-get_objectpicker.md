# IAzPrincipalLocator::get_ObjectPicker


## -description


The <b>ObjectPicker</b> method gets a pointer to the <a href="https://docs.microsoft.com/windows/desktop/api/azroles/nn-azroles-iazobjectpicker">IAzObjectPicker</a> interface associated with this <a href="https://docs.microsoft.com/windows/desktop/api/azroles/nn-azroles-iazprincipallocator">IAzPrincipalLocator</a> object.

This property is read-only.
