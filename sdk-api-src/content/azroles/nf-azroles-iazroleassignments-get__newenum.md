# IAzRoleAssignments::get__NewEnum


## -description

The <b>_NewEnum</b> property retrieves an <a href="/previous-versions/windows/desktop/api/oaidl/nn-oaidl-ienumvariant">IEnumVARIANT</a> interface on an object that can be used to enumerate the <a href="/windows/desktop/api/azroles/nn-azroles-iazroleassignments">IAzRoleAssignments</a> collection. This property is hidden within Visual Basic and Visual Basic Scripting Edition (VBScript).

This property is read-only.

## -parameters
