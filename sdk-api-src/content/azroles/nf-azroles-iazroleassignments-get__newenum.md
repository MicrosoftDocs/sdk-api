# IAzRoleAssignments::get__NewEnum


## -description


The <b>_NewEnum</b> property retrieves an <a href="139e3c93-faef-4003-9079-e0e94494db3e">IEnumVARIANT</a> interface on an object that can be used to enumerate the <a href="https://msdn.microsoft.com/d38fd7e0-6d0b-4b68-b6e5-f7adc2cfef47">IAzRoleAssignments</a> collection. This property is hidden within Visual Basic and Visual Basic Scripting Edition (VBScript).

This property is read-only.
