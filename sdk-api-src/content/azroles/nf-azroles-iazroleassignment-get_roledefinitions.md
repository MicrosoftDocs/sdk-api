# IAzRoleAssignment::get_RoleDefinitions


## -description

The <b>RoleDefinitions</b> property retrieves a collection of the <a href="/windows/desktop/api/azroles/nn-azroles-iazroledefinition">IAzRoleDefinition</a> objects associated with this <a href="/windows/desktop/api/azroles/nn-azroles-iazroleassignment">IAzRoleAssignment</a> object.

This property is read-only.

## -parameters
