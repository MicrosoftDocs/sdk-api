# IAzTask::put_IsRoleDefinition


## -description

The <b>IsRoleDefinition</b> property sets or retrieves a value that indicates whether the task is a role definition.

This property is read/write.

## -parameters

## -remarks

This property represents a user interface abstraction and does not affect the functionality of the task.
