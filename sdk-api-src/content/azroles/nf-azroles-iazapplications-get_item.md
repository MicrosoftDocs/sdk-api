# IAzApplications::get_Item


## -description

The <b>Item</b> property retrieves the <a href="/windows/desktop/api/azroles/nn-azroles-iazapplication">IAzApplication</a> object at the specified index into the <a href="/windows/desktop/api/azroles/nn-azroles-iazapplications">IAzApplications</a> collection.

This property is read-only.

## -parameters
