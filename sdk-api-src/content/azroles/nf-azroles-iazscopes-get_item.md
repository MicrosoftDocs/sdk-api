# IAzScopes::get_Item


## -description

The <b>Item</b> property retrieves the <a href="/windows/desktop/api/azroles/nn-azroles-iazscope">IAzScope</a> object at the specified index into the <a href="/windows/desktop/api/azroles/nn-azroles-iazscopes">IAzScopes</a> collection.

This property is read-only.

## -parameters
