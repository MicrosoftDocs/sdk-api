# IAzObjectPicker::get_Name


## -description


The <b>Name</b> property gets the name of the <a href="https://msdn.microsoft.com/767f30c9-6071-4f04-876d-b8b2392e650c">IAzObjectPicker</a> object.

This property is read-only.
