# IAzRoleAssignments::get_Count


## -description


The <b>Count</b> property retrieves the number of <a href="https://msdn.microsoft.com/d38fd7e0-6d0b-4b68-b6e5-f7adc2cfef47">IAzRoleAssignments</a> objects in the collection.

This property is read-only.
