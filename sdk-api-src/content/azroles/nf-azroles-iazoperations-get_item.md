# IAzOperations::get_Item


## -description

The <b>Item</b> property retrieves the <a href="/windows/desktop/api/azroles/nn-azroles-iazoperation">IAzOperation</a> object at the specified index into the <a href="/windows/desktop/api/azroles/nn-azroles-iazoperations">IAzOperations</a> collection.

This property is read-only.

## -parameters
