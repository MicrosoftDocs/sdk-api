# IAzRoleDefinition::get_RoleDefinitions


## -description


The <b>RoleDefinitions</b> property retrieves a collection of the <a href="https://msdn.microsoft.com/d951f5cc-85da-4898-a70f-9e50ab66ade5">IAzRoleDefinition</a> objects associated with this <b>IAzRoleDefinition</b> object.

This property is read-only.
