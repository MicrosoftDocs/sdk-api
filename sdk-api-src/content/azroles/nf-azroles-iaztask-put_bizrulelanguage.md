# IAzTask::put_BizRuleLanguage


## -description

The <b>BizRuleLanguage</b> property sets or retrieves the scripting language in which the business rule (BizRule) is implemented.

This property is read/write.

## -parameters

## -remarks

This property must be set before the <a href="/windows/desktop/api/azroles/nf-azroles-iaztask-get_bizrule">BizRule</a> property is set.
