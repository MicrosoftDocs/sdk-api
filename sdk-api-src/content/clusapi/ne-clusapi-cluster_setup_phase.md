# CLUSTER_SETUP_PHASE enumeration


## -description

Used by the 
    <a href="/windows/desktop/api/clusapi/nc-clusapi-pcluster_setup_progress_callback">ClusterSetupProgressCallback</a> function 
    to identify the current phase of the cluster setup process.

## -enum-fields

### -field ClusterSetupPhaseInitialize:1

Initialize cluster setup.

### -field ClusterSetupPhaseValidateNodeState:100

Validate cluster nodes.

### -field ClusterSetupPhaseValidateNetft:102

Validate cluster networks.

### -field ClusterSetupPhaseValidateClusDisk:103

Validate cluster disks.

### -field ClusterSetupPhaseConfigureClusSvc:104

Configure cluster service.

### -field ClusterSetupPhaseStartingClusSvc:105

Start  cluster service.

### -field ClusterSetupPhaseQueryClusterNameAccount:106

Query cluster name.

### -field ClusterSetupPhaseValidateClusterNameAccount:107

Validate cluster name.

### -field ClusterSetupPhaseCreateClusterAccount:108

Create cluster account.

### -field ClusterSetupPhaseConfigureClusterAccount:109

Configure cluster account.

### -field ClusterSetupPhaseFormingCluster:200

Form the cluster.

### -field ClusterSetupPhaseAddClusterProperties:201

Add properties to cluster.

### -field ClusterSetupPhaseCreateResourceTypes:202

Create resource types.

### -field ClusterSetupPhaseCreateGroups:203

Create resource groups.

### -field ClusterSetupPhaseCreateIPAddressResources:204

Create IP address resources.

### -field ClusterSetupPhaseCreateNetworkName:205

Create network name.

### -field ClusterSetupPhaseClusterGroupOnline:206

Bring cluster groups online.

### -field ClusterSetupPhaseGettingCurrentMembership:300

Get current cluster membership.

### -field ClusterSetupPhaseAddNodeToCluster:301

Add node to cluster membership.

### -field ClusterSetupPhaseNodeUp:302

Start node.

### -field ClusterSetupPhaseMoveGroup:400

Move group to another node.

### -field ClusterSetupPhaseDeleteGroup:401

Delete group from cluster.

### -field ClusterSetupPhaseCleanupCOs:402

Clean up offline group.

### -field ClusterSetupPhaseOfflineGroup:403

Move group offline.

### -field ClusterSetupPhaseEvictNode:404

Remove a node from the cluster.

### -field ClusterSetupPhaseCleanupNode:405

Return node to pre-clustered state.

### -field ClusterSetupPhaseCoreGroupCleanup:406

Return core resource group to pre-clustered state.

### -field ClusterSetupPhaseFailureCleanup:999

Return failed resource to pre-clustered state.
