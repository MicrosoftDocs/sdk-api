# CLUS_DNN_SODAFS_CLONE_STATUS structure


## -description


Represents the status of a Scale-Out File Server clone.


## -struct-fields




### -field NodeId

The node ID of the clone.


### -field Status

A <a href="https://msdn.microsoft.com/en-us/library/Bb309168(v=VS.85).aspx">CLUSTER_RESOURCE_STATE</a> enumeration value that specifies the status of the clone.
