# CLUSCTL_GET_CONTROL_FUNCTION macro


## -description

Extracts the operation code from a control code. For more information on the various codes that comprise a control code, see  <a href="/previous-versions/windows/desktop/mscs/control-code-architecture">Control Code Architecture</a>.

## -parameters

### -param ControlCode

The control code to be analyzed.
