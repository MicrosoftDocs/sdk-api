# IUIAnimationVariableCurveChangeHandler2::OnCurveChanged


## -description


Handles events that occur when the animation curve of an animation variable changes.


## -parameters




### -param variable [in]

The animation variable for which the animation curve has been updated.


## -returns



If this method succeeds, it returns S_OK. Otherwise, it returns an  <b>HRESULT</b> error code. See <a href="https://docs.microsoft.com/windows/desktop/UIAnimation/uianimation-error-codes">Windows Animation Error Codes</a> for a list of error codes.
