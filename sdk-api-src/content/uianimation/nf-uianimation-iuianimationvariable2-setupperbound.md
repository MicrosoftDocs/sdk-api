# IUIAnimationVariable2::SetUpperBound


## -description


Sets the upper bound (ceiling) for the value of the animation variable. The value of the animation variable should not rise above the specified value.


## -parameters




### -param bound [in]

The upper bound for the value of the animation variable.


## -returns



Returns S_OK if successful; otherwise an <b>HRESULT</b> error code. See <a href="https://docs.microsoft.com/windows/desktop/UIAnimation/uianimation-error-codes">Windows Animation Error Codes</a> for a list of error codes.
