# IUIAnimationStoryboard interface


## -description

Defines a storyboard, which contains a group of transitions
      that are synchronized relative to one another.

## -inheritance

The <b xmlns:loc="http://microsoft.com/wdcml/l10n">IUIAnimationStoryboard</b> interface inherits from the <a href="/windows/desktop/api/unknwn/nn-unknwn-iunknown">IUnknown</a> interface. <b>IUIAnimationStoryboard</b> also has these types of members:

</ul>

         along with
