# DS_KCC_TASKID enumeration


## -description


Specifies tasks that Knowledge Consistency Checker (KCC) can execute.


## -enum-fields




### -field DS_KCC_TASKID_UPDATE_TOPOLOGY

Dynamically adjusts the data replication topology of a network.
