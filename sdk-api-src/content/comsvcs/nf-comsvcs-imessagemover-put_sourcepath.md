# IMessageMover::put_SourcePath


## -description


Sets the path of the source (input) queue.


## -parameters




### -param newVal [in]

The path.


## -returns



This method can return the standard return values E_INVALIDARG, E_OUTOFMEMORY, E_UNEXPECTED, E_FAIL, and S_OK.
