# ICOMLBArguments::GetCLSID


## -description


Retrieves the object's CLSID.


## -parameters




### -param pCLSID [out]

A pointer to the object's CLSID.


## -returns



This method can return the standard return values E_INVALIDARG, E_OUTOFMEMORY, E_UNEXPECTED, E_FAIL, and S_OK.
