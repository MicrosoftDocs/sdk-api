# IServiceSysTxnConfig::ConfigureBYOTSysTxn


## -description


Enables you to run the enclosed code in the scope of an existing transaction that you specify with a transaction proxy.


## -parameters




### -param pTxProxy [in]

The <a href="https://docs.microsoft.com/windows/desktop/api/comsvcs/nn-comsvcs-itransactionproxy">ITransactionProxy</a> interface of the existing transaction in which you will run the enclosed code.


## -returns



This method can return the standard return values E_INVALIDARG, E_OUTOFMEMORY, E_FAIL, and S_OK.
