# IServiceIISIntrinsicsConfig::IISIntrinsicsConfig


## -description

Configures the IIS intrinsics for the enclosed work.

## -parameters

### -param iisIntrinsicsConfig [in]

A value from the <a href="/windows/desktop/api/comsvcs/ne-comsvcs-csc_iisintrinsicsconfig">CSC_IISIntrinsicsConfig</a> enumeration.

## -returns

This method can return the standard return values E_INVALIDARG, E_OUTOFMEMORY, E_FAIL, and S_OK.
