# tagSTROKE_RANGE structure


## -description



Specifies a range of strokes in the <a href="https://msdn.microsoft.com/f942d6a3-f303-49df-a128-de9760b508ef">InkDisp</a> object.




## -struct-fields




### -field iStrokeBegin

Index of the first stroke in the range, inclusive.


### -field iStrokeEnd

Index of the last stroke in the range, inclusive.
