# _DWORD_DWORD structure


## -description


The <b>DWORD_DWORD</b> structure defines a 64-bit integer value.


## -struct-fields




### -field DWord1

Specifies the upper 32 bits of the value.


### -field DWord2

Specifies the lower 32 bits of the value.
