# D2D_POINT_2F structure


## -description


Represents an x-coordinate and y-coordinate pair, expressed as floating-point values, in two-dimensional space.


## -struct-fields




### -field x

Type: <b>FLOAT</b>

The x-coordinate of the point.



### -field y

Type: <b>FLOAT</b>

The y-coordinate of the point.
