# D2D_SIZE_U structure


## -description

Stores an ordered pair of integers, typically the width and height of a rectangle.

## -struct-fields

### -field width

Type: <b>UINT32</b>

The horizontal component of this size.

### -field height

Type: <b>UINT32</b>

The vertical component of this size.
