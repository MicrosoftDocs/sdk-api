# _WMClientProperties structure


## -description



The <b>WM_CLIENT_PROPERTIES </b>structure records information about the client.




## -struct-fields




### -field dwIPAddress

<b>DWORD</b> containing the IP address.


### -field dwPort

<b>DWORD</b> containing the port number.
