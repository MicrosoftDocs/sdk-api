# IWMProfile3::SetStorageFormat


## -description



The <b>SetStorageFormat</b> method is not implemented.




## -parameters




### -param nStorageFormat [in]

Storage format.


## -returns



The method returns E_NOTIMPL.




## -remarks



To retrieve the storage format, use the <a href="https://msdn.microsoft.com/fea5b2e4-f10d-4482-a7b3-7dabf58df085">WM/ContainerFormat</a> attribute.
