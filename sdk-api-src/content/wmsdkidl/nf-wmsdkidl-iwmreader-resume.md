# IWMReader::Resume


## -description

The <b>Resume</b> method starts the reader from the current position, after a <b>Pause</b> method call.

## -parameters

## -returns

If the method succeeds, it returns S_OK. If it fails, it returns an <b>HRESULT</b> error code.
