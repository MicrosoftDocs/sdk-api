# IWMStreamConfig2 interface


## -description

The <b>IWMStreamConfig2</b> interface manages the data unit extensions associated with a stream.

<b>IWMStreamConfig2</b> inherits from <a href="/windows/desktop/api/wmsdkidl/nn-wmsdkidl-iwmstreamconfig">IWMStreamConfig</a>. To obtain a pointer to <b>IWMStreamConfig2</b>, call the <b>QueryInterface</b> method of the <b>IWMStreamConfig</b> interface.

The <b>IWMStreamConfig2</b> interface inherits from <a href="/windows/desktop/api/wmsdkidl/nn-wmsdkidl-iwmstreamconfig">IWMStreamConfig</a>. <b>IWMStreamConfig2</b> also has these types of members:
