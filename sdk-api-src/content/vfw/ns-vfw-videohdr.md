# VIDEOHDR structure


## -description



The <b>VIDEOHDR</b> structure is used by the <a href="https://docs.microsoft.com/windows/desktop/api/vfw/nc-vfw-capvideocallback">capVideoStreamCallback</a> function.




## -struct-fields




### -field lpData

Pointer to locked data buffer.
          


### -field dwBufferLength

Length of data buffer.
          


### -field dwBytesUsed

Bytes actually used.
          


### -field dwTimeCaptured

Milliseconds from start of stream.
          


### -field dwUser

User-defined data.
          


### -field dwFlags

The flags are defined as follows.

<table>
<tr>
<th>Flag</th>
<th>Meaning</th>
</tr>
<tr>
<td>VHDR_DONE</td>
<td>Done bit</td>
</tr>
<tr>
<td>VHDR_PREPARED</td>
<td>Set if this header has been prepared</td>
</tr>
<tr>
<td>VHDR_INQUEUE</td>
<td>Reserved for driver</td>
</tr>
<tr>
<td>VHDR_KEYFRAME</td>
<td>Key Frame</td>
</tr>
</table>
 


### -field dwReserved

Reserved for driver.
