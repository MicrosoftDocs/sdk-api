# IAVIFile::WriteData


## -description



The <b>WriteData</b> method writes file headers. Called when an application uses the <a href="https://docs.microsoft.com/windows/desktop/api/vfw/nf-vfw-avifilewritedata">AVIFileWriteData</a> function.




## -parameters




### -param ckid

A chunk ID.


### -param lpData

A pointer specifying the memory from which the data is written.


### -param cbData

A LONG specifying the number of bytes to write.





## -returns



Returns the HRESULT defined by OLE.




## -remarks



For handlers written in C++, <b>WriteData</b> has the following syntax:


```cpp

HRESULT WriteData(DWORD fcc, LPVOID lpBuffer, LONG cbBuffer); 
 

```
