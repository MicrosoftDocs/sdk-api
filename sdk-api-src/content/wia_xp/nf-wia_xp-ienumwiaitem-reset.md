# IEnumWiaItem::Reset


## -description

The <b>IEnumWiaItem::Reset</b> method is used by applications to restart the enumeration of item information.



## -returns

Type: <b>HRESULT</b>

If this method succeeds, it returns <b>S_OK</b>. Otherwise, it returns an <b>HRESULT</b> error code.
