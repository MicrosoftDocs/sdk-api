# IPSEC_SA_ENUM_TEMPLATE0_ structure


## -description


The <b>IPSEC_SA_ENUM_TEMPLATE0</b> structure specifies a  template used for restricting the enumeration of IPsec security associations (SAs).


## -struct-fields




### -field saDirection

Direction of the SA.

See <a href="https://msdn.microsoft.com/ae0eeb36-1a41-426a-9878-77558464a91b">FWP_DIRECTION</a> for more information.


## -remarks



<b>IPSEC_SA_ENUM_TEMPLATE0</b> is a specific implementation of IPSEC_SA_ENUM_TEMPLATE. See <a href="https://msdn.microsoft.com/FBDF53E5-F7DE-4DEB-AC18-6D2BB59FE670">WFP Version-Independent Names and Targeting Specific Versions of Windows</a>  for more information.
