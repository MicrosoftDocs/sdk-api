# FWP_BYTE_ARRAY16_ structure


## -description


The <b>FWP_BYTE_ARRAY16</b> structure  stores an array of exactly 16 bytes.


## -struct-fields




### -field byteArray16

Array of 16 bytes.


## -remarks



This data type  is useful for IPv6 addresses.
