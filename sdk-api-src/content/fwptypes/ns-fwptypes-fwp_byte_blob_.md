# FWP_BYTE_BLOB_ structure


## -description


The <b>FWP_BYTE_BLOB</b> structure stores an array containing a variable number of bytes.


## -struct-fields




### -field size

Number of  bytes in the array.


### -field data

Pointer to the array.
