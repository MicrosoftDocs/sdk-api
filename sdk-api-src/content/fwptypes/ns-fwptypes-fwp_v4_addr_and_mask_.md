# FWP_V4_ADDR_AND_MASK_ structure


## -description


The <b>FWP_V4_ADDR_AND_MASK</b> structure specifies IPv4 address and mask in host order.. 


## -struct-fields




### -field addr

Specifies an IPv4 address.


### -field mask

Specifies an IPv4 mask.
