# FWP_BYTE_ARRAY6 structure


## -description


The <b>FWP_BYTE_ARRAY6</b> structure  stores an array of exactly 6 bytes. Reserved.


## -struct-fields




### -field byteArray6

Array of 6 bytes.


## -remarks



This structure is reserved.
