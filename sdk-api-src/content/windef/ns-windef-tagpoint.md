# tagPOINT structure


## -description


The POINT structure defines the x- and y-coordinates of a point.


## -struct-fields




### -field x

Specifies the <i>x</i>-coordinate of the point. 


### -field y

Specifies the <i>y</i>-coordinate of the point. 


## -remarks



The POINT structure is identical to the <a href="https://msdn.microsoft.com/68cd23d7-7898-4132-abfe-4dda527889b9">POINTL</a> structure.
