# IAMExtTransport::get_Mode


## -description



The <code>get_Mode</code> method retrieves the current transport mode, such as play, stop, or record.




## -parameters




### -param pMode [out]

Pointer to a <b>long</b> integer that receives the current transport mode. For possible values, see <a href="https://docs.microsoft.com/windows/desktop/api/strmif/nf-strmif-iamexttransport-put_mode">IAMExtTransport::put_Mode</a>.


## -returns



When this method succeeds, it returns S_OK. Otherwise it returns an <b>HRESULT</b> error code.
