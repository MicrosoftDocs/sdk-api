# IVMRMixerBitmap::GetAlphaBitmapParameters


## -description



The <b>GetAlphaBitmapParameters</b> method retrieves a copy of the current image and related blending parameters.




## -parameters




### -param pBmpParms [out]

A pointer to a [VMRALPHABITMAP](https://docs.microsoft.com/windows/desktop/api/strmif/ns-strmif-vmralphabitmap) structure that receives the bitmap, information about the blending values, and the location to blend it.
          


## -returns



If the method succeeds, it returns S_OK. If it fails, it returns an error code.

<tr>
<th>Return code</th>
<th>Description</th>
</tr>
<tr>
<td width="40%">
<dl>
<dt><b>E_POINTER</b></dt>
</dl>
</td>
<td width="60%">
<i>pBmpParms</i> is invalid.

</td>
</tr>
<tr>
<td width="40%">
<dl>
<dt><b>VFW_E_WRONG_STATE</b></dt>
</dl>
</td>
<td width="60%">
The mixer has not been loaded.

</td>
</tr>
</table>
