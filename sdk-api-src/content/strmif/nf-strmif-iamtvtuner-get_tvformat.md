# IAMTVTuner::get_TVFormat


## -description



The <code>get_TVFormat</code> method retrieves the current analog video TV standard in use.




## -parameters




### -param plAnalogVideoStandard [out]

Pointer to a variable that receives a member of the <a href="https://msdn.microsoft.com/6760a40c-550c-4774-a5d1-d7e2a6aa6096">AnalogVideoStandard</a> enumeration.


## -returns



Returns an <b>HRESULT</b> value that depends on the implementation of the interface.
