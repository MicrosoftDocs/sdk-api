# TunerInputType enumeration


## -description

Specifies the frequency of a TV tuner (cable or antenna).

## -enum-fields

### -field TunerInputCable:0

Indicates cable frequency.

### -field TunerInputAntenna

Indicates broadcast antenna frequency.
