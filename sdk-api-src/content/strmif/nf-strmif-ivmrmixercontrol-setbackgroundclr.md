# IVMRMixerControl::SetBackgroundClr


## -description



Sets the background color on the output rectangle.




## -parameters




### -param ClrBkg [in]

Specifies the background color.


## -returns



If the method succeeds, it returns S_OK. If it fails, it returns an error code.
