# IAMFilterMiscFlags::GetMiscFlags


## -description



The <code>GetMiscFlags</code> method returns the filter's type, either source or renderer.




## -parameters






## -returns



This method returns a member of the <a href="https://msdn.microsoft.com/7acc160c-0da8-4b85-b88c-82b59ec38106">_AM_FILTER_MISC_FLAGS</a> enumeration.
