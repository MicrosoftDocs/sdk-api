# IEnumRegFilters::Skip


## -description



<div class="alert"><b>Note</b>  The <code>IEnumRegFilters</code> interface is deprecated.</div>
<div> </div>
This method is not currently implemented and returns E_NOTIMPL.




## -parameters




### -param cFilters






#### - celt [in]

Number of items to skip.


## -returns
