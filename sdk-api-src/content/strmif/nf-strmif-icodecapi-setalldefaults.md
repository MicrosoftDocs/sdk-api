# ICodecAPI::SetAllDefaults


## -description

The <b>SetAllDefaults</b> method resets all codec properties to their default values.



## -returns

If this method succeeds, it returns <b>S_OK</b>. Otherwise, it returns an <b>HRESULT</b> error code.
