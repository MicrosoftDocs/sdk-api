# _DVENCODERFORMAT enumeration


## -description



Indicates the digital video (DV) format.




## -enum-fields




### -field DVENCODERFORMAT_DVSD

Use the 'dvsd' stream handler.


### -field DVENCODERFORMAT_DVHD

Use the 'dvhd' stream handler.


### -field DVENCODERFORMAT_DVSL

Use the 'dvsl' stream handler.


## -remarks



This enumeration specifies the <b>fccType</b> member of the AVI stream header. For more information, see <a href="https://msdn.microsoft.com/ae1ec184-afc3-4ec1-9b92-f53656293446">DV Data in the AVI File Format</a>.
