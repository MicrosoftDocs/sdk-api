# GdiplusBase::operator new


## -description


The <b>GdiplusBase::operator new</b> method allocates memory for one Windows GDI+ object.


## -parameters




### -param in_size [in]

Type: <b>size_t</b>

Size of the object for which memory is to be allocated. 


## -returns



This method returns a pointer to the object.
