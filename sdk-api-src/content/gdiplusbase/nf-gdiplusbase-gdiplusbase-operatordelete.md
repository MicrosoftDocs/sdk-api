# GdiplusBase::operator delete


## -description

The <b>GdiplusBase::operator delete</b> method deallocates memory for one Windows GDI+ object.

## -parameters

### -param in_pVoid [in]

Type: <b>void*</b>

Pointer to the object to be deallocated.

## -returns

This method does not return a value.
