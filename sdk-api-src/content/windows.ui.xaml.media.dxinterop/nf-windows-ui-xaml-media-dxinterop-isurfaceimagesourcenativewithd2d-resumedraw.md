# ISurfaceImageSourceNativeWithD2D::ResumeDraw (windows.ui.xaml.media.dxinterop.h)


## -description

Resume the drawing operation.



## -returns

If this method succeeds, it returns <b>S_OK</b>. Otherwise, it returns an <b>HRESULT</b> error code.
