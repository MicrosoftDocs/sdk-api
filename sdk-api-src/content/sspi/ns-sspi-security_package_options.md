# SECURITY_PACKAGE_OPTIONS structure


## -description


Specifies information about a security package. This structure is used by the <a href="https://docs.microsoft.com/windows/desktop/api/sspi/nf-sspi-addsecuritypackagea">AddSecurityPackage</a> function.


## -struct-fields




### -field Size

The size, in bytes, of this structure.


### -field Type

The type of security package. This can be one of the following values.

<th>Value</th>
<th>Meaning</th>

<dt><b>SECPKG_OPTIONS_TYPE_UNKNOWN</b></dt>

</dl>
</td>

The package type is not known.

</td>
</tr>
<tr>
<td width="40%"><a id="SECPKG_OPTIONS_TYPE_LSA"></a><a id="secpkg_options_type_lsa"></a><dl>
<dt><b>SECPKG_OPTIONS_TYPE_LSA</b></dt>
<dt>1</dt>
</dl>
</td>
<td width="60%">
The security package is an <a href="https://docs.microsoft.com/windows/desktop/SecAuthN/lsa-authentication">LSA authentication</a> package.

</td>

<tr>

</dl>

</td>
</tr>

### -field Flags

This member is reserved. Do not use it.


### -field SignatureSize

The size, in bytes, of a digital signature for this security package.


### -field Signature

A digital signature for this security package.
