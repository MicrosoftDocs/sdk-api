# PrjFreeAlignedBuffer function


## -description

Frees an allocated buffer.

## -parameters

### -param buffer [in]

The buffer to free.

## -returns

If this function succeeds, it returns <b>S_OK</b>. Otherwise, it returns an <b>HRESULT</b> error code.
