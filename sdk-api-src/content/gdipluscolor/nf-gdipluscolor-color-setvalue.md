# Color::SetValue


## -description


The <b>Color::SetValue</b> method sets the color of this <a href="https://msdn.microsoft.com/dae648fd-1302-481e-9f5b-331a4c1b5e0d">Color</a> object.


## -parameters




### -param argb [in]

Type: <b>ARGB</b>

Thirty-two bit <b>ARGB</b> value that specifies the color. An <b>ARGB</b> value consolidates the alpha, red, green, and blue components of a color. <b>ARGB</b> is defined in Gdipluspixelformats.h. 


## -returns



This method does not return a value.
