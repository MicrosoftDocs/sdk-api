# Color::GetRed


## -description

The <b>Color::GetRed</b> method gets the red component of this <a href="/windows/desktop/api/gdipluscolor/nl-gdipluscolor-color">Color</a> object.



## -returns

Type: <b>BYTE</b>

This method returns the red component of this color.
