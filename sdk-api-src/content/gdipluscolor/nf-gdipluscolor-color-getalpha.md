# Color::GetAlpha


## -description

The <b>Color::GetAlpha</b> method gets the alpha component of this <a href="/windows/desktop/api/gdipluscolor/nl-gdipluscolor-color">Color</a> object.

## -parameters

## -returns

Type: <b>BYTE</b>

This method returns the alpha component of this color.
