# Color::GetAlpha


## -description


The <b>Color::GetAlpha</b> method gets the alpha component of this <a href="https://docs.microsoft.com/windows/desktop/api/gdipluscolor/nl-gdipluscolor-color">Color</a> object.


## -parameters






## -returns



Type: <strong>Type: <b>BYTE</b>
</strong>

This method returns the alpha component of this color.
