# Color::GetB


## -description

The <b>Color::GetB</b> method gets the blue component of this <a href="/windows/desktop/api/gdipluscolor/nl-gdipluscolor-color">Color</a> object.



## -returns

Type: <b>BYTE</b>

This method returns the blue component of this color.
