# Color::GetBlue


## -description


The <b>Color::GetBlue</b> method gets the blue component of this <a href="https://docs.microsoft.com/windows/desktop/api/gdipluscolor/nl-gdipluscolor-color">Color</a> object.


## -parameters






## -returns



Type: <b>BYTE</b>

This method returns the blue component of this color.
