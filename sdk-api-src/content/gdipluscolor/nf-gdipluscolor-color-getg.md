# Color::GetG


## -description


The <b>Color::GetG</b> method gets the green component of this <a href="https://docs.microsoft.com/windows/desktop/api/gdipluscolor/nl-gdipluscolor-color">Color</a> object.


## -parameters






## -returns



Type: <strong>Type: <b>BYTE</b>
</strong>

This method returns the green component of this color.
