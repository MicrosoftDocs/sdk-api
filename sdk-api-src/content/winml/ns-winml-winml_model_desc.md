# WINML_MODEL_DESC structure


## -description



<b>These APIs have been deprecated and should no longer be used:  </b>Please use <a href="/uwp/api/windows.ai.machinelearning">Windows.AI.MachineLearning</a> instead.

Contains description properties of the model.

## -struct-fields

### -field Author

The author of the model.

### -field Name

The name of the model.

### -field Domain

The domain of the model.

### -field Description

The description of the model.

### -field Version

The version of the model.
