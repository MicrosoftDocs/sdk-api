# WINML_BINDING_TYPE enumeration


## -description



<b>These APIs have been deprecated and should no longer be used:  </b>Please use <a href="/uwp/api/windows.ai.machinelearning">Windows.AI.MachineLearning</a> instead.

Specifies the different types of WinML bindings.

## -enum-fields

### -field WINML_BINDING_UNDEFINED:0

Binding type undefined.

### -field WINML_BINDING_TENSOR

Binding of type tensor.

### -field WINML_BINDING_SEQUENCE

Binding of type sequence.

### -field WINML_BINDING_MAP

Binding of type map.

### -field WINML_BINDING_IMAGE

Binding of type image.

### -field WINML_BINDING_RESOURCE

Binding of type resource.
