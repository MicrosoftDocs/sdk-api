# D3D12_DEPTH_STENCIL_VALUE structure


## -description

Specifies a depth and stencil value.

## -struct-fields

### -field Depth

Specifies the depth value.

### -field Stencil

Specifies the stencil value.

## -remarks

This structure is used in the <a href="/windows/desktop/api/d3d12/ns-d3d12-d3d12_clear_value">D3D12_CLEAR_VALUE</a> structure.
