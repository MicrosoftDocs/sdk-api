# D3D12_DEPTH_STENCIL_VALUE structure


## -description


Specifies a depth and stencil value.


## -struct-fields




### -field Depth

Specifies the depth value.


### -field Stencil

Specifies the stencil value.


## -remarks



This structure is used in the <a href="https://msdn.microsoft.com/en-us/library/Dn903795(v=VS.85).aspx">D3D12_CLEAR_VALUE</a> structure.
