## -description

Defines constants that specify protected resource session support.

## -enum-fields

### -field D3D12_PROTECTED_RESOURCE_SESSION_SUPPORT_FLAG_NONE (0x0)

Indicates protected resource sessions are not supported.

### -field D3D12_PROTECTED_RESOURCE_SESSION_SUPPORT_FLAG_SUPPORTED (0x1)

Indicates that protected resource sessions are supported.
