# D3D12_TEX1D_UAV structure


## -description

Describes a unordered-access 1D texture resource.

## -struct-fields

### -field MipSlice

The mipmap slice index.

## -remarks

Use this structure with a <a href="/windows/desktop/api/d3d12/ns-d3d12-d3d12_unordered_access_view_desc">D3D12_UNORDERED_ACCESS_VIEW_DESC</a> structure to view the resource as a 1D texture.
