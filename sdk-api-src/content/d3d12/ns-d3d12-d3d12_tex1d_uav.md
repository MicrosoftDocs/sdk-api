# D3D12_TEX1D_UAV structure


## -description


Describes a unordered-access 1D texture resource.


## -struct-fields




### -field MipSlice

The mipmap slice index.


## -remarks



Use this structure with a <a href="https://msdn.microsoft.com/0C3A31FE-625D-4CB3-87FD-D2C33D008DD4">D3D12_UNORDERED_ACCESS_VIEW_DESC</a> structure to view the resource as a 1D texture.
