## -description

Defines constants that specify protected resource session flags. These flags can be bitwise OR'd together to specify multiple flags at once.

## -enum-fields

### -field D3D12_PROTECTED_RESOURCE_SESSION_FLAG_NONE:0

Specifies no flag.
