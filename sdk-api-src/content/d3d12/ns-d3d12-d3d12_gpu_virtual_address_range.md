# D3D12_GPU_VIRTUAL_ADDRESS_RANGE structure


## -description

Represents a GPU virtual address range.

## -struct-fields

### -field StartAddress

The beginning of the virtual address range.

### -field SizeInBytes

The size of the virtual address range, in bytes.
