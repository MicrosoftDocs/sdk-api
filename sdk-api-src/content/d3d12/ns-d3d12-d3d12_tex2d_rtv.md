# D3D12_TEX2D_RTV structure


## -description


Describes the subresource from a 2D texture to use in a render-target view.


## -struct-fields




### -field MipSlice

The index of the mipmap level to use.


### -field PlaneSlice

The index (plane slice number) of the plane to use in the texture.
          


## -remarks



Use this structure with a <a href="https://docs.microsoft.com/windows/desktop/api/d3d12/ns-d3d12-d3d12_render_target_view_desc">D3D12_RENDER_TARGET_VIEW_DESC</a> structure to view the resource as a 2D texture.
