# ID3D12GraphicsCommandList::OMSetStencilRef


## -description


Sets the reference value for depth stencil tests.


## -parameters




### -param StencilRef [in]

Type: <b>UINT</b>

Reference value to perform against when doing a depth-stencil test.
          


## -returns



This method does not return a value.
