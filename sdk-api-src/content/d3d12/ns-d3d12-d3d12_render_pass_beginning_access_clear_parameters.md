# D3D12_RENDER_PASS_BEGINNING_ACCESS_CLEAR_PARAMETERS structure


## -description

Describes the clear value to which resource(s) should be cleared at the beginning of a render pass.

## -struct-fields

### -field ClearValue

A  <a href="/windows/desktop/api/d3d12/ns-d3d12-d3d12_clear_value">D3D12_CLEAR_VALUE</a>. The clear value to which the resource(s) should be cleared.
