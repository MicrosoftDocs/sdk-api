# ID3D12PipelineLibrary::GetSerializedSize


## -description


Returns the amount of memory required to serialize the current contents of the database. 



## -parameters






## -returns



Type: <b>SIZE_T</b>

This method returns a SIZE_T object, containing the size required in bytes.
