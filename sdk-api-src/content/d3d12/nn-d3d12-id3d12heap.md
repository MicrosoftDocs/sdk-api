# ID3D12Heap interface


## -description

A heap  is an abstraction of contiguous memory allocation, used to manage physical memory. This heap can be used with <a href="/windows/desktop/api/d3d12/nn-d3d12-id3d12resource">ID3D12Resource</a> objects to support placed resources or reserved resources.
