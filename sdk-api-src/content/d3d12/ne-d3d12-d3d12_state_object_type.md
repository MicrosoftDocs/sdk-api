# D3D12_STATE_OBJECT_TYPE enumeration


## -description


Specifies the type of a state object. Use with <a href="http://docs.microsoft.com/windows/desktop/api/d3d12/ns-d3d12-d3d12_state_object_desc">D3D12_STATE_OBJECT_DESC</a>.


## -enum-fields




### -field D3D12_STATE_OBJECT_TYPE_COLLECTION

Collection state object.


### -field D3D12_STATE_OBJECT_TYPE_RAYTRACING_PIPELINE

Raytracing pipeline state object.
