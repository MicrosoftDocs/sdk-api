# ID3D10Effect::IsValid


## -description

Test an effect to see if it contains valid syntax.

## -parameters

## -returns

Type: <b><a href="/windows/desktop/WinProg/windows-data-types">BOOL</a></b>

<b>TRUE</b> if the code syntax is valid; otherwise <b>FALSE</b>.
