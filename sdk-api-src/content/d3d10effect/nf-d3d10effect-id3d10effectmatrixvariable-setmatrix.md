# ID3D10EffectMatrixVariable::SetMatrix


## -description

Set a floating-point matrix.

## -parameters

### -param pData [in]

Type: <b>float*</b>

A pointer to the first element in the matrix.

## -returns

Type: <b><a href="/windows/win32/com/structure-of-com-error-codes">HRESULT</a></b>
