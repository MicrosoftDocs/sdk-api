# ID3D10EffectConstantBuffer::SetTextureBuffer


## -description


Set a texture-buffer.


## -parameters




### -param pTextureBuffer [in]

Type: <b><a href="https://msdn.microsoft.com/en-us/library/Bb173854(v=VS.85).aspx">ID3D10ShaderResourceView</a>*</b>

A pointer to a shader-resource-view interface for accessing a texture buffer.


## -returns



Type: <b><a href="https://msdn.microsoft.com/en-us/library/Hh437604(v=VS.85).aspx">HRESULT</a></b>

Returns one of the following <a href="https://msdn.microsoft.com/en-us/library/Bb205278(v=VS.85).aspx">Direct3D 10 Return Codes</a>.
