# ID3D10EffectConstantBuffer::SetTextureBuffer


## -description

Set a texture-buffer.

## -parameters

### -param pTextureBuffer [in]

Type: <b><a href="/windows/desktop/api/d3d10/nn-d3d10-id3d10shaderresourceview">ID3D10ShaderResourceView</a>*</b>

A pointer to a shader-resource-view interface for accessing a texture buffer.

## -returns

Type: <b><a href="/windows/win32/com/structure-of-com-error-codes">HRESULT</a></b>

Returns one of the following <a href="/windows/desktop/direct3d10/d3d10-graphics-reference-returnvalues">Direct3D 10 Return Codes</a>.
