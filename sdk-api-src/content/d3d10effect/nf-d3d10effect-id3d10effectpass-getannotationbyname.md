# ID3D10EffectPass::GetAnnotationByName


## -description

The name of the annotation.

Type: <b><a href="/windows/desktop/api/d3d10effect/nn-d3d10effect-id3d10effectvariable">ID3D10EffectVariable</a>*</b>
