# ID3D10EffectVectorVariable::SetFloatVector


## -description


Set a four-component vector that contains floating-point data.


## -parameters




### -param pData [in]

Type: <b>float*</b>

A pointer to the first component.

Type: <b><a href="455d07e9-52c3-4efb-a9dc-2955cbfd38cc">HRESULT</a></b>
