# IWindowsDriverUpdate5::get_AutoSelection


## -description


Gets an <a href="https://docs.microsoft.com/windows/desktop/api/wuapi/ne-wuapi-autoselectionmode">AutoSelectionMode</a> value indicating the automatic selection mode of an update in the Control Panel of Windows Update.

This property is read-only.


## -parameters


## -remarks



The AutoSelection property indicates whether the update will be automatically selected when the user views the available updates in the Windows Update user interface.
