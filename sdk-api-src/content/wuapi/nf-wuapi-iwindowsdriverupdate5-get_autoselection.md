# IWindowsDriverUpdate5::get_AutoSelection


## -description


Gets an <a href="https://msdn.microsoft.com/847fd8a3-eb00-43f7-a89f-579cd79d0620">AutoSelectionMode</a> value indicating the automatic selection mode of an update in the Control Panel of Windows Update.

This property is read-only.


## -parameters


## -remarks



The AutoSelection property indicates whether the update will be automatically selected when the user views the available updates in the Windows Update user interface.
