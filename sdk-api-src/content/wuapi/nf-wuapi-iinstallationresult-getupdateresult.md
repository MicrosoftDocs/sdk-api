# IInstallationResult::GetUpdateResult


## -description


Returns an <a href="https://docs.microsoft.com/windows/desktop/api/wuapi/nn-wuapi-iupdateinstallationresult">IUpdateInstallationResult</a> interface that contains the installation results  for a specified update.


## -parameters




### -param updateIndex [in]

The index of an update.


### -param retval [out]

An interface that contains results for a specified update.


## -returns



Returns <b>S_OK</b> if successful. Otherwise, returns a COM or Windows 

error code.
