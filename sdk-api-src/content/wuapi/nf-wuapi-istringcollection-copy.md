# IStringCollection::Copy


## -description


Creates a deep read/write copy of the collection.


## -parameters




### -param retval [out]

A deep read/write copy of the collection.


## -returns



Returns <b>S_OK</b> if successful. Otherwise, returns a COM or Windows error code.
