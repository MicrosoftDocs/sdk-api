# IUpdateCollection::Copy


## -description


Creates a shallow read/write copy of the collection.


## -parameters




### -param retval [out]

A shallow read/write copy of the collection.


## -returns



Returns <b>S_OK</b> if successful. Otherwise, returns a COM or Windows error code.
