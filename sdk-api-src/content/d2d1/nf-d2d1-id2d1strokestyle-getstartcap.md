# ID2D1StrokeStyle::GetStartCap


## -description

Retrieves the type of shape used at the beginning of a stroke.

Type: <b><a href="/windows/win32/api/d2d1/ne-d2d1-d2d1_cap_style">D2D1_CAP_STYLE</a></b>
