# D2D1_BITMAP_PROPERTIES structure


## -description


Describes the pixel format and dpi 
    of a bitmap.


## -struct-fields




### -field pixelFormat

Type: <b><a href="https://msdn.microsoft.com/e95afd9c-5793-4cb7-bcb8-aae4d28b6532">D2D1_PIXEL_FORMAT</a></b>

The bitmap's pixel format and alpha mode.


### -field dpiX

Type: <b>FLOAT</b>

The horizontal dpi of the bitmap.


### -field dpiY

Type: <b>FLOAT</b>

The vertical dpi of the bitmap.
