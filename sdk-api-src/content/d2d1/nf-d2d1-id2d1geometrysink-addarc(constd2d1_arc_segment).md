## -description

Adds a single arc to the path geometry.

## -parameters

### -param arc

Type: [in] <b>const <a href="/windows/win32/api/d2d1/ns-d2d1-d2d1_arc_segment">D2D1_ARC_SEGMENT</a>*</b>

The arc segment to add to the figure.
