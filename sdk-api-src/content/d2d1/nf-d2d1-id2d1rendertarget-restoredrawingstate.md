# ID2D1RenderTarget::RestoreDrawingState


## -description


Sets the render target's drawing state to that of the specified <a href="https://msdn.microsoft.com/9a3d9146-0e1b-4642-ad5d-ff1d09a93d2b">ID2D1DrawingStateBlock</a>.


## -parameters




### -param drawingStateBlock [in]

Type: <b><a href="https://msdn.microsoft.com/9a3d9146-0e1b-4642-ad5d-ff1d09a93d2b">ID2D1DrawingStateBlock</a>*</b>

The new drawing state of the render target.


## -returns



This method does not return a value.
