# ID2D1RadialGradientBrush::GetRadiusY


## -description


Retrieves the y-radius of the gradient ellipse.


## -parameters






## -returns



Type: <b>FLOAT</b>

The y-radius of the gradient ellipse. This value is expressed in the brush's coordinate space.
