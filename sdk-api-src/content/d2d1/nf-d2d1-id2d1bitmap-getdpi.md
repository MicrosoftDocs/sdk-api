# ID2D1Bitmap::GetDpi


## -description


Return the dots per inch (DPI) of the bitmap.


## -parameters




### -param dpiX [out]

Type: <b>FLOAT*</b>

The horizontal DPI of the image. You must allocate storage for this parameter.


### -param dpiY [out]

Type: <b>FLOAT*</b>

The vertical DPI of the image.  You must allocate storage for this parameter.
