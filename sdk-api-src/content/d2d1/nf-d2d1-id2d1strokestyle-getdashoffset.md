# ID2D1StrokeStyle::GetDashOffset


## -description


Retrieves a value that specifies how far in the dash sequence the stroke will start. 


## -parameters






## -returns



Type: <b>FLOAT</b>

A value that specifies how far in the dash sequence the stroke will start.
