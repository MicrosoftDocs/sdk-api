# ID2D1GeometrySink::AddBezier(const D2D1_BEZIER_SEGMENT)


## -description


Creates  a cubic Bezier curve between the current point and the specified endpoint.


## -parameters




### -param bezier [in]

Type: <b>const <a href="https://msdn.microsoft.com/cf8df7d2-c4fe-4a46-a4b2-7e0eed67df2a">D2D1_BEZIER_SEGMENT</a>*</b>

A structure that describes the control points and endpoint of the Bezier curve to add. 


## -returns



This method does not return a value.
