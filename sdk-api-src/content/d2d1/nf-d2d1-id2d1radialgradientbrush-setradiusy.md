# ID2D1RadialGradientBrush::SetRadiusY


## -description


Specifies the y-radius of the gradient ellipse, in the brush's coordinate space.


## -parameters




### -param radiusY

Type: <b>FLOAT</b>

The y-radius of the gradient ellipse. This value is in the brush's coordinate space.
