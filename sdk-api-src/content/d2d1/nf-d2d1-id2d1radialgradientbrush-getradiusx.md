# ID2D1RadialGradientBrush::GetRadiusX


## -description


Retrieves the x-radius of the gradient ellipse.


## -parameters






## -returns



Type: <b>FLOAT</b>

The x-radius of the gradient ellipse. This value is expressed in the brush's coordinate space.
