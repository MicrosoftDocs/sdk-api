# PXE_DHCP_OPTION structure


## -description


The <b>PXE_DHCP_OPTION</b> structure can be used with the Windows Deployment Services PXE Server API. 


## -struct-fields




### -field OptionType

A DHCP option type.


### -field OptionLength

Length of the option value.


### -field OptionValue

The option value.
