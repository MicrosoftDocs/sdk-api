# Matrix4x4F::SetProduct


## -description


Multiplies the two matrices and stores the result in this matrix.


## -parameters




### -param a [ref]

Type: <b>const <a href="https://docs.microsoft.com/windows/desktop/api/d2d1_1helper/nl-d2d1_1helper-matrix4x4f">Matrix4x4F</a></b>

The first matrix to multiply.


### -param b [ref]

Type: <b>const <a href="https://docs.microsoft.com/windows/desktop/api/d2d1_1helper/nl-d2d1_1helper-matrix4x4f">Matrix4x4F</a></b>

The second matrix to multiply.


## -returns



This method does not return a value.
