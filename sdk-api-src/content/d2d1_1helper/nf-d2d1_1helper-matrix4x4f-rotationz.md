# Matrix4x4F::RotationZ


## -description


Rotates the transform matrix around the Z axis.


## -parameters




### -param degreeZ

Type: <b>FLOAT</b>

The amount of rotation.


## -returns



Type: <b><a href="https://docs.microsoft.com/windows/desktop/api/d2d1_1helper/nl-d2d1_1helper-matrix4x4f">Matrix4x4F</a></b>

The result matrix.
