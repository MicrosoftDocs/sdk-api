# Matrix4x4F::Scale


## -description

Scales the perspective plane of the matrix.

## -parameters

### -param x

Type: <b>FLOAT</b>

The scale in the X direction.

### -param y

Type: <b>FLOAT</b>

The scale in the Y direction.

### -param z

Type: <b>FLOAT</b>

The scale in the Z direction.

## -returns

Type: <b><a href="/windows/desktop/api/d2d1_1helper/nl-d2d1_1helper-matrix4x4f">Matrix4x4F</a></b>

The result matrix.
