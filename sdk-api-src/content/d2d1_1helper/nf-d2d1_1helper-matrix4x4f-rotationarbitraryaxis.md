# Matrix4x4F::RotationArbitraryAxis


## -description

Determines the 3-D Rotation matrix for an arbitrary axis.

## -parameters

### -param x

Type: <b>FLOAT</b>

The X point of the axis.

### -param y

Type: <b>FLOAT</b>

The Y point of the axis.

### -param z

Type: <b>FLOAT</b>

The Z point of the axis.

### -param degree

Type: <b>FLOAT</b>

The amount of rotation.

## -returns

Type: <b><a href="/windows/desktop/api/d2d1_1helper/nl-d2d1_1helper-matrix4x4f">Matrix4x4F</a></b>

The result matrix
