# Matrix4x4F::SkewY


## -description

Skews the matrix in the Y direction.

## -parameters

### -param degreeY

Type: <b>FLOAT</b>

The skew amount.

## -returns

Type: <b><a href="/windows/desktop/api/d2d1_1helper/nl-d2d1_1helper-matrix4x4f">Matrix4x4F</a></b>

The result matrix.
