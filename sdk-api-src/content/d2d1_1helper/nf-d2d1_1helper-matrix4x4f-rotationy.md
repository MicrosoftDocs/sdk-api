# Matrix4x4F::RotationY


## -description

Rotates the transform matrix around the Y axis.

## -parameters

### -param degreeY

Type: <b>FLOAT</b>

The amount of rotation.

## -returns

Type: <b><a href="/windows/desktop/api/d2d1_1helper/nl-d2d1_1helper-matrix4x4f">Matrix4x4F</a></b>

The result matrix.
