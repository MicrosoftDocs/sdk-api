# Matrix4x4F::RotationY


## -description


Rotates the transform matrix around the Y axis.


## -parameters




### -param degreeY

Type: <b>FLOAT</b>

The amount of rotation.


## -returns



Type: <b><a href="https://msdn.microsoft.com/113861DF-2E6D-4930-82DC-AA592882E21C">Matrix4x4F</a></b>

The result matrix.
