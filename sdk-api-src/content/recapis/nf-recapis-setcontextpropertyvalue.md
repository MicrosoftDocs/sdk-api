# SetContextPropertyValue function


## -description



Adds a property to the recognizer context.

If a property already exists, its value is modified.




## -parameters




### -param hrc

The handle to the recognizer context.


### -param pGuid

The property to set. Specify a predefined property globally unique identifier (GUID) or application-defined property GUID. For a list of predefined properties, see the recognition <a href="https://docs.microsoft.com/windows/desktop/tablet/property-guids">Property GUIDs</a>.


### -param cbSize

The size, in bytes, of the <i>pProperty</i> buffer.


### -param pProperty

The buffer that contains the property value.
