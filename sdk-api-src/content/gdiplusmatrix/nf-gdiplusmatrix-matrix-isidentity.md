# Matrix::IsIdentity


## -description


The <b>Matrix::IsIdentity</b> method determines whether this matrix is the identity matrix.


## -parameters






## -returns



Type: <strong>Type: <b>BOOL</b>
</strong>

If this matrix is the identity matrix, this method returns <b>TRUE</b>; otherwise, it returns <b>FALSE</b>.
