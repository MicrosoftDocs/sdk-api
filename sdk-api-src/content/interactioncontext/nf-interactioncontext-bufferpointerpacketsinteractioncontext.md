# BufferPointerPacketsInteractionContext function


## -description

Adds the history for a single input pointer to the buffer of the <a href="/previous-versions/windows/desktop/input_intcontext/interaction-context-portal">Interaction Context</a> object.

## -parameters

### -param interactionContext [in]

The handle of the interaction context.

### -param entriesCount [in]

The number of entries in the pointer history.

### -param pointerInfo [in]

Basic pointer information common to all pointer types.

## -returns

If this function succeeds, it returns S_OK.
 
Otherwise, it returns an HRESULT error code.
