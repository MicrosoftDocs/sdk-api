# _EVT_OPEN_LOG_FLAGS enumeration


## -description


Defines the values that specify whether to open a channel or exported log file.


## -enum-fields




### -field EvtOpenChannelPath

Open a channel.


### -field EvtOpenFilePath

Open an exported log file.
