# BLUETOOTH_FIND_RADIO_PARAMS structure


## -description


The <b>BLUETOOTH_FIND_RADIO_PARAMS</b> structure facilitates enumerating installed Bluetooth radios.


## -struct-fields




### -field dwSize

Size of the <b>BLUETOOTH_FIND_RADIO_PARAMS</b> structure, in bytes.
