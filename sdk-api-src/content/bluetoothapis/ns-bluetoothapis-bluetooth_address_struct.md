# BLUETOOTH_ADDRESS_STRUCT structure


## -description


The 
<b>BLUETOOTH_ADDRESS</b> structure provides the address of a Bluetooth device.


## -struct-fields




### -field ullLong

Address of a Bluetooth device.


### -field rgBytes

Address of a Bluetooth device, expressed as an array of bytes.
