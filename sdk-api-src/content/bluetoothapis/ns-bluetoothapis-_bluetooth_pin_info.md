# _BLUETOOTH_PIN_INFO structure


## -description


The <b>BLUETOOTH_PIN_INFO</b> structure contains  information used for authentication via PIN.


## -struct-fields




### -field pin

The PIN  used for authentication.


### -field pinLength

The length of <i>pin</i>.
