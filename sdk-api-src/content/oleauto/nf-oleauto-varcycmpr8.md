# VarCyCmpR8 function


## -description


Compares a variant of type currency with a value of type double.


## -parameters




### -param cyLeft [in]

The first variant.


### -param dblRight [in]

The second variant.


## -returns



This function can return one of these values.

</dl>

</td>
