# VarCyMulI4 function


## -description

Multiplies a currency value by a 32-bit integer.

## -parameters

### -param cyLeft [in]

The first variant.

### -param lRight [in]

The second variant.

### -param pcyResult [out]

The resulting variant.

## -returns

If this function succeeds, it returns <b>S_OK</b>. Otherwise, it returns an <b>HRESULT</b> error code.
