# VarDecCmpR8 function


## -description


Compares a variant of type decimal with the a value of type double.


## -parameters




### -param pdecLeft [in]

The first variant.


### -param dblRight [in]

The second variant.


## -returns



This function can return one of these values.

</dl>

</td>
