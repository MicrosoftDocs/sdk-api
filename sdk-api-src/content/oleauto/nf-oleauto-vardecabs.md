# VarDecAbs function


## -description

Retrieves the absolute value of a variant of type decimal.

## -parameters

### -param pdecIn [in]

The first variant.

### -param pdecResult [out]

The second variant.

## -returns

If this function succeeds, it returns <b>S_OK</b>. Otherwise, it returns an <b>HRESULT</b> error code.
