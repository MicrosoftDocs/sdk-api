# VarCyAbs function


## -description

Retrieves the absolute value of a variant of type currency.

## -parameters

### -param cyIn [in]

The currency variant.

### -param pcyResult [out]

The resulting variant.

## -returns

If this function succeeds, it returns <b>S_OK</b>. Otherwise, it returns an <b>HRESULT</b> error code.
