# REGKIND enumeration


## -description

Controls how a type library is registered.

## -enum-fields

### -field REGKIND_DEFAULT

Use default register behavior.

### -field REGKIND_REGISTER

Register this type library.

### -field REGKIND_NONE

Do not register this type library.
