# VarBstrFromI8 function


## -description


Converts an 8-byte unsigned integer value to a BSTR value.


## -parameters




### -param i64In [in]

The value to convert.


### -param lcid [in]

The locale identifier.


### -param dwFlags [in]

Reserved. Set to zero.


### -param pbstrOut [out]

The resulting value.
