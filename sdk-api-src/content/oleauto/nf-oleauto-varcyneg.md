# VarCyNeg function


## -description

Performs a  logical negation on a variant of type currency.

## -parameters

### -param cyIn [in]

The variant to negate.

### -param pcyResult [out]

The resulting variant.

## -returns

If this function succeeds, it returns <b>S_OK</b>. Otherwise, it returns an <b>HRESULT</b> error code.
