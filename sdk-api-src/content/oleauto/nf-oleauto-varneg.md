# VarNeg function


## -description

Performs logical negation on a variant.

## -parameters

### -param pvarIn [in]

The variant.

### -param pvarResult [out]

The result variant.

## -returns

If this function succeeds, it returns <b>S_OK</b>. Otherwise, it returns an <b>HRESULT</b> error code.
