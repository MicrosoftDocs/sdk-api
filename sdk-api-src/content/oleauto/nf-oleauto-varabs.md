# VarAbs function


## -description

Returns the absolute value of a variant.

## -parameters

### -param pvarIn [in]

The variant.

### -param pvarResult [out]

The result variant.

## -returns

If this function succeeds, it returns <b>S_OK</b>. Otherwise, it returns an <b>HRESULT</b> error code.
