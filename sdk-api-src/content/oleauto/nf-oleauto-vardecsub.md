# VarDecSub function


## -description

Subtracts two variants of type decimal.

## -parameters

### -param pdecLeft [in]

The first variant.

### -param pdecRight [in]

The second variant.

### -param pdecResult [out]

The resulting variant.

## -returns

If this function succeeds, it returns <b>S_OK</b>. Otherwise, it returns an <b>HRESULT</b> error code.
