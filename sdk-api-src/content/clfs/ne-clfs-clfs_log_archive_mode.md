# CLFS_LOG_ARCHIVE_MODE enumeration


## -description


Specifies whether a log is ephemeral.


## -enum-fields




### -field ClfsLogArchiveEnabled

Enables log archive (ephemeral logs) support.


### -field ClfsLogArchiveDisabled

Disables ephemeral logs.
