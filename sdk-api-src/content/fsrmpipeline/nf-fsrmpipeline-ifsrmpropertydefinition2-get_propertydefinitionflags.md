# IFsrmPropertyDefinition2::get_PropertyDefinitionFlags


## -description

<p class="CCE_Message">[This property is supported for compatibility but it's recommended to use the 
    <a href="/previous-versions/windows/desktop/fsrm/fsrm-wmi-classes">FSRM WMI Classes</a> to manage FSRM. Please see the 
    <a href="/previous-versions/windows/desktop/fsrm/msft-fsrmclassificationpropertydefinition">MSFT_FSRMClassificationPropertyDefinition</a> 
    class.]

This property contains the flags for the property definition as enumerated by the 
    <a href="/windows/desktop/api/fsrmenums/ne-fsrmenums-fsrmpropertydefinitionflags">FsrmPropertyDefinitionFlags</a> enumeration.

This property is read-only.
