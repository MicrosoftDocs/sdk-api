# IFsrmPropertyDefinition2::get_ValueDefinitions


## -description


<p class="CCE_Message">[This property is supported for compatibility but it's recommended to use the 
    <a href="https://msdn.microsoft.com/1CE772FA-CE33-4900-A499-058175A7C37E">FSRM WMI Classes</a> to manage FSRM. Please see the 
    <a href="https://msdn.microsoft.com/2221a226-60cd-45db-86c1-eae11ad41ff0">MSFT_FSRMClassificationPropertyDefinition</a> 
    class.]

This property contains the possible value definitions of the property definition.

This property is read-only.
