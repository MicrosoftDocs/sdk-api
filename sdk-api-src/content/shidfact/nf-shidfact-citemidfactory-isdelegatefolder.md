# CItemIDFactory::IsDelegateFolder


## -description


Gets a Boolean value specifying whether the factory is a delegate folder.


## -parameters






## -returns



Returns <b>TRUE</b> if this factory is a delegate folder, otherwise <b>FALSE</b>.
