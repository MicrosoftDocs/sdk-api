# IGraphicsEffectD2D1Interop::effects


## -description


Retrieve the property count for the effect.


## -parameters




### -param count [out]

Type: <b>UINT*</b>

When the method returns, this parameter will contain the property count.


## -returns



Type: <b>HRESULT</b>

This method returns an HRESULT success or error code.
