# IGraphicsEffectD2D1Interop::GetSourceCount (windows.graphics.effects.interop.h)


## -description

Retrieves the source count for the effect.

## -parameters

### -param count [out]

Type: <b>UINT*</b>

When the method returns, this parameter will contain the source count for the effect.

## -returns

Type: <b>HRESULT</b>

This method returns an HRESULT success or error code.
