# IGraphicsEffectD2D1Interop::effects


## -description


Retrieves the ID of the effect.


## -parameters




### -param id [out]

Type: <b>GUID*</b>

When this method returns, this parameter will contain the ID of the effect.


## -returns



Type: <b>HRESULT</b>

This method returns an HRESULT success or error code.
