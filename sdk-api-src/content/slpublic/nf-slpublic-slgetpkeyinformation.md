# SLGetPKeyInformation function


## -description


Gets the information of the specified product key.


## -parameters




### -param hSLC [in]

Type: <b>HSLC</b>

The handle to the current SLC context.


### -param pPKeyId [in]

Type: <b>const SLID*</b>

A pointer to the PKey ID.
