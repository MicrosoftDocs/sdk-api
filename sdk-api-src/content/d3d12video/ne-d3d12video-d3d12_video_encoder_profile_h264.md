## -description

Specifies the encoder profiles for H.264 encoding.

## -enum-fields

### -field D3D12_VIDEO_ENCODER_PROFILE_H264_MAIN

Main profile.

### -field D3D12_VIDEO_ENCODER_PROFILE_H264_HIGH

High profile.

### -field D3D12_VIDEO_ENCODER_PROFILE_H264_HIGH_10

High 10 profile.

## -remarks

Use this enumeration to specify the encoder profile in a [D3D12_VIDEO_ENCODER_PROFILE_DESC](ns-d3d12video-d3d12_video_encoder_profile_desc.md) structure.
