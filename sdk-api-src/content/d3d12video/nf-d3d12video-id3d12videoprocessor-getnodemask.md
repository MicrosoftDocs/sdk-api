# ID3D12VideoProcessor::GetNodeMask

Gets the node mask provided when the video processor was created with a call to [ID3D12VideoDevice::CreateVideoProcessor](nf-d3d12video-id3d12videodevice-createvideoprocessor).


## -description

The node mask specifying the physical adapter on which the video processor will be used.



## -returns

This method returns a UINT.
