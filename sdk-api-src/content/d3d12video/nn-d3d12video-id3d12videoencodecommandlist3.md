## -description

Encapsulates a list of graphics commands for video encoding. This interface derives from [ID3D12VideoEncodeCommandList2](nn-d3d12video-id3d12videoencodecommandlist2.md), and adds support for barriers.

Requires the DirectX 12 Agility SDK 1.7 or later.
