# ITfInputScope::GetSRGS


## -description

Gets the Speech Recognition Grammar Specification (SRGS) string to be recognized.

## -parameters




### -param pbstrSRGS [out]

The xml string. The calling function must call <b>SysFreeString()</b> to free the buffer.
