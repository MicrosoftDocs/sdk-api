# INPUT_MESSAGE_DEVICE_TYPE enumeration


## -description


The type of device that sent the input message.


## -enum-fields




### -field IMDT_UNAVAILABLE

The device type isn't identified.


### -field IMDT_KEYBOARD

Keyboard input.


### -field IMDT_MOUSE

Mouse input.


### -field IMDT_TOUCH

Touch input.


### -field IMDT_PEN

Pen or stylus input.


### -field IMDT_TOUCHPAD

Touchpad input (Windows 8.1 and later).
