# NTSAMPROPERTIES enumeration


## -description


The values of the 
<b>NTSAMPROPERTIES</b> enumeration type specify properties related to the NT Security Accounts Manager (SAM).


## -enum-fields




### -field PROPERTY_NTSAM_ALLOW_LM_AUTHENTICATION

Specifies whether to allow NT LAN Manager (NTLM) authentication.
