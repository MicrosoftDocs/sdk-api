# PROTOCOLPROPERTIES enumeration


## -description


The values of the 
<b>PROTOCOLPROPERTIES</b> enumeration type enumerate properties of an authentication protocol.


## -enum-fields




### -field PROPERTY_PROTOCOL_REQUEST_HANDLER

The value is reserved for system use.


### -field PROPERTY_PROTOCOL_START

This value is the default.
