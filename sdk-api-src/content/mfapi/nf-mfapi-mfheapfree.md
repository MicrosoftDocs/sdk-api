# MFHeapFree function


## -description

Frees a block of memory that was allocated by calling the <a href="/windows/desktop/api/mfapi/nf-mfapi-mfheapalloc">MFHeapAlloc</a> function.

## -parameters

### -param pv [in]

Pointer to the memory to free.
