# ICompositionDrawingSurfaceInterop::composition


## -description


Resumes drawing on the surface object.


## -parameters






## -returns



Type: <b>HRESULT</b>

If this method succeeds, it returns S_OK. Otherwise, it returns an HRESULT error code.
