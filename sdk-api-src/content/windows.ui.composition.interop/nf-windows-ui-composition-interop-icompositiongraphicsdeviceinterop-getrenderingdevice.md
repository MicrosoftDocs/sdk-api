# ICompositionGraphicsDeviceInterop::composition


## -description

Gets the rendering device.

## -parameters

### -param value [out]

Type: <b>IUnknown**</b>

The retrieved rendering device.

## -returns

Type: <b>HRESULT</b>

If this method succeeds, it returns S_OK. Otherwise, it returns an HRESULT error code.
