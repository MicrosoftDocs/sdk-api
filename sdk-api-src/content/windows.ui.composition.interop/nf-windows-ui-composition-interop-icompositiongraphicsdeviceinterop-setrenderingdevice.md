# ICompositionGraphicsDeviceInterop::composition


## -description


Sets the rendering device.


## -parameters




### -param value [in]

Type: <b>IUnknown*</b>

The new rendering device.


## -returns



Type: <b>HRESULT</b>

If this method succeeds, it returns S_OK. Otherwise, it returns an HRESULT error code.
