# ICompositionDrawingSurfaceInterop::composition


## -description


Changes the size of the surface.


## -parameters




### -param sizePixels [in]

Type: <b>SIZE</b>

Width and height of the surface in pixels.


## -returns



Type: <b>HRESULT</b>

If this method succeeds, it returns S_OK. Otherwise, it returns an HRESULT error code.
