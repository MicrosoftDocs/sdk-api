# Ipv6Address structure


## -description

<div class="alert"><b>Note</b>  The Network Access Protection platform is not available starting with Windows 10</div><div> </div>The <b>Ipv6Address</b> structure defines an IPv6 address.

## -struct-fields

### -field addr

An array that contains an IPv6 network address in network byte order.
