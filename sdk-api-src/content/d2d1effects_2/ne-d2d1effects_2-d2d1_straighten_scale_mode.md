# D2D1_STRAIGHTEN_SCALE_MODE enumeration


## -description

Values for the <a href="/windows/desktop/api/d2d1effects_2/ne-d2d1effects_2-d2d1_straighten_prop">D2D1_STRAIGHTEN_PROP_SCALE_MODE</a> property of the <a href="/windows/desktop/Direct2D/straighten-effect">Straighten effect</a>.

## -enum-fields

### -field D2D1_STRAIGHTEN_SCALE_MODE_NEAREST_NEIGHBOR:0

Indicates nearest neighbor interpolation should be used.

### -field D2D1_STRAIGHTEN_SCALE_MODE_LINEAR:1

Indicates linear interpolation should be used.

### -field D2D1_STRAIGHTEN_SCALE_MODE_CUBIC:2

Indicates cubic interpolation should be used.

### -field D2D1_STRAIGHTEN_SCALE_MODE_MULTI_SAMPLE_LINEAR:3

Indicates multi-sample linear interpolation should be used.

### -field D2D1_STRAIGHTEN_SCALE_MODE_ANISOTROPIC:4

Indicates anisotropic filtering should be used.

### -field D2D1_STRAIGHTEN_SCALE_MODE_FORCE_DWORD:0xffffffff
