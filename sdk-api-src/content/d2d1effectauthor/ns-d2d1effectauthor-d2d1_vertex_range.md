# D2D1_VERTEX_RANGE structure


## -description


Defines a range of vertices that are used when rendering less than the full contents of a vertex buffer.


## -struct-fields




### -field startVertex

The first vertex in the range to process.


### -field vertexCount

The number of vertices to use.
