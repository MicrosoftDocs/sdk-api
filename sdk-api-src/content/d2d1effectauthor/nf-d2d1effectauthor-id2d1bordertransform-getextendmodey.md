# ID2D1BorderTransform::GetExtendModeY


## -description

Gets the extend mode in the y direction.



## -returns

Type: <b><a href="/windows/desktop/api/d2d1/ne-d2d1-d2d1_extend_mode">D2D1_EXTEND_MODE</a></b>

This method returns the extend mode in the y direction.
