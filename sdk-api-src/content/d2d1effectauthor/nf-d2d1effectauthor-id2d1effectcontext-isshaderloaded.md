# ID2D1EffectContext::IsShaderLoaded


## -description

This tests to see if the given shader is loaded.

## -parameters

### -param shaderId

Type: <b>REFGUID</b>

The unique id that identifies the shader.

## -returns

Type: <b>BOOL</b>

Whether the shader is loaded.
