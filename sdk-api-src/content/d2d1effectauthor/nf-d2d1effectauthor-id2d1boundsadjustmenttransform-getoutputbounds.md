# ID2D1BoundsAdjustmentTransform::GetOutputBounds


## -description


Returns the output rectangle of the support transform.

Type: <b><a href="https://docs.microsoft.com/previous-versions/windows/desktop/legacy/hh847950(v=vs.85)">D2D1_RECT_L</a>*</b>

The output bounds.


## -returns



This method does not return a value.
