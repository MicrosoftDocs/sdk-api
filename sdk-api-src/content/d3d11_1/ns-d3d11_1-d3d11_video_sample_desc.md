# D3D11_VIDEO_SAMPLE_DESC structure


## -description

Describes a video sample.

## -struct-fields

### -field Width

The width of the video sample.

### -field Height

The height of the video sample.

### -field Format

The format of the video sample.

### -field ColorSpace

The colorspace of the sample.
