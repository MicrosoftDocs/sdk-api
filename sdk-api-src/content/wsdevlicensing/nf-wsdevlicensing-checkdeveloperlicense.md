# CheckDeveloperLicense function


## -description


Checks to see if a developer license exists.


## -parameters




### -param pExpiration [out]

Indicates when the developer license expires.


## -returns



Returns an <a href="https://docs.microsoft.com/uwp/api/windows.foundation.hresult">HResult structure</a> with any error codes that occurred.
