# WarpMode enumeration


## -description

The <b>WarpMode</b> enumeration specifies warp modes that can be used to transform images.

## -enum-fields

### -field WarpModePerspective

Specifies the perspective warp mode.

### -field WarpModeBilinear

Specifies the bilinear warp mode.
