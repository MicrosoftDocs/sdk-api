# IDWriteFontFace::GetType


## -description


 Obtains the file format type of a font face.


## -parameters






## -returns



Type: <b><a href="/windows/win32/api/dwrite/ne-dwrite-dwrite_font_face_type">DWRITE_FONT_FACE_TYPE</a></b>

A value that indicates the type of format for the font face (such as Type 1, TrueType, vector, or bitmap).
