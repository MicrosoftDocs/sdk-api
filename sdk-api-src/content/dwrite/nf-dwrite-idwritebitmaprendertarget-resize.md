# IDWriteBitmapRenderTarget::Resize


## -description

 Resizes the bitmap.

## -parameters

### -param width

Type: <b>UINT32</b>

The new bitmap width, in pixels.

### -param height

Type: <b>UINT32</b>

The new bitmap height, in pixels.

## -returns

Type: <b>HRESULT</b>

If this method succeeds, it returns <b>S_OK</b>. Otherwise, it returns an <b>HRESULT</b> error code.
