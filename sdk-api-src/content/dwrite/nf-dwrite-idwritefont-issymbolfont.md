# IDWriteFont::IsSymbolFont


## -description


 Determines whether the font is a symbol font.


## -parameters






## -returns



Type: <b>BOOL</b>

<b>TRUE</b> if the font is a symbol font; otherwise, <b>FALSE</b>.
