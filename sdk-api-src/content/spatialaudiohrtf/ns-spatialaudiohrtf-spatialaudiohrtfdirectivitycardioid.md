# SpatialAudioHrtfDirectivityCardioid structure


## -description

Represents a cardioid-shaped directivity model for an <a href="/windows/desktop/api/spatialaudiohrtf/nn-spatialaudiohrtf-ispatialaudioobjectforhrtf">ISpatialAudioObjectForHrtf</a>.

## -struct-fields

### -field directivity

A structure that expresses the direction in which sound is emitted by an <a href="/windows/desktop/api/spatialaudiohrtf/nn-spatialaudiohrtf-ispatialaudioobjectforhrtf">ISpatialAudioObjectForHrtf</a>.

### -field Order

The order of the cardioid.
