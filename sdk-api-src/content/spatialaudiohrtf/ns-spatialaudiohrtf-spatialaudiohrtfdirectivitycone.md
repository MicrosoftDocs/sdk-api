# SpatialAudioHrtfDirectivityCone structure


## -description


Represents a cone-shaped directivity model for an <a href="https://msdn.microsoft.com/E69F1D09-B937-4BCC-A040-18EF8A838289">ISpatialAudioObjectForHrtf</a>.


## -struct-fields




### -field directivity

A structure that expresses the direction in which sound is emitted by an <a href="https://msdn.microsoft.com/E69F1D09-B937-4BCC-A040-18EF8A838289">ISpatialAudioObjectForHrtf</a>.


### -field InnerAngle

The inner angle of the cone.


### -field OuterAngle

The outer angle of the cone.
