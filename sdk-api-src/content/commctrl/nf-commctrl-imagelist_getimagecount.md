# ImageList_GetImageCount function


## -description

Retrieves the number of images in an image list.

## -parameters

### -param himl [in]

Type: <b>HIMAGELIST</b>

A handle to the image list.

## -returns

Type: <b>int</b>

Returns the number of images.
