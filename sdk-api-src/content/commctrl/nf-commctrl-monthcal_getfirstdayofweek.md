# MonthCal_GetFirstDayOfWeek macro


## -description


Retrieves the first day of the week for a month calendar control. You can use this macro or send the <a href="https://docs.microsoft.com/windows/desktop/Controls/mcm-getfirstdayofweek">MCM_GETFIRSTDAYOFWEEK</a> message explicitly. 


## -parameters




### -param hmc

Type: <b><a href="https://docs.microsoft.com/windows/desktop/WinProg/windows-data-types">HWND</a></b>

Handle to a month calendar control.
