# ListView_GetSelectedCount macro


## -description


Determines the number of selected items in a list-view control. You can use this macro or send the <a href="https://msdn.microsoft.com/library/Bb761069(v=VS.85).aspx">LVM_GETSELECTEDCOUNT</a> message explicitly.
