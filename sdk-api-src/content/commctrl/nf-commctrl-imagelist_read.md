# ImageList_Read function


## -description


Reads an image list from a stream. 


## -parameters




### -param pstm

Type: <b>LPSTREAM</b>

A pointer to the stream. 


## -returns



Type: <b>HIMAGELIST</b>

Returns the handle to the image list if successful, or <b>NULL</b> otherwise.
