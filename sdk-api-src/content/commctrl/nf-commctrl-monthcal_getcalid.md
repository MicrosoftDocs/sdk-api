# MonthCal_GetCALID macro


## -description


Gets the current calendar ID for the given calendar control. You can use this macro or send the <a href="https://msdn.microsoft.com/library/Bb760951(v=VS.85).aspx">MCM_GETCALID</a> message explicitly.


## -parameters




### -param hmc

Type: <b><a href="https://msdn.microsoft.com/4553cafc-450e-4493-a4d4-cb6e2f274d46">HWND</a></b>

Handle to a month calendar control.
