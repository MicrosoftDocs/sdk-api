# ListView_GetUnicodeFormat macro


## -description


Gets the Unicode character format flag for the control. You can use this macro or send the <a href="https://docs.microsoft.com/windows/desktop/Controls/lvm-getunicodeformat">LVM_GETUNICODEFORMAT</a> message explicitly. 


## -parameters




### -param hwnd

Type: <b><a href="https://docs.microsoft.com/windows/desktop/WinProg/windows-data-types">HWND</a></b>

A handle to the control.
