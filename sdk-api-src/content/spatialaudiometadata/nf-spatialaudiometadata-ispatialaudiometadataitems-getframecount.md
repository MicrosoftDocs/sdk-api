# ISpatialAudioMetadataItems::GetFrameCount


## -description


Gets the total frame count of the <a href="https://msdn.microsoft.com/54A6B7DE-A41E-4214-AF02-CC19250B9037">ISpatialAudioMetadataItems</a>, which defines valid item offsets.


## -parameters




### -param frameCount [out]

The total frame count.


## -returns



If the method succeeds, it returns S_OK.
