# ISpatialAudioMetadataItems::GetMaxItemCount


## -description


The maximum number of items allowed by the <a href="https://msdn.microsoft.com/54A6B7DE-A41E-4214-AF02-CC19250B9037">ISpatialAudioMetadataItems</a>, defined when the object is created.


## -parameters




### -param maxItemCount






#### - frameCount [out]

The maximum number of items allowed.


## -returns



If the method succeeds, it returns S_OK.
