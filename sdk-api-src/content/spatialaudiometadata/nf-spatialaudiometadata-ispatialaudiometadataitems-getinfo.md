# ISpatialAudioMetadataItems::GetInfo


## -description


Gets the total frame count for the <a href="https://docs.microsoft.com/windows/desktop/api/spatialaudiometadata/nn-spatialaudiometadata-ispatialaudiometadataitems">ISpatialAudioMetadataItems</a>, which defines valid item offsets.


## -parameters




### -param info [out]

The total frame count, which defines valid item offsets.


## -returns



If the method succeeds, it returns S_OK.
