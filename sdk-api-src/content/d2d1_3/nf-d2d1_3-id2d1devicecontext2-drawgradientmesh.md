# ID2D1DeviceContext2::DrawGradientMesh


## -description


Renders a given gradient mesh to the target.

Type: <b><a href="https://docs.microsoft.com/windows/desktop/api/d2d1_3/nn-d2d1_3-id2d1gradientmesh">ID2D1GradientMesh</a>*</b>
