# ID2D1Factory7 interface


## -description


Creates Direct2D resources. This interface performs all the same functions as the <a href="https://msdn.microsoft.com/79656026-7197-4EF4-A8DD-7E65D43D1F18">ID2D1Factory6</a> interface, 
        plus it enables the creation of <a href="https://msdn.microsoft.com/4C8AA3D6-E08E-45CE-958A-DBC8C9EB7B97">ID2D1Device6</a> objects.
