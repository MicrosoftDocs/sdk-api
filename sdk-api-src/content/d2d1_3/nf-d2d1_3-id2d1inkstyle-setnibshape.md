# ID2D1InkStyle::SetNibShape


## -description

Sets the pre-transform nib shape for this style.

## -parameters

### -param nibShape

Type: <b><a href="/windows/desktop/api/d2d1_3/ne-d2d1_3-d2d1_ink_nib_shape">D2D1_INK_NIB_SHAPE</a></b>

The pre-transform nib shape to use in this style.
