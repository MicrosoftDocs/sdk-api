# ID2D1DeviceContext5 interface


## -description

This interface performs all the same functions as the <a href="/windows/desktop/api/d2d1_3/nn-d2d1_3-id2d1devicecontext4">ID2D1DeviceContext4</a> interface, 
        plus it enables the creation of color contexts and Svg documents.
