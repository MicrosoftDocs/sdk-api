# ID2D1ImageSourceFromWic::GetSource


## -description


Retrieves the underlying bitmap image source from the Windows Imaging Component (WIC).


## -parameters




### -param wicBitmapSource

Type: <b><a href="https://msdn.microsoft.com/abcc84af-6067-4856-8618-fb66aff4255a">IWICBitmapSource</a>**</b>

On return contains the bitmap image source.


## -returns



This method does not return a value.
