# ID2D1Device3 interface


## -description

Represents a resource domain whose objects and device contexts can be used together.
        This interface performs all the same functions as the <a href="/windows/desktop/api/d2d1_3/nn-d2d1_3-id2d1device2">ID2D1Device2</a> interface. 
        It also enables the creation of <a href="/windows/desktop/api/d2d1_3/nn-d2d1_3-id2d1devicecontext3">ID2D1DeviceContext3</a> objects.
