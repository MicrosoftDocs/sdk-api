# IUIAutomationElement9::get_CurrentIsDialog


## -description

Retrieves the current dialog window indicator for the element.

This property is read-only.

## -parameters

## -remarks

When the <b>CurrentIsDialog</b> property is <b>TRUE</b>, a client application can assume the current window is a dialog.
