## -description

Enables interoperability with a WinRT [UserActivity](/uwp/api/windows.applicationmodel.useractivities.useractivity) object and provides the ability to create a [UserActivitySession](/uwp/api/windows.applicationmodel.useractivities.useractivitysession) associated with a window.
