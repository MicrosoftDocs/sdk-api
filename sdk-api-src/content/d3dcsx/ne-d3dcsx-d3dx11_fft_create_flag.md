# D3DX11_FFT_CREATE_FLAG enumeration


## -description


FFT creation flags.


## -enum-fields




### -field D3DX11_FFT_CREATE_FLAG_NO_PRECOMPUTE_BUFFERS

Do not AddRef or Release temp and precompute buffers, caller is responsible for holding 
          references to these buffers.
