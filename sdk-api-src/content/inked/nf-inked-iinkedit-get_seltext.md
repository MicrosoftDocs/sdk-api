# IInkEdit::get_SelText


## -description


Gets or sets the selected text within the <a href="https://msdn.microsoft.com/a1dfa254-cade-44c5-8fdd-74bb40849063">InkEdit</a> control (run time only).

This property is read/write.


## -parameters


## -remarks



Setting <b>SelText</b> to a new value sets <a href="https://msdn.microsoft.com/6295a536-831e-4603-9412-87b2fa5b7f53">SelLength</a> to 0 and replaces the selected text with the new string.
