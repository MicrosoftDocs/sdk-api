# BorderStyleConstants enumeration


## -description

Specifies how the borders of an <a href="/windows/desktop/tablet/inkedit-control-reference">InkEdit</a> control appear on the screen.

## -enum-fields

### -field rtfNoBorder

No border.

### -field rtfFixedSingle

Single line border.
