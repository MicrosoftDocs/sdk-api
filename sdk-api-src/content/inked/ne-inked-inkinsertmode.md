## -description

Specifies how ink is inserted onto the <a href="/windows/win32/tablet/inkedit-control-reference">InkEdit</a> control.

## -enum-fields

### -field IEM_InsertText

Default. Drawn ink is inserted as text.

### -field IEM_InsertInk

Drawn ink is inserted as ink.
