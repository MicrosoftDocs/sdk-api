# AppearanceConstants enumeration


## -description



Specifies how an <a href="https://msdn.microsoft.com/52761cb2-4433-4824-ba19-fe597de2faf0">InkEdit</a> control appears on the screen.




## -enum-fields




### -field rtfFlat

Flat. Paints without visual effects.


### -field rtfThreeD

Default. 3D. Paints with three-dimensional effects.
