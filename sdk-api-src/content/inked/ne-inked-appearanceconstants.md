# AppearanceConstants enumeration


## -description

Specifies how an <a href="/windows/desktop/tablet/inkedit-control-reference">InkEdit</a> control appears on the screen.

## -enum-fields

### -field rtfFlat:0

Flat. Paints without visual effects.

### -field rtfThreeD:1

Default. 3D. Paints with three-dimensional effects.
