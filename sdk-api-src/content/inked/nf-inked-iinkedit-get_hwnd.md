# IInkEdit::get_Hwnd


## -description

Gets a handle to the <a href="/windows/desktop/tablet/inkedit-control-reference">InkEdit</a> control.

This property is read-only.

## -parameters

## -remarks

<div class="alert"><b>Note</b>  Because the value of this property can change while a program is running, never store the Hwnd value in a variable.
</div>
