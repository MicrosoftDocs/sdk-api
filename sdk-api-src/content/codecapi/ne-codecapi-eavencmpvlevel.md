# eAVEncMPVLevel enumeration


## -description



Specifies the MPEG-2 profile. This enumeration is used with the <a href="https://docs.microsoft.com/windows/desktop/DirectShow/avencmpvlevel-property">AVEncMPVLevel</a> property.




## -enum-fields




### -field eAVEncMPVLevel_Low

Low Level.


### -field eAVEncMPVLevel_Main

Main Level.


### -field eAVEncMPVLevel_High1440

High 1440 Level.


### -field eAVEncMPVLevel_High

High Level.
