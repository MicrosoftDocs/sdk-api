# eAVEncDDPreferredStereoDownMixMode enumeration


## -description



Specifies the preferred stereo downmix mode for a Dolby Digital audio stream. This enumeration is used with the <a href="https://msdn.microsoft.com/3cf9fba7-6895-4dfb-9aef-571c512b7955">AVEncDDPreferredStereoDownMixMode</a> property.




## -enum-fields




### -field eAVEncDDPreferredStereoDownMixMode_LtRt

Left total/right total (Lt/Rt) downmix.


### -field eAVEncDDPreferredStereoDownMixMode_LoRo

Left only/right only (Lo/Ro) downmix.
