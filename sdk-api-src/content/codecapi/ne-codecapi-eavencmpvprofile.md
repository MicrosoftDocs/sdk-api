# eAVEncMPVProfile enumeration


## -description



Specifies the MPEG-2 profile. This enumeration is used with the <a href="https://docs.microsoft.com/windows/desktop/DirectShow/avencmpvprofile-property">AVEncMPVProfile</a> property.




## -enum-fields




### -field eAVEncMPVProfile_unknown

The profile is not known.


### -field eAVEncMPVProfile_Simple

Simple Profile.


### -field eAVEncMPVProfile_Main

Main Profile.


### -field eAVEncMPVProfile_High

High Profile.


### -field eAVEncMPVProfile_422

4:2:2 Profile.
