# eAVEncVideoFilmContent enumeration


## -description



Specifies whether the original source of the input video was film or video. This enumeration is used with the <a href="https://docs.microsoft.com/windows/desktop/DirectShow/avencvideosourcefilmcontent-property">AVEncVideoSourceFilmContent</a> property.




## -enum-fields




### -field eAVEncVideoFilmContent_VideoOnly

The original source was video.


### -field eAVEncVideoFilmContent_FilmOnly

The original source was film.


### -field eAVEncVideoFilmContent_Mixed

The original source contains a mix of video and film.
