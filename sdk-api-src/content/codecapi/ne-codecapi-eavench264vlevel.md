# eAVEncH264VLevel enumeration


## -description


Specifies an H.264 encoding level.


## -enum-fields

Level 1b.

Level 4.1.


### -field eAVEncH264VLevel4_2

Level 4.2.

Level 5.


### -field eAVEncH264VLevel5_1

Level 5.1.
