# eAVEncH264VLevel enumeration


## -description

Specifies an H.264 encoding level.

## -enum-fields

### -field eAVEncH264VLevel1:10

Level 1.

### -field eAVEncH264VLevel1_b:11

Level 1b.

### -field eAVEncH264VLevel1_1:11

Level 1.1.

### -field eAVEncH264VLevel1_2:12

Level 1.2.

### -field eAVEncH264VLevel1_3:13

Level 1.3.

### -field eAVEncH264VLevel2:20

Level 2.

### -field eAVEncH264VLevel2_1:21

Level 2.1.

### -field eAVEncH264VLevel2_2:22

Level 2.2.

### -field eAVEncH264VLevel3:30

Level 3.

### -field eAVEncH264VLevel3_1:31

Level 3.1.

### -field eAVEncH264VLevel3_2:32

Level 3.2.

### -field eAVEncH264VLevel4:40

Level 4.

### -field eAVEncH264VLevel4_1:41

Level 4.1.

### -field eAVEncH264VLevel4_2:42

Level 4.2.

### -field eAVEncH264VLevel5:50

Level 5.

### -field eAVEncH264VLevel5_1:51

Level 5.1.

### -field eAVEncH264VLevel5_2:52

Level 5.2

Supported in Windows 8.1 and later.

## -remarks

This values are used with the <a href="/windows/desktop/medfound/mf-mt-mpeg2-level-attribute">MF_MT_MPEG2_LEVEL</a> attribute.
