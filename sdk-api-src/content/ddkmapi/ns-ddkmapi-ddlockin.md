# DDLOCKIN structure


## -description


The DDLOCKIN structure contains the Microsoft DirectDraw object and DirectDraw surface handle information. 


## -struct-fields




### -field hDirectDraw

Specifies the DirectDraw handle.


### -field hSurface

Specifies the DirectDrawSurface handle.
