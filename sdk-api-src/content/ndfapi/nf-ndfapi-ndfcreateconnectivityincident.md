# NdfCreateConnectivityIncident function


## -description


The <b>NdfCreateConnectivityIncident</b> function diagnoses generic Internet connectivity problems.


## -parameters




### -param handle [out]

Type: <b>NDFHANDLE*</b>

Handle to the Network Diagnostics Framework incident.
