# ITypeLib::ReleaseTLibAttr


## -description

Releases the TLIBATTR originally obtained from <a href="/previous-versions/windows/desktop/api/oaidl/nf-oaidl-itypelib-getlibattr">GetLibAttr</a>.

## -parameters

### -param pTLibAttr [in]

The TLIBATTR to be freed.
