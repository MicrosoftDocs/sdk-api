# ITypeInfo2::GetFuncCustData


## -description


Gets the custom data from the specified function.


## -parameters




### -param index [in]

The index of the function for which to get the custom data.




### -param guid [in]

The GUID used to identify the data.


### -param pVarVal [out]

The custom data.
