# ICreateTypeInfo2::DeleteFuncDescByMemId


## -description


Deletes the specified function description (FUNCDESC).


## -parameters




### -param memid [in]

The member identifier of the FUNCDESC to delete.




### -param invKind [in]

The type of the invocation.
