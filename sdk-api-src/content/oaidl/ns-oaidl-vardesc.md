# VARDESC structure


## -description


Describes a variable, constant, or data member.


## -struct-fields




### -field memid

The member ID.


### -field lpstrSchema

Reserved.


### -field DUMMYUNIONNAME

 


### -field DUMMYUNIONNAME.oInst

With VAR_PERINSTANCE, the offset of this variable within the instance.


### -field DUMMYUNIONNAME.lpvarValue

With VAR_CONST, the value of the constant.


### -field elemdescVar

The variable type.


### -field wVarFlags

The variable flags. See <a href="https://docs.microsoft.com/windows/desktop/api/oaidl/ne-oaidl-varflags">VARFLAGS</a>.


### -field varkind

The variable type.
