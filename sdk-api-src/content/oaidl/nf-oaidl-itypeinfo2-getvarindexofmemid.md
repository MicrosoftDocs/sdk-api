# ITypeInfo2::GetVarIndexOfMemId


## -description

Binds to a specific member based on a known DISPID, where the member name is not known (for example, when binding to a default member).

## -parameters

### -param memid [in]

The member identifier.

### -param pVarIndex [out]

The index.
