# ITypeInfo2::GetImplTypeCustData


## -description


Gets the custom data of the implementation type.


## -parameters




### -param index [in]

The index of the implementation type for the custom data.


### -param guid [in]

The GUID used to identify the data.


### -param pVarVal [out]

The retrieved data.
