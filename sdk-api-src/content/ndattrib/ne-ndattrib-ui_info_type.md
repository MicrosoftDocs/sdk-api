# UI_INFO_TYPE enumeration


## -description


The <b>UI_INFO_TYPE</b> enumeration identifies repairs that perform user interface tasks.


## -enum-fields




### -field UIT_INVALID


### -field UIT_NONE

No additional repair interfaces are present.


### -field UIT_SHELL_COMMAND

Execute shell command.


### -field UIT_HELP_PANE

Launch help pane.


### -field UIT_DUI

Direct UI.
