# LIFE_TIME structure


## -description


The <b>LIFE_TIME</b> structure contains a start time and an end time.


## -struct-fields




### -field startTime

Type: <b>FILETIME</b>

The time the problem instance began.


### -field endTime

Type: <b>FILETIME</b>

The time the problem instance ended.
