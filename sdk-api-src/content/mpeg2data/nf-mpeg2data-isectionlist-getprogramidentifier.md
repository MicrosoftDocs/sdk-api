# ISectionList::GetProgramIdentifier


## -description



The <b>GetProgramIdentifier</b> method retrieves the program identifier (PID) of the packets that this object is receiving.




## -parameters




### -param pPid

Receives the PID.


## -returns



The method returns an <b>HRESULT</b>. Possible values include those in the following table.

<table>
<tr>
<th>Return code</th>
<th>Description</th>
</tr>
<tr>
<td width="40%">
<dl>
<dt><b>E_POINTER</b></dt>
</dl>
</td>
<td width="60%">
NULL pointer argument.

</td>
</tr>
<tr>
<td width="40%">
<dl>
<dt><b>S_OK</b></dt>
</dl>
</td>
<td width="60%">
The method succeeded.

</td>
</tr>
</table>
 




## -remarks



The PID value is set when the object is first initialized.
