# IFsrmQuotaManager::get_ActionVariableDescriptions


## -description


<p class="CCE_Message">[This property is supported for compatibility but it's recommended to use the 
    <a href="https://msdn.microsoft.com/1CE772FA-CE33-4900-A499-058175A7C37E">FSRM WMI Classes</a> to manage FSRM. Please see the 
    <a href="https://msdn.microsoft.com/9308f1de-ba8e-45f5-81ec-d8203839ee79">MSFT_FSRMQuota</a> class.]

Retrieves the descriptions for the macros contained in the 
    <a href="https://msdn.microsoft.com/e7fe0139-692b-4f88-8411-ffd31d29e40d">IFsrmQuotaManager::ActionVariables</a> 
    property.

This property is read-only.
