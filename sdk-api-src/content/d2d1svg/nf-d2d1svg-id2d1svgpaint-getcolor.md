# ID2D1SvgPaint::GetColor


## -description

Gets the paint color that is used if the paint type is D2D1_SVG_PAINT_TYPE_COLOR.

## -parameters

### -param color [out]

Type: <b>D2D1_COLOR_F*</b>

The paint color that is used if the paint type is D2D1_SVG_PAINT_TYPE_COLOR.
