# ID2D1SvgElement::SetAttributeValue(PCWSTR,const D2D1_SVG_LENGTH &)


## -description

Sets an attribute of this element as a length value.

## -parameters

### -param name [in]

Type: <b>PCWSTR</b>

Name of the attribute to set.

### -param value [ref]

Type: <b>const <a href="/windows/desktop/api/d2d1svg/ns-d2d1svg-d2d1_svg_length">D2D1_SVG_LENGTH</a></b>

The new value of the attribute.

## -returns

Type: <b><a href="/windows/win32/com/structure-of-com-error-codes">HRESULT</a></b>

This method returns an HRESULT success or error code.
