# ID2D1SvgElement::SetAttributeValue(PCWSTR,D2D1_EXTEND_MODE)


## -description


Sets an attribute of this element as an extend mode value. This method can be used to set the value of a spreadMethod attribute.


## -parameters




### -param name [in]

Type: <b>PCWSTR</b>

Name of the attribute to set.


### -param value

Type: <b><a href="https://msdn.microsoft.com/6b6e1fe1-d43a-46cf-904d-5266b9bd6bf4">D2D1_EXTEND_MODE</a></b>

The new value of the attribute.


## -returns



Type: <b><a href="455d07e9-52c3-4efb-a9dc-2955cbfd38cc">HRESULT</a></b>

This method returns an HRESULT success or error code.
