# ID2D1SvgElement::SetAttributeValue(PCWSTR,const D2D1_SVG_PRESERVE_ASPECT_RATIO &)


## -description


Sets an attribute of this element as a preserve aspect ratio value. This method can be used to set the value of a preserveAspectRatio attribute.


## -parameters




### -param name [in]

Type: <b>PCWSTR</b>

Name of the attribute to set.


### -param value [ref]

Type: <b>const <a href="https://docs.microsoft.com/windows/desktop/api/d2d1svg/ns-d2d1svg-d2d1_svg_preserve_aspect_ratio">D2D1_SVG_PRESERVE_ASPECT_RATIO</a></b>

The new value of the attribute.


## -returns



Type: <b><a href="/windows/win32/com/structure-of-com-error-codes">HRESULT</a></b>

This method returns an HRESULT success or error code.
