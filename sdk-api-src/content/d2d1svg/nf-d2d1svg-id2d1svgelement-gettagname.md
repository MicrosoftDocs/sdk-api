# ID2D1SvgElement::GetTagName


## -description

Gets the tag name.

## -parameters

### -param name [out]

Type: <b>PWSTR</b>

The tag name.

### -param nameCount

Type: <b>UINT32</b>

Length of the value in the name argument.

## -returns

Type: <b><a href="/windows/win32/com/structure-of-com-error-codes">HRESULT</a></b>

This method returns an HRESULT success or error code.
