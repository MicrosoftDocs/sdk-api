# ID2D1SvgElement::GetAttributeValue(PCWSTR,D2D1_SVG_VISIBILITY)


## -description


Gets an attribute of this element as a visibility value. This method can be used to get the value of the visibility property.


## -parameters




### -param name [in]

Type: <b>PCWSTR</b>

The name of the attribute.


### -param value [out]

Type: <b><a href="https://docs.microsoft.com/windows/desktop/api/d2d1svg/ne-d2d1svg-d2d1_svg_visibility">D2D1_SVG_VISIBILITY</a>*</b>

The value of the attribute.


## -returns



Type: <b><a href="/windows/win32/com/structure-of-com-error-codes">HRESULT</a></b>

This method returns an HRESULT success or error code.
