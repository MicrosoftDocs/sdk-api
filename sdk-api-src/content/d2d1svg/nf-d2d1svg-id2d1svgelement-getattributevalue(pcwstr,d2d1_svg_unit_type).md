# ID2D1SvgElement::GetAttributeValue(PCWSTR,D2D1_SVG_UNIT_TYPE)


## -description


Gets an attribute of this element as a unit type value. This method can be used to get the value of a gradientUnits or clipPathUnits attribute.


## -parameters




### -param name [in]

Type: <b>PCWSTR</b>

The name of the attribute.


### -param value [out]

Type: <b><a href="https://msdn.microsoft.com/EF7CD98A-AE8D-42B3-BA1F-D50EB2FB88A6">D2D1_SVG_UNIT_TYPE</a>*</b>

The value of the attribute.


## -returns



Type: <b><a href="455d07e9-52c3-4efb-a9dc-2955cbfd38cc">HRESULT</a></b>

This method returns an HRESULT success or error code.
