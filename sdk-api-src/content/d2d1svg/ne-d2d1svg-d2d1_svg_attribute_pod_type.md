# D2D1_SVG_ATTRIBUTE_POD_TYPE enumeration


## -description

Defines the type of SVG POD attribute to set or get.

## -enum-fields

### -field D2D1_SVG_ATTRIBUTE_POD_TYPE_FLOAT:0

The attribute is a FLOAT.

### -field D2D1_SVG_ATTRIBUTE_POD_TYPE_COLOR:1

The attribute is a D2D1_COLOR_F.

### -field D2D1_SVG_ATTRIBUTE_POD_TYPE_FILL_MODE:2

The attribute is a D2D1_FILL_MODE.

### -field D2D1_SVG_ATTRIBUTE_POD_TYPE_DISPLAY:3

The attribute is a D2D1_SVG_DISPLAY.

### -field D2D1_SVG_ATTRIBUTE_POD_TYPE_OVERFLOW:4

The attribute is a D2D1_SVG_OVERFLOW.

### -field D2D1_SVG_ATTRIBUTE_POD_TYPE_LINE_CAP:5

The attribute is a D2D1_SVG_LINE_CAP.

### -field D2D1_SVG_ATTRIBUTE_POD_TYPE_LINE_JOIN:6

The attribute is a D2D1_SVG_LINE_JOIN.

### -field D2D1_SVG_ATTRIBUTE_POD_TYPE_VISIBILITY:7

The attribute is a D2D1_SVG_VISIBILITY.

### -field D2D1_SVG_ATTRIBUTE_POD_TYPE_MATRIX:8

The attribute is a D2D1_MATRIX_3X2_F.

### -field D2D1_SVG_ATTRIBUTE_POD_TYPE_UNIT_TYPE:9

The attribute is a D2D1_SVG_UNIT_TYPE.

### -field D2D1_SVG_ATTRIBUTE_POD_TYPE_EXTEND_MODE:10

The attribute is a D2D1_EXTEND_MODE.

### -field D2D1_SVG_ATTRIBUTE_POD_TYPE_PRESERVE_ASPECT_RATIO:11

The attribute is a D2D1_SVG_PRESERVE_ASPECT_RATIO.

### -field D2D1_SVG_ATTRIBUTE_POD_TYPE_VIEWBOX:12

The attribute is a D2D1_SVG_VIEWBOX.

### -field D2D1_SVG_ATTRIBUTE_POD_TYPE_LENGTH:13

The attribute is a D2D1_SVG_LENGTH.

### -field D2D1_SVG_ATTRIBUTE_POD_TYPE_FORCE_DWORD:0xffffffff
