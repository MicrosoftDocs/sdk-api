## -description

Sets the value of a text content element.

## -parameters

### -param name [in]

Type: <b>const WCHAR*</b>

The new value of the text content element.

### -param nameCount

Type: <b>UINT32</b>

## -returns

This method returns an HRESULT success or error code.
