# IDCompositionArithmeticCompositeEffect::SetCoefficient3


## -description

Sets the third coefficient for the equation used to composite the two input images.

## -parameters

### -param Coefficient3 [in]

Type: <b>float</b>

The third coefficient for the equation used to composite the two input images.

## -returns

Type: <b><a href="/windows/win32/com/structure-of-com-error-codes">HRESULT</a></b>

If this method succeeds, it returns <b>S_OK</b>. Otherwise, it returns an <b>HRESULT</b> error code.
