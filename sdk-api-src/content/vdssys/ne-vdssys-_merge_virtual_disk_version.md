# _MERGE_VIRTUAL_DISK_VERSION enumeration


## -description


Contains the version of the virtual hard disk (VHD) 
    <a href="https://msdn.microsoft.com/5bb84e11-677d-42f5-b5b6-59f23b9a8ab7">MERGE_VIRTUAL_DISK_PARAMETERS</a> structure to 
    use in calls to VHD functions.


## -enum-fields




### -field MERGE_VIRTUAL_DISK_VERSION_UNSPECIFIED

Not supported.


### -field MERGE_VIRTUAL_DISK_VERSION_1

The <b>Version1</b> member structure will be used.


### -field MERGE_VIRTUAL_DISK_VERSION_2

The <b>Version2</b> member structure will be used.

<b>Windows 7 and Windows Server 2008 R2:  </b>This value is not supported until Windows 8 and Windows Server 2012.
