# CERT_SYSTEM_STORE_INFO structure


## -description

The <b>CERT_SYSTEM_STORE_INFO</b> structure contains information used by functions that work with system stores. Currently, no essential information is contained in this structure.

## -struct-fields

### -field cbSize

Size of this structure in bytes.

## -remarks

Currently, no system store information is persisted.
