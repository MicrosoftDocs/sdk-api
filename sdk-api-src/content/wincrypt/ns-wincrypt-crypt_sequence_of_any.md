# CRYPT_SEQUENCE_OF_ANY structure


## -description


The <b>CRYPT_SEQUENCE_OF_ANY</b> structure contains an arbitrary list of encoded BLOBs.


## -struct-fields




### -field cValue

Number of elements in the <b>rgValue</b> array.


### -field rgValue

An array of 
<a href="https://docs.microsoft.com/previous-versions/windows/desktop/legacy/aa381414(v=vs.85)">CRYPT_DER_BLOB</a> structures.
