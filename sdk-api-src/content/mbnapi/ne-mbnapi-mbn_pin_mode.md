# MBN_PIN_MODE enumeration


## -description

> [!IMPORTANT]
> Starting in Windows 10, version 1803, the Win32 APIs described in this section are replaced by the Windows Runtime APIs in the [Windows.Networking.Connectivity](/uwp/api/windows.networking.connectivity) namespace.

The <b>MBN_PIN_MODE</b> enumerated type indicates if the PIN type is enabled.


## -enum-fields




### -field MBN_PIN_MODE_ENABLED

The PIN type is currently enabled. 


### -field MBN_PIN_MODE_DISABLED

The PIN type is currently disabled.
