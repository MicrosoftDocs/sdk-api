# MBN_CELLULAR_CLASS enumeration


## -description


The <b>MBN_CELLULAR_CLASS</b> enumerated type defines the type of cellular device.


## -enum-fields




### -field MBN_CELLULAR_CLASS_NONE

No cellular class.


### -field MBN_CELLULAR_CLASS_GSM

GSM cellular class.


### -field MBN_CELLULAR_CLASS_CDMA

CDMA cellular class.
