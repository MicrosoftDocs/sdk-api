# MBN_SIGNAL_CONSTANTS enumeration


## -description


THE <b>MBN_SIGNAL_CONSTANTS</b> enumerated type contains specific values used by <a href="https://msdn.microsoft.com/2b60d078-ccbd-4cc5-addf-e6e95832b3a1">IMbnSignal</a> interface operations. 


## -enum-fields




### -field MBN_RSSI_DEFAULT

Use the default value for signal state reporting.


### -field MBN_RSSI_DISABLE

Disable signal state reporting.


### -field MBN_RSSI_UNKNOWN

Signal strength is unknown.


### -field MBN_ERROR_RATE_UNKNOWN

Signal error rate is unknown.
