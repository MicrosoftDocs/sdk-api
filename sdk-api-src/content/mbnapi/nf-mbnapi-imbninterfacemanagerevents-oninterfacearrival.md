# IMbnInterfaceManagerEvents::OnInterfaceArrival


## -description


Notification method that signals that a device has been added to the system.


## -parameters




### -param newInterface [in]

An <a href="https://msdn.microsoft.com/958bce42-4772-4706-8900-1f83c5d3d52b">IMbnInterface</a> that represents the new device.


## -returns



This method must return <b>S_OK</b>.
