# IMbnSmsReadMsgPdu interface


## -description


A collection of properties that represent an SMS message read from the device memory.


## -remarks



This interface is provided by the 
     <a href="https://msdn.microsoft.com/58478c9d-6df2-466b-85bc-1c571f4429ce">OnSmsReadComplete</a> and 
     <a href="https://msdn.microsoft.com/e6d13393-557c-462c-a640-2228ab0c9c17">OnSmsNewClass0Message</a> methods of the 
     <a href="https://msdn.microsoft.com/06dfb631-fe5a-45d9-89f9-1f13990500ee">IMbnSmsEvents</a> interface.
