# AMVP_SELECT_FORMAT_BY enumeration


## -description



Specifies the criteria that the <a href="https://docs.microsoft.com/windows/desktop/DirectShow/overlay-mixer-filter">Overlay Mixer Filter</a> should use to select the video format.




## -enum-fields




### -field AMVP_DO_NOT_CARE

Format does not matter.
          


### -field AMVP_BEST_BANDWIDTH

Use the largest bandwidth.
          


### -field AMVP_INPUT_SAME_AS_OUTPUT

Use the same input format as output format.
