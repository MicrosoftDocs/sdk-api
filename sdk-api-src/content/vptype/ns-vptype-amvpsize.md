# AMVPSIZE structure


## -description



The <b>AMVPSIZE</b> structure specifies the width and height for a VP image.




## -struct-fields




### -field dwWidth

Width, in pixels.


### -field dwHeight

Height.


## -remarks



The context could be anything such as scaling, cropping, and so on.
