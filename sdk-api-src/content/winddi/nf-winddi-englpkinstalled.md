# EngLpkInstalled function


## -description

The <b>EngLpkInstalled</b> function determines whether the language pack is installed on the system.



## -returns

<b>EngLpkInstalled</b> returns <b>TRUE</b> when the language pack is installed. Otherwise, it returns <b>FALSE</b>.
