# EngUnlockDriverObj function


## -description


The <b>EngUnlockDriverObj</b> function causes GDI to unlock the driver object.


## -parameters




### -param hdo

Identifies the object to be unlocked.


## -returns



The return value is <b>TRUE</b> if the function is successful; otherwise, it is <b>FALSE</b>.




## -remarks



The specified driver object must have been previously locked by a call to <a href="https://docs.microsoft.com/windows/desktop/api/winddi/nf-winddi-englockdriverobj">EngLockDriverObj</a>. The object is not unlockable by another thread while it is locked down.
