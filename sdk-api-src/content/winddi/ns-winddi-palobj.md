# PALOBJ structure


## -description

<p class="CCE_Message">[The PALOBJ structure has no public members.]

The <b>PALOBJ</b> 
  structure is a user object that represents an indexed color palette.

## -struct-fields

## -remarks

This structure is used in conjunction with the <a href="/windows/desktop/api/winddi/nf-winddi-palobj_cgetcolors">PALOBJ_cGetColors</a> service routine.
