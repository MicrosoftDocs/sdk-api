# IADsPrintQueueOperations::Pause


## -description

The <b>IADsPrintQueueOperations::Pause</b> method suspends the processing of print jobs within a print queue service.

## -parameters

## -returns

This method supports the standard return values. For more information about other return values, see the  <a href="/windows/desktop/ADSI/adsi-error-codes">ADSI Error Codes</a>.
