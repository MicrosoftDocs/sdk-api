# IADsPrintQueueOperations::Resume


## -description

The <b>IADsPrintQueueOperations::Resume</b> method resumes processing of suspended print jobs in the print queue.

## -parameters

## -returns

This method supports the standard return values. For more information about other return values, see the  <a href="/windows/desktop/ADSI/adsi-error-codes">ADSI Error Codes</a>.
