# ADS_EMAIL structure


## -description


The <b>ADS_EMAIL</b> structure is an ADSI representation of the <b>EMail Address</b> attribute syntax.


## -struct-fields




### -field Address

The null-terminated Unicode string that contains the user address.


### -field Type

Type of the email message.
