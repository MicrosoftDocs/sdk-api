# ColumnSortOrder enumeration


## -description


<p class="CCE_Message">[Windows Search 2.x is available for use in the operating system specified in the 

Requirements section. It might be altered or unavailable in later versions. Use the <a href="https://msdn.microsoft.com/library/Ee872061(v=VS.85).aspx">Windows Search API</a> instead.]



Used by <a href="https://msdn.microsoft.com/library/Aa965786(v=VS.85).aspx">IResultsViewer::SortOrderProperty</a> to indicate or set how a query is to be sorted.


## -enum-fields




### -field SortOrder_Ascending


### -field SortOrder_Descending




#### - AscItems

Indicates that the sort order is ascending.


#### - DescItems

Indicates that the sort order is descending.
