# ID2D1ImageBrush::GetExtendModeX


## -description


Gets the extend mode of the image brush on the x-axis.


## -parameters






## -returns



Type: <b><a href="https://docs.microsoft.com/windows/desktop/api/d2d1/ne-d2d1-d2d1_extend_mode">D2D1_EXTEND_MODE</a></b>

This method returns the x-extend mode.
