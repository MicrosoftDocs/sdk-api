# ID2D1CommandSink::PopAxisAlignedClip


## -description

Removes an axis-aligned clip from the layer and clip stack.



## -returns

Type: <b>HRESULT</b>

If the method succeeds, it returns <b>S_OK</b>. If it fails, it returns an <b>HRESULT</b> error code.
