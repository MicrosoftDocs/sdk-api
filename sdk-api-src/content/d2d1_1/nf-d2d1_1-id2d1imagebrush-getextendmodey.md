# ID2D1ImageBrush::GetExtendModeY


## -description


Gets the extend mode of the image brush on the y-axis of the image.


## -parameters






## -returns



Type: <b><a href="https://msdn.microsoft.com/6b6e1fe1-d43a-46cf-904d-5266b9bd6bf4">D2D1_EXTEND_MODE</a></b>

This method returns the  y-extend mode.
