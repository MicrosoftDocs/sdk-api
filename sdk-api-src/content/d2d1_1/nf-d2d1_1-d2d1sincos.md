# D2D1SinCos function


## -description


Returns the sine and cosine of an angle.


## -parameters




### -param angle [in]

Type: <b>FLOAT</b>

The angle to calculate.


### -param s [out]

Type: <b>FLOAT*</b>

The sine of the angle.


### -param c [out]

Type: <b>FLOAT*</b>

The cosine of the angle.
