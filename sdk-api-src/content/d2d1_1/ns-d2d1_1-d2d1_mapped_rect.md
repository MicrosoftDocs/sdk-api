# D2D1_MAPPED_RECT structure


## -description


 Describes mapped memory from the <a href="https://docs.microsoft.com/windows/desktop/api/d2d1_1/nf-d2d1_1-id2d1bitmap1-map">ID2D1Bitmap1::Map</a> API.


## -struct-fields




### -field pitch

The size in bytes of an individual scanline in the bitmap.


### -field bits

The data inside the bitmap.


## -remarks



The mapped rectangle is used to map a rectangle into the caller's address space.
