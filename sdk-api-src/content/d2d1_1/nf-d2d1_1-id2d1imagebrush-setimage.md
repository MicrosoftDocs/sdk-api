# ID2D1ImageBrush::SetImage


## -description

Sets the image associated with the provided image brush.

## -parameters

### -param image [in]

Type: <b><a href="/windows/desktop/api/d2d1/nn-d2d1-id2d1image">ID2D1Image</a>*</b>

The image to be associated with the image brush.
