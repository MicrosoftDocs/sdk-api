# D2D1Vec3Length function


## -description

Returns the length of a 3 dimensional vector.

## -parameters

### -param x [in]

Type: <b>FLOAT</b>

The  x value of the vector.

### -param y [in]

Type: <b>FLOAT</b>

The  y value of the vector.

### -param z [in]

Type: <b>FLOAT</b>

The  z value of the vector.

## -returns

Type: <b>FLOAT</b>

The length of the vector.
