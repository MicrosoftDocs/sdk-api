# WBEM_TEXT_FLAG_TYPE enumeration


## -description

Contains flags to control the execution of the <a href="/windows/desktop/api/wbemcli/nf-wbemcli-iwbemclassobject-getobjecttext">IWbemClassObject::GetObjectText</a> method.

## -enum-fields

### -field WBEM_FLAG_NO_FLAVORS

Present qualifiers without propagation or flavor information.
