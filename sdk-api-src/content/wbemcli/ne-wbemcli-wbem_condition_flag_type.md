# WBEM_CONDITION_FLAG_TYPE enumeration


## -description

Contains flags used with the <a href="/windows/desktop/api/wbemcli/nf-wbemcli-iwbemclassobject-getnames">IWbemClassObject::GetNames</a> method.

## -enum-fields

### -field WBEM_FLAG_ALWAYS:0

### -field WBEM_FLAG_ONLY_IF_TRUE:0x1

### -field WBEM_FLAG_ONLY_IF_FALSE:0x2

### -field WBEM_FLAG_ONLY_IF_IDENTICAL:0x3

### -field WBEM_MASK_PRIMARY_CONDITION:0x3

### -field WBEM_FLAG_KEYS_ONLY:0x4

### -field WBEM_FLAG_REFS_ONLY:0x8

### -field WBEM_FLAG_LOCAL_ONLY:0x10

### -field WBEM_FLAG_PROPAGATED_ONLY:0x20

### -field WBEM_FLAG_SYSTEM_ONLY:0x30

### -field WBEM_FLAG_NONSYSTEM_ONLY:0x40

### -field WBEM_MASK_CONDITION_ORIGIN:0x70

### -field WBEM_FLAG_CLASS_OVERRIDES_ONLY:0x100

### -field WBEM_FLAG_CLASS_LOCAL_AND_OVERRIDES:0x200

### -field WBEM_MASK_CLASS_CONDITION:0x300
