# IWbemHiPerfEnum::RemoveAll


## -description


The 
<b>IWbemHiPerfEnum::RemoveAll</b> method empties all objects from the enumerator.


## -parameters




### -param lFlags

Reserved. This parameter must be 0 (zero).


## -returns



This method returns <b>WBEM_S_NO_ERROR</b> if successful.
