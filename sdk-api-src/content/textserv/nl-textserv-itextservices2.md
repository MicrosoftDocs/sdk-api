# ITextServices2 class


## -description


The <b>ITextServices2</b> interface extends the <a href="https://msdn.microsoft.com/b0bc844f-2d20-4e67-84c5-0a5313bf6dee">ITextServices</a> interface.


## -inheritance

The <b xmlns:loc="http://microsoft.com/wdcml/l10n">ITextServices2</b> interface inherits from <a href="https://msdn.microsoft.com/b0bc844f-2d20-4e67-84c5-0a5313bf6dee">ITextServices</a>. <b>ITextServices2</b> also has these types of members:
