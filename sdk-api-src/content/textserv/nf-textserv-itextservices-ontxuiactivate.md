# ITextServices::OnTxUIActivate


## -description


Informs the text services object that the control is now UI active.


## -parameters






## -returns



Type: <b><a href="https://msdn.microsoft.com/4553cafc-450e-4493-a4d4-cb6e2f274d46">HRESULT</a></b>

The method always returns <b>S_OK</b>.




## -remarks



See <a href="https://msdn.microsoft.com/en-us/library/Bb787621(v=VS.85).aspx">ITextServices::OnTxInPlaceActivate</a> for a detailed description of activation.
