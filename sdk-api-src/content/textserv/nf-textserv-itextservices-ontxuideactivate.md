# ITextServices::OnTxUIDeactivate


## -description

Informs the text services object that the control is no longer UI active.

## -parameters

## -returns

Type: <b><a href="/windows/desktop/WinProg/windows-data-types">HRESULT</a></b>

The method always returns <b>S_OK</b>.

## -remarks

See <a href="/windows/desktop/api/textserv/nf-textserv-itextservices-ontxinplaceactivate">ITextServices::OnTxInPlaceActivate</a> for a detailed description of deactivation.
