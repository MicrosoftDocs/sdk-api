# SYNCMGR_CONFLICT_ID_INFO structure


## -description


Describes conflict ID information structure.


## -struct-fields




### -field pblobID

Type: <b>BYTE_BLOB*</b>

A pointer to a blob used for comparison.


### -field pblobExtra

Type: <b>BYTE_BLOB*</b>

A pointer to extra data used to initialize conflict objects.
