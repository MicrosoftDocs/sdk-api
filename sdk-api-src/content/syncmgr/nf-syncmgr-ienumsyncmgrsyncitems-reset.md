# IEnumSyncMgrSyncItems::Reset


## -description

Resets the current position in the enumeration to 0.



## -returns

Type: <b>HRESULT</b>

If this method succeeds, it returns <b>S_OK</b>. Otherwise, it returns an <b>HRESULT</b> error code.
