# ISyncMgrControl::StopSyncAll


## -description

Stops the synchronization of all items managed by all handlers.



## -returns

Type: <b>HRESULT</b>

If this method succeeds, it returns <b>S_OK</b>. Otherwise, it returns an <b>HRESULT</b> error code.
