# ISyncMgrConflictResolutionItems::GetCount


## -description

Gets item count.

## -parameters

### -param pCount [out]

Type: <b>UINT*</b>

A pointer to an item count.

## -returns

Type: <b>HRESULT</b>

If this method succeeds, it returns <b>S_OK</b>. Otherwise, it returns an <b>HRESULT</b> error code.
