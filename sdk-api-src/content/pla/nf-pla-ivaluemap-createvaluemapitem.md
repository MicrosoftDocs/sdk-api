# IValueMap::CreateValueMapItem


## -description


Creates a value map item.


## -parameters




### -param Item [out]

An <a href="https://msdn.microsoft.com/5fab2a62-d974-49f7-ac81-c704d9d8624c">IValueMapItem</a> interface that you use to define a name/value pair.


## -returns



Returns S_OK if successful.
