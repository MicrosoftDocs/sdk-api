# IDataCollectorSet::get_OutputLocation


## -description

Retrieves the decorated folder name if PLA were to create it now.

This property is read-only.

## -parameters

## -remarks

The <a href="/previous-versions/windows/desktop/api/pla/nf-pla-idatacollectorset-get_latestoutputlocation">IDataCollectorSet::LatestOutputLocation</a> property contains the decorated file name used the last time the collector ran.
