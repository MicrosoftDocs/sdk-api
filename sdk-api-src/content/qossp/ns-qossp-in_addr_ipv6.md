# IN_ADDR_IPV6 structure


## -description

The <b>IN_ADDR_IPV6</b> structure stores an IPv6 address for use with RSVP FILTERSPECs.

## -struct-fields

### -field Addr

IPv6 address.

## -remarks

When working with IPv4 addresses, use <a href="/windows/desktop/api/qossp/ns-qossp-in_addr_ipv4">IN_ADDR_IPV4</a>.
