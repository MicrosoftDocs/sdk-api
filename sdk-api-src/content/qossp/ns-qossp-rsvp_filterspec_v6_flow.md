# RSVP_FILTERSPEC_V6_FLOW structure


## -description


The <b>RSVP_FILTERSPEC_V6_FLOW</b> structure provides flow label information for an IPv6 FILTERSPEC.


## -struct-fields




### -field Address

IPv4 address for which the FILTERSPEC flow label applies, expressed as an <a href="https://docs.microsoft.com/windows/desktop/api/qossp/ns-qossp-in_addr_ipv6">IN_ADDR_IPV6</a> structure.


### -field UnUsed

 


### -field FlowLabel

Label for the flow.


#### - Unused

Reserved. Do not use.
