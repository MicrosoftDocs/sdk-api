# _IN_ADDR_IPV6 structure


## -description


The <b>IN_ADDR_IPV6</b> structure stores an IPv6 address for use with RSVP FILTERSPECs.


## -struct-fields




### -field Addr

IPv6 address.


## -remarks



When working with IPv4 addresses, use <a href="https://msdn.microsoft.com/7e10cc9c-7ed4-449d-aeb9-21e3d75d0224">IN_ADDR_IPV4</a>.
