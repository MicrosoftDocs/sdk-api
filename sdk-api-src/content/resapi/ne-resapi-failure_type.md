# FAILURE_TYPE enumeration


## -description

Defines the failure types for cluster resources.

## -enum-fields

### -field FAILURE_TYPE_GENERAL:0

A general failure.

### -field FAILURE_TYPE_EMBEDDED:1

An embedded failure.

### -field FAILURE_TYPE_NETWORK_LOSS:2

A network failure.
