# RESUTIL_ULARGEINT_DATA structure


## -description

Describes the default, maximum, and minimum values allowed for an unsigned large integer.

## -struct-fields

### -field Default

The default value.

### -field Minimum

The minimum value.

### -field Maximum

The maximum value.
