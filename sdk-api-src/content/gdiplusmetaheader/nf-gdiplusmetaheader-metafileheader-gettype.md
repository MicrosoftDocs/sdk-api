# MetafileHeader::GetType


## -description


The <b>MetafileHeader::GetType</b> method gets the type of the associated metafile.


## -parameters






## -returns



Type: <strong>Type: <b><a href="https://msdn.microsoft.com/e29e5413-12c4-4706-be9f-807f0b88debe">MetafileType</a></b>
</strong>

This method returns an element of the <a href="https://msdn.microsoft.com/e29e5413-12c4-4706-be9f-807f0b88debe">MetafileType</a> enumeration that indicates the type of the associated metafile.
