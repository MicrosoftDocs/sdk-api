# MetafileHeader::GetMetafileSize


## -description


The <b>MetafileHeader::GetMetafileSize</b> method gets the size, in bytes, of the metafile.


## -parameters






## -returns



Type: <b>UINT</b>

This method returns a <b>UINT</b> that specifies the size, in bytes, of the metafile.
