# IComHandlerAction::put_ClassId


## -description

Gets or sets the identifier of the handler class.

This property is read/write.

## -parameters

## -remarks

When reading or writing XML, the class of a COM handler is specified in the <a href="/windows/desktop/TaskSchd/taskschedulerschema-classid-comhandlertype-element">ClassId</a> element of the Task Scheduler schema.
