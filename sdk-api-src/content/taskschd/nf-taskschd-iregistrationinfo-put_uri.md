# IRegistrationInfo::put_URI


## -description


Gets or sets the URI of the task.

This property is read/write.


## -parameters


## -remarks



When reading or writing XML for a task, the task URI is specified using the <a href="https://msdn.microsoft.com/5b438e00-ed8a-4ec8-854f-e8eda48d1cfc">URI</a> element of the Task Scheduler schema.
