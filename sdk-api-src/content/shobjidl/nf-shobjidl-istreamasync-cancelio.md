# IStreamAsync::CancelIo


## -description

Marks all pending input/output (I/O) operations as canceled.



## -returns

Type: <b>HRESULT</b>

If this method succeeds, it returns <b>S_OK</b>. Otherwise, it returns an <b>HRESULT</b> error code.
