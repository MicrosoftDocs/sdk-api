# IFolderBandPriv::SetNoIcons


## -description

Sets whether icons are displayed in a folder band.

## -parameters

### -param fNoIcons

Type: <b>BOOL</b>

<b>TRUE</b> to display icons; otherwise, <b>FALSE</b>.

## -returns

Type: <b>HRESULT</b>

Always returns S_OK.
