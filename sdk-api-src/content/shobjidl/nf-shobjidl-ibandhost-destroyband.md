# IBandHost::DestroyBand


## -description

Destroys a specified band.

## -parameters

### -param rclsidBand [in]

Type: <b>REFCLSID</b>

A reference to the CLSID of a band.

## -returns

Type: <b>HRESULT</b>

If this method succeeds, it returns <b>S_OK</b>. Otherwise, it returns an <b>HRESULT</b> error code.
