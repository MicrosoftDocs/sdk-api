# IFolderBandPriv::SetNoText


## -description


Sets whether text is displayed in a folder band.


## -parameters




### -param fNoText

Type: <b>BOOL</b>

<b>TRUE</b> to display text; otherwise, <b>FALSE</b>.


## -returns



Type: <b>HRESULT</b>

Always returns S_OK.
