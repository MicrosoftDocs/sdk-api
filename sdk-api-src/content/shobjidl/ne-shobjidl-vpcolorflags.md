# VPCOLORFLAGS enumeration


## -description


Specifies the use of a color. Used by <a href="https://msdn.microsoft.com/0f4921ad-18b4-447e-8e4b-0a40ce430137">IVisualProperties</a> methods.


## -enum-fields




### -field VPCF_TEXT

A text color.


### -field VPCF_BACKGROUND

A background color.


### -field VPCF_SORTCOLUMN

A sort-column color.


### -field VPCF_SUBTEXT

A subtext color.


### -field VPCF_TEXTBACKGROUND

<b>Windows 7 and later</b>. A text background color.
