# IVisualProperties::SetItemHeight


## -description

Sets the specified item height.

## -parameters

### -param cyItemInPixels [in]

Type: <b>int</b>

The item height, in pixels.

## -returns

Type: <b>HRESULT</b>

If this method succeeds, it returns <b>S_OK</b>. Otherwise, it returns an <b>HRESULT</b> error code.
