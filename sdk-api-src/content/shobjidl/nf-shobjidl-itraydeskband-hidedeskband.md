# ITrayDeskBand::HideDeskBand


## -description

Hides a specified deskband.

## -parameters

### -param clsid [in]

Type: <b>REFCLSID</b>

A reference to a deskband CLSID.

## -returns

Type: <b>HRESULT</b>

If this method succeeds, it returns <b>S_OK</b>. Otherwise, it returns an <b>HRESULT</b> error code.
