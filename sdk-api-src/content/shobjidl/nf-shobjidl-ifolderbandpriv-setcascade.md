# IFolderBandPriv::SetCascade


## -description

Sets a cascade folder.

## -parameters

### -param fCascade

Type: <b>BOOL</b>

<b>TRUE</b> to enable cascade folders; <b>FALSE</b> to disable them.

## -returns

Type: <b>HRESULT</b>

Always returns S_OK.
