# IPropertyChangeArray::RemoveAt


## -description

Removes a specified change.

## -parameters

### -param iIndex [in]

Type: <b>UINT</b>

The index of the change to remove.

## -returns

Type: <b>HRESULT</b>

If this method succeeds, it returns <b>S_OK</b>. Otherwise, it returns an <b>HRESULT</b> error code.
