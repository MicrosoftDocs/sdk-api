# IObjectWithPropertyKey::SetPropertyKey


## -description

Sets the property key.

## -parameters

### -param key [in]

Type: <b>REFPROPERTYKEY</b>

The property key.

## -returns

Type: <b>HRESULT</b>

If this method succeeds, it returns <b>S_OK</b>. Otherwise, it returns an <b>HRESULT</b> error code.
