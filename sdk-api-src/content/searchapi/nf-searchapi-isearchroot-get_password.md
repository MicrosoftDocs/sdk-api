# ISearchRoot::get_Password


## -description

Not implemented.

## -parameters

### -param ppszPassword

Type: <b>HRESULT</b>

If this method succeeds, it returns <b>S_OK</b>. Otherwise, it returns an <b>HRESULT</b> error code.
