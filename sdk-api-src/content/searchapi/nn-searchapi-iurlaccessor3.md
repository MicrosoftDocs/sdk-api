# IUrlAccessor3 interface


## -description

Extends the functionality of the <a href="/windows/desktop/api/searchapi/nn-searchapi-iurlaccessor2">IUrlAccessor2</a> interface with the <a href="/windows/desktop/api/searchapi/nf-searchapi-iurlaccessor3-getimpersonationsidblobs">IUrlAccessor3::GetImpersonationSidBlobs</a> method to identify user security identifiers (SIDs) for a specified URL.
