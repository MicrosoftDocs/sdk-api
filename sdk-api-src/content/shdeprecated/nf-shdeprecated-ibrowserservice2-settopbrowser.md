# IBrowserService2::SetTopBrowser


## -description

Deprecated. Informs the base class when it becomes the topmost browser instance.



## -returns

Type: <b>HRESULT</b>

If this method succeeds, it returns <b>S_OK</b>. Otherwise, it returns an <b>HRESULT</b> error code.
