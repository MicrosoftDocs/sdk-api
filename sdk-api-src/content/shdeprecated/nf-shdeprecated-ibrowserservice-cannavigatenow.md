# IBrowserService::CanNavigateNow


## -description

Deprecated. Returns a value that indicates whether navigation is currently allowed.



## -returns

Type: <b>HRESULT</b>

Returns S_OK if navigation is allowed, or S_FALSE if the browser is busy.
