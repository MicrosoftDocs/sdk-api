# IBrowserService2::_get_itbLastFocus


## -description


Deprecated. Gets the ID of the last toolbar or view that had focus.


## -parameters






## -returns



Type: <b>UINT</b>

Returns the index of the last toolbar with focus, or ITB_VIEW if the focus was last on the view.
