# ICameraUIControl::RemoveCapturedItem


## -description

Removes  the captured item.

## -parameters

### -param pszPath [in]

The path to the item to delete.

## -returns

If this method succeeds, it returns <b>S_OK</b>. Otherwise, it returns an <b>HRESULT</b> error code.
