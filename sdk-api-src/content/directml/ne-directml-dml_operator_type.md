# DML_OPERATOR_TYPE enumeration


## -description






Defines the type of an operator description.

    See [DML_OPERATOR_DESC](/windows/desktop/api/directml/ns-directml-dml_operator_desc) for the usage of this enumeration.


## -enum-fields




### -field DML_OPERATOR_INVALID

Indicates an unknown operator type,
and is never valid. Using this value results in an error.
