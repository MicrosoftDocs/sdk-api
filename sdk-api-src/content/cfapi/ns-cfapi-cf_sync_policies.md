# CF_SYNC_POLICIES structure


## -description


Defines the sync policies used by  a sync root.


## -struct-fields




### -field StructSize

The size of the CF_SYNC_POLICIES structure.


### -field Hydration

The hydration policy.


### -field Population

The population policy.


### -field InSync

The in-sync policy.


### -field HardLink

The hard link policy.
