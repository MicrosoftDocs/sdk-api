# IMSVidFilePlayback::put_FileName


## -description


The <b>put_FileName</b> method sets the name of the file to play.


## -parameters




### -param FileName [in]

Specifies the file name.


## -returns



The method returns an <b>HRESULT</b>. Possible values include the following.
