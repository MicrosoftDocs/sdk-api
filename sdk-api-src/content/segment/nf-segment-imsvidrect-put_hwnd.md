# IMSVidRect::put_HWnd


## -description


The <b>put_HWnd</b> method specifies the window associated with the rectangle.


## -parameters




### -param HWndVal [in]

Specifies the handle to the window.


## -returns



If the method succeeds, it returns S_OK. If it fails, it returns an error code.
