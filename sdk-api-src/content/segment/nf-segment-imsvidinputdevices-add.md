# IMSVidInputDevices::Add


## -description

The <b>Add</b> method adds an input device to the collection.

## -parameters

### -param pDB [in]

Pointer to the device's <a href="/previous-versions/windows/desktop/mstv/msvidinputdevice">IMSVidInputDevice</a> interface.
