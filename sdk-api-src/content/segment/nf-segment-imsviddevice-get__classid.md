# IMSVidDevice::get__ClassID


## -description


The <b>get__ClassID</b> method retrieves the device's class identifier (CLSID) as a <b>GUID</b>.


## -parameters




### -param Clsid [out]

Pointer to a variable of type <b>GUID</b> that receives the CLSID.
