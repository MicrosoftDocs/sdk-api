# IMSVidFeatures::Add


## -description


The <b>Add</b> method adds a feature to the collection.


## -parameters




### -param pDB [in]

Specifies a pointer to the feature's <a href="https://docs.microsoft.com/previous-versions/windows/desktop/mstv/msvidfeature">IMSVidFeature</a> interface.


## -returns
