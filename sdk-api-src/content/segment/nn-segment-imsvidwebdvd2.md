# IMSVidWebDVD2 interface


## -description


Contains methods that save and load the current location and state for DVD playback.


## -remarks



To declare the interface identifier (IID) for this interface, use the <b>__uuidof</b> operator: <code>__uuidof(IMSVidWebDVD2)</code>.
