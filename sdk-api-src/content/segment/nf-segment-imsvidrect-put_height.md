# IMSVidRect::put_Height


## -description


The <b>put_Height</b> method specifies the height of the rectangle.


## -parameters




### -param HeightVal [in]

Specifies the height, in pixels.


## -returns



If the method succeeds, it returns S_OK. If it fails, it returns an error code.
