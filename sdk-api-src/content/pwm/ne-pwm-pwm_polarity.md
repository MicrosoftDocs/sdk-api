# PWM_POLARITY enumeration


## -description


Represents the Pulse Width Modulation (PWM) signal level polarity with respect to the duty cycle.


## -enum-fields




### -field PWM_ACTIVE_HIGH

Active High.


### -field PWM_ACTIVE_LOW

Active Low.
