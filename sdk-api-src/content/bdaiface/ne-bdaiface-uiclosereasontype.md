# UICloseReasonType enumeration


## -description

The <b>UICloseReasonType</b> enumeration type specifies the reason that a user-interface dialog for a smart card was closed.

## -enum-fields

### -field NotReady:0

Not ready.

### -field UserClosed

User.

### -field SystemClosed

System.

### -field DeviceClosed

Device.

### -field ErrorClosed

Error.
