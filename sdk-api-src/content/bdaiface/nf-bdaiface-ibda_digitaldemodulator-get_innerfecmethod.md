# IBDA_DigitalDemodulator::get_InnerFECMethod


## -description



The <b>get_InnerFECMethod</b> method retrieves the inner forward error correction method.




## -parameters




### -param pFECMethod [out]

Pointer that receives an <a href="https://docs.microsoft.com/previous-versions/windows/desktop/mstv/fecmethod">FECMethod</a> variable.


## -returns



If the method succeeds, it returns S_OK. If it fails, it returns an error code.
