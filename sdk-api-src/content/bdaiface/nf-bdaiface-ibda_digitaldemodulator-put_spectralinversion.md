# IBDA_DigitalDemodulator::put_SpectralInversion method


## -description



The <b>put_SpectralInversion</b> method specifies the spectral inversion value for the signal.




## -parameters




### -param pSpectralInversion [in]

Pointer to a <a href="https://msdn.microsoft.com/library/windows/hardware/ff568154">SpectralInversion</a> variable.


## -returns



If the method succeeds, it returns S_OK. If it fails, it returns an error code.
