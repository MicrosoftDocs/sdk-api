# IEnumPIDMap::Skip


## -description



The <code>Skip</code> method skips the specified number of elements in the collection.




## -parameters




### -param cRecords [in]

Specifies the number of elements to skip.


## -returns



Returns an <b>HRESULT</b> value. Possible values include the following.
