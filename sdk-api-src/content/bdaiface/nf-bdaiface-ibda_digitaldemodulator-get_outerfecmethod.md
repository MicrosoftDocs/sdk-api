# IBDA_DigitalDemodulator::get_OuterFECMethod


## -description



The <b>get_OuterFECMethod</b> method retrieves the outer forward error correction method for the signal .




## -parameters




### -param pFECMethod [out]

Pointer that receives an <a href="https://docs.microsoft.com/previous-versions/windows/desktop/mstv/fecmethod">FECMethod</a> variable that indicates the FEC method.


## -returns



If the method succeeds, it returns S_OK. If it fails, it returns an error code.
