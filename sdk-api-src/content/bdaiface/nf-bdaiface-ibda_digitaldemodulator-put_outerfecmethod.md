# IBDA_DigitalDemodulator::put_OuterFECMethod


## -description



The <b>put_OuterFECMethod</b> method specifies the outer forward error correction method for the signal.




## -parameters




### -param pFECMethod [in]

Pointer to an <a href="https://docs.microsoft.com/previous-versions/windows/desktop/mstv/fecmethod">FECMethod</a> variable.


## -returns



If the method succeeds, it returns S_OK. If it fails, it returns an error code.
