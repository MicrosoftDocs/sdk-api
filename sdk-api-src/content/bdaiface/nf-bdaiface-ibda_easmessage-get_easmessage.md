# IBDA_EasMessage::get_EasMessage


## -description


The <b>get_EasMessage</b> method retrieves an EAS message.


## -parameters




### -param ulEventID [in]

Specifies the event ID of the EAS message.


### -param ppEASObject [in, out]

Pointer to a pointer variable that receives a pointer to the <b>IUnknown</b> interface of the EAS object. The caller can query this object for its <a href="https://docs.microsoft.com/previous-versions/windows/desktop/api/atscpsipparser/nn-atscpsipparser-iscte_eas">ISCTE_EAS</a> interface.


## -returns



If the method succeeds, it returns S_OK. If it fails, it returns an error code.




## -remarks



This method retrieves a counted reference to an <b>IUnknown</b> interface instance. The caller is responsible for releasing the interface when it is no longer required.
