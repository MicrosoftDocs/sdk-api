# SmartCardAssociationType enumeration


## -description

The <b>SmartCardAssociationType</b> enumeration type specifies a smart card association.

## -enum-fields

### -field NotAssociated:0

Not associated.

### -field Associated:0

Associated.

### -field AssociationUnknown

Association status is unknown.
