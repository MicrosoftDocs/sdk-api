# SmartCardApplication structure


## -description



The <b>SmartCardApplication</b> structure identifies a smart card application.




## -struct-fields




### -field ApplicationType

Member of the <a href="https://docs.microsoft.com/previous-versions/windows/desktop/mstv/applicationtypetype">ApplicationTypeType</a> enumeration.


### -field ApplicationVersion

Application version.


### -field pbstrApplicationName

Application name.


### -field pbstrApplicationURL

Application URL.
