# IBDA_IPV6Filter::PutMulticastMode


## -description



The <b>PutMulticastMode</b> method specifies the multicast mode.




## -parameters




### -param ulModeMask [in]

Specifies the mode mask.


## -returns



If the method succeeds, it returns S_OK. If it fails, it returns an error code.
