# IBDA_FrequencyFilter::put_Polarity


## -description

\[The feature associated with this page, [Microsoft TV Technologies](/previous-versions/windows/desktop/mstv/microsoft-tv-technologies-portal), is a legacy feature. Microsoft strongly recommends that new code does not use this feature.\]

The <b>put_Polarity</b> method specifies the polarity of the signal.

## -parameters

### -param Polarity [in]

Specifies the polarity as a member of the <a href="/previous-versions/windows/desktop/mstv/polarisation">Polarisation</a> enumerated type.

## -returns

If the method succeeds, it returns S_OK. If it fails, it returns an error code.
