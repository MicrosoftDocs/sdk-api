# DWRITE_PANOSE_LETTERFORM enumeration


## -description

The <b>DWRITE_PANOSE_LETTERFORM</b> enumeration contains values that specify the roundness of letterform for text.

## -enum-fields

### -field DWRITE_PANOSE_LETTERFORM_ANY:0

Any letterform.

### -field DWRITE_PANOSE_LETTERFORM_NO_FIT:1

No fit letterform.

### -field DWRITE_PANOSE_LETTERFORM_NORMAL_CONTACT:2

Normal contact letterform.

### -field DWRITE_PANOSE_LETTERFORM_NORMAL_WEIGHTED:3

Normal weighted letterform.

### -field DWRITE_PANOSE_LETTERFORM_NORMAL_BOXED:4

Normal boxed letterform.

### -field DWRITE_PANOSE_LETTERFORM_NORMAL_FLATTENED:5

Normal flattened letterform.

### -field DWRITE_PANOSE_LETTERFORM_NORMAL_ROUNDED:6

Normal rounded letterform.

### -field DWRITE_PANOSE_LETTERFORM_NORMAL_OFF_CENTER:7

Normal off-center letterform.

### -field DWRITE_PANOSE_LETTERFORM_NORMAL_SQUARE:8

Normal square letterform.

### -field DWRITE_PANOSE_LETTERFORM_OBLIQUE_CONTACT:9

Oblique contact letterform.

### -field DWRITE_PANOSE_LETTERFORM_OBLIQUE_WEIGHTED:10

Oblique weighted letterform.

### -field DWRITE_PANOSE_LETTERFORM_OBLIQUE_BOXED:11

Oblique boxed letterform.

### -field DWRITE_PANOSE_LETTERFORM_OBLIQUE_FLATTENED:12

Oblique flattened letterform.

### -field DWRITE_PANOSE_LETTERFORM_OBLIQUE_ROUNDED:13

Oblique rounded letterform.

### -field DWRITE_PANOSE_LETTERFORM_OBLIQUE_OFF_CENTER:14

Oblique off-center letterform.

### -field DWRITE_PANOSE_LETTERFORM_OBLIQUE_SQUARE:15

Oblique square letterform.
