# IDWriteFont1::IsMonospacedFont


## -description

Determines if the font is monospaced, that is, the characters are the
    same fixed-pitch width (non-proportional).



## -returns

Type: <b>BOOL</b>

Returns true if the font is monospaced, else it returns false.
