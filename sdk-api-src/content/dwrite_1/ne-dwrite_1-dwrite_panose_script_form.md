# DWRITE_PANOSE_SCRIPT_FORM enumeration


## -description

The <b>DWRITE_PANOSE_SCRIPT_FORM</b> enumeration contains values that specify the general look of the character face, with consideration of its  slope and tails.

## -enum-fields

### -field DWRITE_PANOSE_SCRIPT_FORM_ANY:0

Any script form.

### -field DWRITE_PANOSE_SCRIPT_FORM_NO_FIT:1

No fit for script form.

### -field DWRITE_PANOSE_SCRIPT_FORM_UPRIGHT_NO_WRAPPING:2

Script form is upright with no wrapping.

### -field DWRITE_PANOSE_SCRIPT_FORM_UPRIGHT_SOME_WRAPPING:3

Script form is upright with some wrapping.

### -field DWRITE_PANOSE_SCRIPT_FORM_UPRIGHT_MORE_WRAPPING:4

Script form is upright with more wrapping.

### -field DWRITE_PANOSE_SCRIPT_FORM_UPRIGHT_EXTREME_WRAPPING:5

Script form is upright with extreme wrapping.

### -field DWRITE_PANOSE_SCRIPT_FORM_OBLIQUE_NO_WRAPPING:6

Script form is oblique with no wrapping.

### -field DWRITE_PANOSE_SCRIPT_FORM_OBLIQUE_SOME_WRAPPING:7

Script form is oblique with some wrapping.

### -field DWRITE_PANOSE_SCRIPT_FORM_OBLIQUE_MORE_WRAPPING:8

Script form is oblique with more wrapping.

### -field DWRITE_PANOSE_SCRIPT_FORM_OBLIQUE_EXTREME_WRAPPING:9

Script form is oblique with extreme wrapping.

### -field DWRITE_PANOSE_SCRIPT_FORM_EXAGGERATED_NO_WRAPPING:10

Script form is exaggerated with no wrapping.

### -field DWRITE_PANOSE_SCRIPT_FORM_EXAGGERATED_SOME_WRAPPING:11

Script form is exaggerated with some wrapping.

### -field DWRITE_PANOSE_SCRIPT_FORM_EXAGGERATED_MORE_WRAPPING:12

Script form is exaggerated with more wrapping.

### -field DWRITE_PANOSE_SCRIPT_FORM_EXAGGERATED_EXTREME_WRAPPING:13

Script form is exaggerated with extreme wrapping.
