# DWRITE_UNICODE_RANGE structure


## -description

The <b>DWRITE_UNICODE_RANGE</b> structure specifies the range of Unicode code points.

## -struct-fields

### -field first

The first code point in the Unicode range.

### -field last

The last code point in the Unicode range.
