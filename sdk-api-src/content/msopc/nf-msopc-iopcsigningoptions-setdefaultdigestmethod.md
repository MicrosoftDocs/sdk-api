# IOpcSigningOptions::SetDefaultDigestMethod


## -description

Sets the default digest method that will be used to compute digest values for objects to be signed.

## -parameters

### -param digestMethod [in]

The default digest method.

## -returns

The method returns an <b>HRESULT</b>. Possible values include, but are not limited to, those in the following table.

<table>
<tr>
<th>Return code</th>
<th>Description</th>
</tr>
<tr>
<td width="40%">
<dl>
<dt><b>S_OK</b></dt>
</dl>
</td>
<td width="60%">
The method succeeded.

</td>
</tr>
<tr>
<td width="40%">
<dl>
<dt><b>E_POINTER</b></dt>
</dl>
</td>
<td width="60%">
The <i>digestMethod</i> parameter is <b>NULL</b>.

</td>
</tr>
</table>

## -remarks

To access the default digest method before the signature is generated, call the <a href="/previous-versions/windows/desktop/api/msopc/nf-msopc-iopcsigningoptions-getdefaultdigestmethod">IOpcSigningOptions::GetDefaultDigestMethod</a>.

<div class="alert"><b>Important</b>  The default digest method for entities to be signed must be set before the signature is generated by calling the <a href="/previous-versions/windows/desktop/api/msopc/nf-msopc-iopcdigitalsignaturemanager-sign">IOpcDigitalSignatureManager::Sign</a> method.</div>
<div> </div>
