# XPS_DASH_CAP enumeration


## -description

Specifies the style of a dash cap  on a dashed stroke.

## -enum-fields

### -field XPS_DASH_CAP_FLAT:1

Flat-line cap.

### -field XPS_DASH_CAP_ROUND

Round-line cap.

### -field XPS_DASH_CAP_SQUARE

Square-line cap.

### -field XPS_DASH_CAP_TRIANGLE

Triangular-line cap.
