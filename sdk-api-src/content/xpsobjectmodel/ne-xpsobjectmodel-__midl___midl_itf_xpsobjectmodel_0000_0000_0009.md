# __MIDL___MIDL_itf_xpsobjectmodel_0000_0000_0009 enumeration


## -description


Describes the color type used by the <a href="https://msdn.microsoft.com/710f3ef1-bbc3-416d-9faf-aa4a716007c2">XPS_COLOR</a> structure.


## -enum-fields




### -field XPS_COLOR_TYPE_SRGB

The color value is an sRGB value.


### -field XPS_COLOR_TYPE_SCRGB

The color value is an scRGB value.


### -field XPS_COLOR_TYPE_CONTEXT

The color value is specified using context color syntax.
