# MI_ConstInstanceA structure


## -description


Represents an array of <a href="https://docs.microsoft.com/windows/desktop/api/mi/ns-mi-mi_instance">MI_Instance</a> types.


## -struct-fields




### -field data

An array of <a href="https://docs.microsoft.com/windows/desktop/api/mi/ns-mi-mi_instance">MI_Instance</a> types.


### -field size

Number of items in the data array.
