# _MI_UsernamePasswordCreds structure


## -description


A username/password combination used for subscription operations.


## -struct-fields




### -field domain

The Domain <b>username</b> resides in.


### -field username

The <b>username</b> in <b>domain</b>.


### -field password

The password for <b>username</b>.
