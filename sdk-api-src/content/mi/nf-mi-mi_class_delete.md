# MI_Class_Delete function


## -description

Deletes an <a href="/windows/desktop/api/mi/ns-mi-mi_class">MI_Class</a> object.

## -parameters

### -param self [in, out]

A pointer to the class to be deleted.

## -returns

This function returns MI_INLINE MI_Result.
