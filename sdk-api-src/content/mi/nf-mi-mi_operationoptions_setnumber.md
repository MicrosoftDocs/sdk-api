# MI_OperationOptions_SetNumber function


## -description


Sets a custom number option value.


## -parameters




### -param options [in, out]

A pointer to a <a href="https://docs.microsoft.com/windows/desktop/api/mi/ns-mi-mi_operationoptions">MI_OperationOptions</a> structure.


### -param optionName

A null-terminated string that represents the name of the option to set.


### -param value [in]

New option value.


### -param flags

Option flags.


## -returns



This function returns MI_INLINE MI_Result.
