# MI_StringA structure


## -description

Represents an array of pointers to null-terminated <b>MI_Char</b>* strings.

## -struct-fields

### -field data

An array of pointers to null-terminated <b>MI_Char</b>* strings.

### -field size

The number of items in the array.
