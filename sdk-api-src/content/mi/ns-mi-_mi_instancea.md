# _MI_InstanceA structure


## -description


Represents an array of <a href="https://msdn.microsoft.com/3dce1817-7995-49e5-8cc0-ee9496665e5c">MI_Instance</a> structures.


## -struct-fields




### -field data

An array of pointers to <a href="https://msdn.microsoft.com/3dce1817-7995-49e5-8cc0-ee9496665e5c">MI_Instance</a> structures.


### -field size

Number of items in the data array.
