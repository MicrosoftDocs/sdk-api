# PHONECAPS_STRING enumeration


## -description


The 
<b>PHONECAPS_STRING</b> enum is used by methods that set or get phone capabilities described by a string.


## -enum-fields




### -field PCS_PHONENAME

Name of the phone.


### -field PCS_PHONEINFO

Phone information string.


### -field PCS_PROVIDERINFO

Phone provider string.
