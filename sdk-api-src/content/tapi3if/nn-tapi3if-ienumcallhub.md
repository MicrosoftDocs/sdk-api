# IEnumCallHub interface


## -description

The 
<b>IEnumCallHub</b> interface provides COM-standard enumeration methods for the 
<a href="/windows/desktop/api/tapi3if/nn-tapi3if-itcallhub">ITCallHub</a> interface. The 
<a href="/windows/desktop/api/tapi3if/nf-tapi3if-ittapi-enumeratecallhubs">ITTAPI::EnumerateCallHubs</a> method returns a pointer to 
<b>IEnumCallHub</b>.

The 
<b>IEnumCallHub</b> interface is hidden from Visual Basic and scripting languages.

## -inheritance

The <b xmlns:loc="http://microsoft.com/wdcml/l10n">IEnumCallHub</b> interface inherits from the <a href="/windows/desktop/api/unknwn/nn-unknwn-iunknown">IUnknown</a> interface. <b>IEnumCallHub</b> also has these types of members:
