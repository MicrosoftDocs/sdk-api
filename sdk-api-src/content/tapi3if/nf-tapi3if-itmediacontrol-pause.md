# ITMediaControl::Pause


## -description

The 
<b>Pause</b> method pauses the action, remaining at the current location in the file.



## -returns

If this method succeeds, it returns <b>S_OK</b>. Otherwise, it returns an <b>HRESULT</b> error code.
