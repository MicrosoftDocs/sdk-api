# ITForwardInformation::get_NumRingsNoAnswer


## -description


The 
<b>get_NumRingsNoAnswer</b> method retrieves the number of rings after which a no answer condition is assumed.


## -parameters




### -param plNumRings [out]

Pointer to number of rings.
