# ITPrivateEvent interface


## -description


The 
<b>ITPrivateEvent</b> interface exposes methods that allows an application to process events generated by a private object.


## -inheritance

The <b xmlns:loc="http://microsoft.com/wdcml/l10n">ITPrivateEvent</b> interface inherits from the <a href="ebbff4bc-36b2-4861-9efa-ffa45e013eb5">IDispatch</a> interface. <b>ITPrivateEvent</b> also has these types of members:
