# PHONECAPS_BUFFER enumeration


## -description

The 
<b>PHONECAPS_BUFFER</b> enum is used by methods that set or get phone capabilities described by a buffer.

## -enum-fields

### -field PCB_DEVSPECIFICBUFFER

Device-specific phone capabilities.
