# ITMediaControl::Stop


## -description

The 
<b>Stop</b> method stops the current action and sets the current location to the beginning of the file.



## -returns

If this method succeeds, it returns <b>S_OK</b>. Otherwise, it returns an <b>HRESULT</b> error code.
