# PHONE_BUTTON_MODE enumeration


## -description


The 
<b>PHONE_BUTTON_MODE</b> enum describes the mode of a phone button.


## -enum-fields




### -field PBM_DUMMY

Dummy button.


### -field PBM_CALL

Call button.


### -field PBM_FEATURE

Feature button.


### -field PBM_KEYPAD

Keypad button.


### -field PBM_LOCAL

Local function button, such as mute or volume control.


### -field PBM_DISPLAY

Display button.
