# ITQOSEvent::get_Event


## -description


The 
<b>get_Event</b> method gets the 
<a href="https://docs.microsoft.com/windows/desktop/api/tapi3if/ne-tapi3if-qos_event">QOS_EVENT</a> indicator.


## -parameters




### -param pQosEvent [out]

Indicator of the QOS event type.
