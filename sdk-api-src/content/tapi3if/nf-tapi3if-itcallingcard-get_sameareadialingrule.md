# ITCallingCard::get_SameAreaDialingRule


## -description


The 
<b>get_SameAreaDialingRule</b> method gets the dialing rules for calls within the same area code.


## -parameters




### -param ppRule [out]

Pointer to <b>BSTR</b> representation of local area dialing rules.
