# ITCallHub::get_State


## -description


The 
<b>get_State</b> method gets the current state of the CallHub.


## -parameters




### -param pState [out]

Pointer to 
<a href="https://docs.microsoft.com/windows/desktop/api/tapi3if/ne-tapi3if-callhub_state">CALLHUB_STATE</a> indicator of state.
