# ITFileTrack interface


## -description

The 
<b>ITFileTrack</b> interface exposes methods that allow an application to get and set information concerning file terminal tracks. The 
<a href="/windows/desktop/api/tapi3if/nf-tapi3if-itfileterminalevent-get_track">ITFileTerminalEvent::get_Track</a> method creates the 
<b>ITFileTrack</b> interface.

## -inheritance

The <b xmlns:loc="http://microsoft.com/wdcml/l10n">ITFileTrack</b> interface inherits from the <a href="/previous-versions/windows/desktop/api/oaidl/nn-oaidl-idispatch">IDispatch</a> interface. <b>ITFileTrack</b> also has these types of members:

</ul>
