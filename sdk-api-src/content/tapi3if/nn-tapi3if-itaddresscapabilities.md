# ITAddressCapabilities interface


## -description

The 
<b>ITAddressCapabilities</b> interface is used to obtain information about an address's capabilities. It is on the Address object, and an application can access it by calling <b>QueryInterface</b> on the Address object.

## -inheritance

The <b xmlns:loc="http://microsoft.com/wdcml/l10n">ITAddressCapabilities</b> interface inherits from the <a href="/previous-versions/windows/desktop/api/oaidl/nn-oaidl-idispatch">IDispatch</a> interface. <b>ITAddressCapabilities</b> also has these types of members:
