# ITCallHub interface


## -description

The 
<b>ITCallHub</b> interface provides methods to retrieve information concerning a CallHub object. The 
<a href="/windows/desktop/api/tapi3if/nf-tapi3if-ienumcallhub-next">IEnumCallHub::Next</a> and 
<a href="/windows/desktop/api/tapi3if/nf-tapi3if-ittapi-get_callhubs">ITTapi::get_CallHubs</a> methods create the 
<b>ITCallHub</b> interface.
