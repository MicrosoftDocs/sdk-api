# SOCKET_USAGE_TYPE enumeration


## -description

The Windows Sockets 
<b>SOCKET_USAGE_TYPE</b> enumeration is used to specified the usage type for the socket.

## -enum-fields

### -field SYSTEM_CRITICAL_SOCKET:1

The usage type is critical to the system.
