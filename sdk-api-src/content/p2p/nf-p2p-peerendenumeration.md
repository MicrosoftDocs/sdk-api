# PeerEndEnumeration function


## -description



      The <b>PeerEndEnumeration</b> function releases an enumeration, for example, a record or member enumeration, and deallocates all resources associated with the enumeration.


## -parameters




### -param hPeerEnum [in]

Handle to the enumeration to be released. This handle is generated by a peer enumeration function.


## -returns



Returns <b>S_OK</b> if the operation succeeds. Otherwise, the function returns the following value.
