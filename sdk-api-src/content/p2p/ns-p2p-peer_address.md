# PEER_ADDRESS structure


## -description


The <b>PEER_ADDRESS</b> structure specifies the  information about the IP address.


## -struct-fields




### -field dwSize

Specifies the size of this structure.


### -field sin6

Specifies the IP address of the node in the Peer Infrastructure.
