# ITextDocument2::GetStory


## -description


Retrieves the story that corresponds to a particular index.


## -parameters




### -param Index [in]

Type: <b>long</b>

The index of the story to retrieve.


### -param ppStory [out, retval]

Type: <b><a href="https://docs.microsoft.com/windows/desktop/api/tom/nn-tom-itextstory">ITextStory</a>**</b>

The story.


## -returns



Type: <b><a href="https://docs.microsoft.com/windows/desktop/WinProg/windows-data-types">HRESULT</a></b>

If the method succeeds, it returns <b>NOERROR</b>. Otherwise, it returns an <b>HRESULT</b> error code.
