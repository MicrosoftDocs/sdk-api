# ITextStoryRanges::GetCount


## -description


Retrieves the number of stories in the specified stories collection.


## -parameters




### -param pCount

Type: <b>long*</b>

The count of stories. 


## -returns



Type: <b>HRESULT</b>

This method returns <b>S_OK</b>.
