# ITextPara::GetPageBreakBefore


## -description


Determines whether each paragraph in the range must begin on a new page.


## -parameters




### -param pValue

Type: <b>long*</b>

A variable that is one of the following values: 
					

<table class="clsStd">
<tr>
<th>Value</th>
<th>Meaning</th>
</tr>
<tr>
<td>tomTrue</td>
<td>Each paragraph in this range must begin on a new page.</td>
</tr>
<tr>
<td>tomFalse</td>
<td>The paragraphs in this range do not need to begin on a new page.</td>
</tr>
<tr>
<td>tomUndefined</td>
<td>The property is undefined.</td>
</tr>
</table>
 


## -returns



Type: <b>HRESULT</b>

If <b>ITextPara::GetPageBreakBefore</b> succeeds, it returns <b>S_OK</b>. If the method fails, it returns one of the following COM error codes.
