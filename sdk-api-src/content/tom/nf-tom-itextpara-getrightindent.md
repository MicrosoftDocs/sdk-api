# ITextPara::GetRightIndent


## -description


Retrieves the size of the right margin indent of a paragraph. 


## -parameters




### -param pValue

Type: <b>float*</b>

The right indentation, in floating-point points. 


## -returns



Type: <b>HRESULT</b>

If <b>ITextPara::GetRightIndent</b> succeeds, it returns <b>S_OK</b>. If the method fails, it returns one of the following COM error codes. For more information about COM error codes, see <a href="https://docs.microsoft.com/windows/desktop/com/error-handling-in-com">Error Handling in COM</a>.
