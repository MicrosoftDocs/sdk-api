# ITextStrings::Cat2


## -description

Concatenates two strings.

## -parameters

### -param iString [in]

Type: <b>long</b>

The string.

## -returns

Type: <b><a href="/windows/desktop/WinProg/windows-data-types">HRESULT</a></b>

If the method succeeds, it returns <b>NOERROR</b>. Otherwise, it returns an <b>HRESULT</b> error code.
