# MANCODE enumeration


## -description

Represents mathematical alphanumeric codes.

### -field MITAL:0x20

Italics

### -field MGREEK:0x40

Greek

### -field MROMN:0

Roman

### -field MSCRP:1

Script

### -field MFRAK:2

Fraktur

### -field MOPEN:3

Double struck

### -field MSANS:4

Sans-serif

### -field MMONO:5

Monospaced

### -field MMATH:6

Math

### -field MISOL:7

Isolated

### -field MINIT:8

Initial

### -field MTAIL:9

Tailed

### -field MSTRCH:10

Stretched

### -field MLOOP:11

Looped

### -field MOPENA:12

Arabic double-struck
