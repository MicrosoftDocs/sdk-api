# ITextRange::ChangeCase


## -description


Changes the case of letters in this range according to the 
			<i>Type</i> parameter.


## -parameters




### -param Type [in]

Type: <b>long</b>

Type of case change. The default value is <i>tomLower</i>.

This method returns an
