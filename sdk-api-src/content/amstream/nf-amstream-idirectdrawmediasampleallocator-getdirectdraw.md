# IDirectDrawMediaSampleAllocator::GetDirectDraw


## -description



The <code>GetDirectDraw</code> method retrieves a pointer to the DirectDraw instance used to allocate surfaces.




## -parameters




### -param ppDirectDraw [out]

Address of a pointer that receives the DirectDraw object's <b>IDirectDraw</b> interface.


## -returns



Returns an <b>HRESULT</b> value.




## -remarks



The caller should release the returned <b>IDirectDraw</b> pointer, except when calling the <a href="https://docs.microsoft.com/windows/desktop/DirectShow/overlay-mixer-filter">Overlay Mixer</a> filter's implementation of this interface.
