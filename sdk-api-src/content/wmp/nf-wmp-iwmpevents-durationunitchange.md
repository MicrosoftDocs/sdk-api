# IWMPEvents::DurationUnitChange


## -description



The <b>DurationUnitChange</b> event is reserved for future use.




## -parameters




### -param NewDurationUnit [in]

Not supported.


## -returns



This method does not return a value.




## -remarks



This event is reserved for future use.
