# IWMPEvents::OpenPlaylistSwitch


## -description

The <b>OpenPlaylistSwitch</b> event occurs when a title on a DVD begins playing.

## -parameters

### -param pItem [in]

Pointer to an <b>IDispatch</b> interface for the given playlist.
