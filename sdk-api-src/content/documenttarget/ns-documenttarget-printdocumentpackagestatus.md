# PrintDocumentPackageStatus structure


## -description


Defines a payload to be used by the <a href="https://docs.microsoft.com/windows/desktop/api/documenttarget/nf-documenttarget-iprintdocumentpackagestatusevent-packagestatusupdated">PackageStatusUpdated</a> method. This structure is a generic version of XPS_JOB_STATUS.


## -struct-fields




### -field JobId

The job ID.


### -field CurrentDocument

The zero-based index of the most recently processed document.


### -field CurrentPage

The zero-based index of the most recently processed page in the current document


### -field CurrentPageTotal

A running total of the number of pages that have been processed by the print job.


### -field Completion

The completion status of the job.


### -field PackageStatus

The error state of the job.
