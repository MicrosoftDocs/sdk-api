# StringFormat::GetDigitSubstitutionMethod


## -description


The <b>StringFormat::GetDigitSubstitutionMethod</b> method gets an element of the 
			<a href="https://msdn.microsoft.com/library/ms534179(v=VS.85).aspx">StringDigitSubstitute</a> enumeration that indicates the digit substitution method that is used by this 
			<a href="https://msdn.microsoft.com/library/ms534510(v=VS.85).aspx">StringFormat</a> object.


## -parameters






## -returns



Type: <strong>Type: <b><a href="https://msdn.microsoft.com/library/ms534179(v=VS.85).aspx">StringDigitSubstitute</a></b>
</strong>

This method returns an element of the 
						<a href="https://msdn.microsoft.com/library/ms534179(v=VS.85).aspx">StringDigitSubstitute</a> enumeration.




## -remarks



The digit substitution method replaces, in a string, Western European digits with digits that correspond to a user's locale or language.
