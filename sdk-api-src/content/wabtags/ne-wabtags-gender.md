# Gender enumeration


## -description


Do not use. The <b>Gender</b> enumeration specifies the possible values for the PR_GENDER property.


## -enum-fields




### -field genderUnspecified

No gender is specified.


### -field genderFemale

Specifies a gender of female.


### -field genderMale

Specifies a gender of male.
