# ID3D10ShaderReflectionConstantBuffer::GetDesc


## -description


Get a constant-buffer description.


## -parameters




### -param pDesc [in]

Type: <b><a href="https://msdn.microsoft.com/75036245-6ec6-4ca8-8e62-87be47aeee8a">D3D10_SHADER_BUFFER_DESC</a>*</b>

A pointer to a shader-buffer description (see <a href="https://msdn.microsoft.com/75036245-6ec6-4ca8-8e62-87be47aeee8a">D3D10_SHADER_BUFFER_DESC</a>.


## -returns



Type: <b><a href="455d07e9-52c3-4efb-a9dc-2955cbfd38cc">HRESULT</a></b>

Returns one of the following <a href="https://msdn.microsoft.com/7b67d428-d000-4c3e-adc1-b5fc67a15a6a">Direct3D 10 Return Codes</a>.
