# ID3D10ShaderReflectionType::GetDesc


## -description

Get the description of a shader-reflection-variable type.

## -parameters

### -param pDesc [in]

Type: <b><a href="/windows/desktop/api/d3d10shader/ns-d3d10shader-d3d10_shader_type_desc">D3D10_SHADER_TYPE_DESC</a>*</b>

A pointer to a shader-type description (see <a href="/windows/desktop/api/d3d10shader/ns-d3d10shader-d3d10_shader_type_desc">D3D10_SHADER_TYPE_DESC</a>).

## -returns

Type: <b><a href="/windows/win32/com/structure-of-com-error-codes">HRESULT</a></b>

Returns one of the following <a href="/windows/desktop/direct3d10/d3d10-graphics-reference-returnvalues">Direct3D 10 Return Codes</a>.
