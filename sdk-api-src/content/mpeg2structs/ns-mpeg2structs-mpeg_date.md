# MPEG_DATE structure


## -description

The <b>MPEG_DATE</b> structure specifies a date.

## -struct-fields

### -field Date

Specifies the day of the month. The value can range from 1 to 31, inclusive.

### -field Month

Specifies the month. The value can range from 1 to 12, inclusive.

### -field Year

Specifies the year.
