# MPEG_BCS_DEMUX structure


## -description



The <b>MPEG_BCS_DEMUX</b> structure identifies the filter graph that is providing the MPEG-2 data stream.




## -struct-fields




### -field AVMGraphId

Specifies the filter graph instance.
