# TypeTraits structure


## -description

Contains implementations of <b>Point</b>, <b>Size</b>, and <b>Rect</b> that store their data using the specified type.

## -struct-fields

### -field field

Contains the typedef for <b>Point</b>, <b>Size</b>, or <b>Rect</b> with the type specified in the template type.

### -field .Type

The template type.
