# Point2F function


## -description


Creates a <a href="https://docs.microsoft.com/windows/desktop/Direct2D/d2d1-point-2f">D2D1_POINT_2F</a> structure that contains the specified x-coordinates and y-coordinates.


## -parameters




### -param x

Type: <b>FLOAT</b>

The x-coordinate of the point. The default value is 0.f.


### -param y

Type: <b>FLOAT</b>

The y-coordinate of the point. The default value is 0.f.


## -returns



Type: <b><a href="https://docs.microsoft.com/windows/desktop/Direct2D/d2d1-point-2f">D2D1_POINT_2F</a></b>

The new point.
