# Point2F function


## -description


Creates a <a href="https://msdn.microsoft.com/b317ae75-d738-4e1a-bcd1-adf3e95b197e">D2D1_POINT_2F</a> structure that contains the specified x-coordinates and y-coordinates.


## -parameters




### -param x

Type: <b>FLOAT</b>

The x-coordinate of the point. The default value is 0.f.


### -param y

Type: <b>FLOAT</b>

The y-coordinate of the point. The default value is 0.f.


## -returns



Type: <b><a href="https://msdn.microsoft.com/b317ae75-d738-4e1a-bcd1-adf3e95b197e">D2D1_POINT_2F</a></b>

The new point.
