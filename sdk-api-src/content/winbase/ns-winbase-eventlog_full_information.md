# EVENTLOG_FULL_INFORMATION structure


## -description


Indicates whether the event log is full.


## -struct-fields




### -field dwFull

Indicates whether the event log is full. If the log is full, this member is <b>TRUE</b>. Otherwise, it is <b>FALSE</b>.
