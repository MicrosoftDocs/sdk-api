# IFileSystemImage2::get_BootImageOptionsArray


## -description


Retrieves the boot option array that will be utilized to generate the file system image.


## -parameters




### -param pVal [out]

Pointer to a boot option array that contains a list of <a href="https://docs.microsoft.com/windows/desktop/api/imapi2fs/nn-imapi2fs-ibootoptions">IBootOptions</a> interfaces of boot images used to generate the file system image. Each element of the list is a <b>VARIANT</b> of type <b>VT_DISPATCH</b>.

## -remarks



 The <b>SAFEARRAY</b> will be a one-dimensional array. If a boot image is not specified, a zero-sized array will be returned.
