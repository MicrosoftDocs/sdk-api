# _DDGETPREVIOUSAUTOFLIPININFO structure


## -description


The DDGETPREVIOUSAUTOFLIPININFO structure provides the <a href="https://msdn.microsoft.com/a1de1905-09f3-4689-ace9-06690a1f930a">video port extensions (VPE)</a> object information. 


## -struct-fields




### -field lpVideoPortData

Points to a <a href="https://msdn.microsoft.com/library/windows/hardware/ff550395">DDVIDEOPORTDATA</a> structure that contains the VPE object information.
