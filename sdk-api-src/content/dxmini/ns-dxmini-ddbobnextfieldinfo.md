# DDBOBNEXTFIELDINFO structure


## -description

The DDBOBNEXTFIELDINFO structure contains the bob information for the surface.

## -struct-fields

### -field lpSurface

Points to a <a href="/windows/desktop/api/dxmini/ns-dxmini-ddsurfacedata">DDSURFACEDATA</a> structure that represents the surface.
