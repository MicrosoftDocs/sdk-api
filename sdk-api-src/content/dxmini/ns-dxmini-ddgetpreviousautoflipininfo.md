# DDGETPREVIOUSAUTOFLIPININFO structure


## -description

The DDGETPREVIOUSAUTOFLIPININFO structure provides the <a href="/windows-hardware/drivers/">video port extensions (VPE)</a> object information.

## -struct-fields

### -field lpVideoPortData

Points to a <a href="/windows/desktop/api/dxmini/ns-dxmini-ddvideoportdata">DDVIDEOPORTDATA</a> structure that contains the VPE object information.
