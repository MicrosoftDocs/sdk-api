# RASTERIZER_STATUS structure


## -description



The <b>RASTERIZER_STATUS</b> structure contains information about whether TrueType is installed. This structure is filled when an application calls the <a href="https://docs.microsoft.com/windows/desktop/api/wingdi/nf-wingdi-getrasterizercaps">GetRasterizerCaps</a> function.




## -struct-fields




### -field nSize

The size, in bytes, of the <b>RASTERIZER_STATUS</b> structure.


### -field wFlags

Specifies whether at least one TrueType font is installed and whether TrueType is enabled. This value is TT_AVAILABLE, TT_ENABLED, or both if TrueType is on the system.


### -field nLanguageID

The language in the system's Setup.inf file.
