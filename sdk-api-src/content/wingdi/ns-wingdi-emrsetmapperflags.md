# EMRSETMAPPERFLAGS structure


## -description



The <b>EMRSETMAPPERFLAGS</b> structure contains members for the <b>SetMapperFlags</b> enhanced metafile record.




## -struct-fields




### -field emr

The base structure for all record types.


### -field dwFlags

Font mapper flag.
