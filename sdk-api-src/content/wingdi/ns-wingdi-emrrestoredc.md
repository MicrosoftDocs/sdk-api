# EMRRESTOREDC structure


## -description



The <b>EMRRESTOREDC</b> structure contains members for the <b>RestoreDC</b> enhanced metafile record.




## -struct-fields




### -field emr

The base structure for all record types.


### -field iRelative

Relative instance to restore.
