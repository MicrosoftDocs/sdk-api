# GetBValue macro


## -description



The <b>GetBValue</b> macro retrieves an intensity value for the blue component of a red, green, blue (RGB) value.




## -parameters




### -param rgb

Specifies an RGB color value.


## -remarks



The intensity value is in the range 0 through 255.
