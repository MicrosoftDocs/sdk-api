# tagEMRCREATEPEN structure


## -description



The <b>EMRCREATEPEN</b> structure contains members for the <b>CreatePen</b> enhanced metafile record.




## -struct-fields




### -field emr

The base structure for all record types.


### -field ihPen

Index to pen in handle table.


### -field lopn

Logical pen.
