# EMRSETMITERLIMIT structure


## -description

The <b>EMRSETMITERLIMIT</b> structure contains members for the <b>SetMiterLimit</b> enhanced metafile record.

## -struct-fields

### -field emr

The base structure for all record types.

### -field eMiterLimit

New miter limit.
