# OffsetWindowOrgEx function


## -description

The <b>OffsetWindowOrgEx</b> function modifies the window origin for a device context using the specified horizontal and vertical offsets.

## -parameters

### -param hdc [in]

A handle to the device context.

### -param x [in]

The horizontal offset, in logical units.

### -param y [in]

The vertical offset, in logical units.

### -param lppt [out]

A pointer to a <a href="/previous-versions/dd162805(v=vs.85)">POINT</a> structure. The logical coordinates of the previous window origin are placed in this structure. If <i>lpPoint</i> is <b>NULL</b>, the previous origin is not returned.

## -returns

If the function succeeds, the return value is nonzero.

If the function fails, the return value is zero.
