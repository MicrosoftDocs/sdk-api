# ICryptAttributes::get_Count


## -description


The <b>Count</b> property retrieves the number of <a href="https://docs.microsoft.com/windows/desktop/api/certenroll/nn-certenroll-icryptattribute">ICryptAttribute</a> objects in the collection.

This property is read-only.
