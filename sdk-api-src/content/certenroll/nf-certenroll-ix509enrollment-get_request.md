# IX509Enrollment::get_Request method


## -description


The <b>Request</b> property retrieves the certificate request associated with the enrollment object.

This property is read-only.


## -parameters


## -remarks



This property can be set when the <a href="https://msdn.microsoft.com/04cb00af-f786-4548-bee3-2cc5083278c3">InitializeFromRequest</a> method is called.
