# ICspStatuses::get_ItemByName


## -description

The <b>ItemByName</b> property retrieves an <a href="/windows/desktop/api/certenroll/nn-certenroll-icspstatus">ICspStatus</a> object from the collection by provider and algorithm name. This property is web enabled.

This property is read-only.
