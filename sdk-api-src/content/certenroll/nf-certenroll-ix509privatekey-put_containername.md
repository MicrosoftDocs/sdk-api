# IX509PrivateKey::put_ContainerName method


## -description


The <b>ContainerName</b> property specifies or retrieves the name of the key container. This property is web enabled for both input and output.

This property is read/write.


## -parameters


## -remarks



If you do not specify a name, one is created when the <a href="https://msdn.microsoft.com/e8c6564a-6009-437e-9b83-3711e43a7374">Create</a> method is called.
