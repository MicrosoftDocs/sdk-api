# Pkcs10AllowedSignatureTypes enumeration


## -description


The <b>Pkcs10AllowedSignatureTypes</b> enumeration type specifies the type of signature permitted when signing a <a href="https://msdn.microsoft.com/en-us/library/ms721572(v=VS.85).aspx">certificate request</a>. This enumeration is used by the <a href="https://msdn.microsoft.com/en-us/library/Aa377510(v=VS.85).aspx">CheckSignature</a> method on the <a href="https://msdn.microsoft.com/en-us/library/Aa377505(v=VS.85).aspx">IX509CertificateRequestPkcs10</a> interface to determine whether a signature exists and is valid.


## -enum-fields




### -field AllowedKeySignature

Signatures generated by using <a href="https://msdn.microsoft.com/en-us/library/ms721572(v=VS.85).aspx">cryptographic keys</a> are allowed.


### -field AllowedNullSignature

Null signatures are allowed. A null signature is a SHA-1 <a href="https://msdn.microsoft.com/en-us/library/ms721586(v=VS.85).aspx">hash</a>.
