# ICertPropertyKeyProvInfo::get_PrivateKey


## -description

The <b>PrivateKey</b> property retrieves the private key associated with the certificate.

This property is read-only.

## -parameters

## -remarks

 Call the <a href="/windows/desktop/api/certenroll/nf-certenroll-icertpropertykeyprovinfo-initialize">Initialize</a> method to create a description.
