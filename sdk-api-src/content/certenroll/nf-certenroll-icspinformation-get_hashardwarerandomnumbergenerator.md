# ICspInformation::get_HasHardwareRandomNumberGenerator


## -description

The <b>HasHardwareRandomNumberGenerator</b> property retrieves a Boolean value that specifies whether the provider supports a hardware random number generator that can be used to create random bytes for cryptographic operations.

This property is read-only.

## -parameters

## -remarks

There are currently no Microsoft cryptographic providers that support this feature.
