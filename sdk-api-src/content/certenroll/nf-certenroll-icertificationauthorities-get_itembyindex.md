# ICertificationAuthorities::get_ItemByIndex


## -description

The <b>ItemByIndex</b> property retrieves an <a href="/windows/desktop/api/certenroll/nn-certenroll-icertificationauthority">ICertificationAuthority</a> object from the collection by index number.   This property is web enabled.

This property is read-only.
