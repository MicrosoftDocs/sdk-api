# ICspInformations::get_ItemByName


## -description


The <b>ItemByName</b> property retrieves an <a href="https://docs.microsoft.com/windows/desktop/api/certenroll/nn-certenroll-icspinformation">ICspInformation</a> object from the collection by name. This property is web enabled.

This property is read-only.
