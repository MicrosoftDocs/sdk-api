# ICertificatePolicies interface


## -description

The <b>ICertificatePolicies</b> interface contains methods and properties that enable you to manage a collection of <a href="/windows/desktop/api/certenroll/nn-certenroll-icertificatepolicy">ICertificatePolicy</a> objects.

## -inheritance

The <b xmlns:loc="http://microsoft.com/wdcml/l10n">ICertificatePolicies</b> interface inherits from the <a href="/previous-versions/windows/desktop/api/oaidl/nn-oaidl-idispatch">IDispatch</a> interface. <b>ICertificatePolicies</b> also has these types of members:

</ul>
