# IX509Extensions::get_ItemByIndex


## -description

The <b>ItemByIndex</b> property retrieves an <a href="/windows/desktop/api/certenroll/nn-certenroll-ix509extension">IX509Extension</a> object from the collection by index number.  This property is web enabled.

This property is read-only.
