# IX509CertificateRequestPkcs10V3::get_EncryptionStrength


## -description


Identifies the bit length for the <a href="https://msdn.microsoft.com/en-us/library/Dn379349(v=VS.85).aspx">EncryptionAlgorithm</a> to use for encryption. If the <b>EncryptionAlgorithm</b> only supports one bit length, then you do not need to specify a value for the <b>EncryptionStrength</b> property.

This property is read/write.
