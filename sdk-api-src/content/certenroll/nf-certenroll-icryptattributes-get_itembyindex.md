# ICryptAttributes::get_ItemByIndex


## -description


The <b>ItemByIndex</b> property retrieves an <a href="https://msdn.microsoft.com/en-us/library/Aa375929(v=VS.85).aspx">ICryptAttribute</a> object from the collection by index number.

This property is read-only.
