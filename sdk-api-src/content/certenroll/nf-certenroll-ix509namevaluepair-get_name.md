# IX509NameValuePair::get_Name


## -description


The <b>Name</b> property retrieves the name portion of the name-value pair.

This property is read-only.


## -parameters


## -remarks



You must call the <a href="https://msdn.microsoft.com/library/windows/hardware/ff550945">Initialize</a> method before calling this property.
