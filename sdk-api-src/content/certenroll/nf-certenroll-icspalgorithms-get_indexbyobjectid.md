# ICspAlgorithms::get_IndexByObjectId


## -description


The <b>IndexByObjectId</b> property retrieves the index of an <a href="https://msdn.microsoft.com/en-us/library/Aa375947(v=VS.85).aspx">ICspAlgorithm</a> object by <a href="https://msdn.microsoft.com/en-us/library/ms721599(v=VS.85).aspx">object identifier</a> (OID).

This property is read-only.
