# ICertPropertyDescription::get_Description


## -description


The <b>Description</b> property retrieves  a description of the certificate. The description identifies the intended use of the certificate.

This property is read-only.


## -parameters


## -remarks



 Call the <a href="https://msdn.microsoft.com/17e8ee9c-c861-4437-a70d-ccad6a5a293d">Initialize</a> method to create a description.
