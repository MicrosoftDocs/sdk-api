# ICertPropertyDescription::get_Description


## -description


The <b>Description</b> property retrieves  a description of the certificate. The description identifies the intended use of the certificate.

This property is read-only.


## -parameters


## -remarks



 Call the <a href="https://docs.microsoft.com/windows/desktop/api/certenroll/nf-certenroll-icertpropertydescription-initialize">Initialize</a> method to create a description.
