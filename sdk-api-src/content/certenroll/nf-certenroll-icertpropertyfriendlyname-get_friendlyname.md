# ICertPropertyFriendlyName::get_FriendlyName


## -description

The <b>FriendlyName</b> property retrieves the display name of the certificate.

This property is read-only.

## -parameters

## -remarks

 Call the <a href="/windows/desktop/api/certenroll/nf-certenroll-icertpropertyfriendlyname-initialize">Initialize</a> method to create a description.
