# ICertPropertyBackedUp::get_BackedUpValue method


## -description


The <b>BackedUpValue</b> property retrieves a Boolean value that identifies whether the certificate was backed up.

This property is read-only.


## -parameters


## -remarks



Call the <a href="https://msdn.microsoft.com/library/windows/hardware/ff550945">Initialize</a> method or the <a href="https://msdn.microsoft.com/2033c947-661c-4a52-b24f-82fa71ba7868">InitializeFromCurrentTime</a> method to set the <b>BackedUpValue</b> property value.
