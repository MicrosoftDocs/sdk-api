# ICspStatuses::get_Count


## -description

The <b>Count</b> property retrieves the number of <a href="/windows/desktop/api/certenroll/nn-certenroll-icspstatus">ICspStatus</a> objects in the collection. This property is web enabled.

This property is read-only.
