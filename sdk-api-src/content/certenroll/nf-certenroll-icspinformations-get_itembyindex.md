# ICspInformations::get_ItemByIndex


## -description


The <b>ItemByIndex</b> property retrieves an <a href="https://docs.microsoft.com/windows/desktop/api/certenroll/nn-certenroll-icspinformation">ICspInformation</a> object from the collection by index number.  This property is web enabled.

This property is read-only.
