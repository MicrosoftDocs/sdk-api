# ICspInformations::get_ItemByIndex


## -description


The <b>ItemByIndex</b> property retrieves an <a href="https://msdn.microsoft.com/e337ae2c-6f86-4025-8d31-47bc5d8a4ca8">ICspInformation</a> object from the collection by index number.  This property is web enabled.

This property is read-only.
