# IX509NameValuePairs::get_ItemByIndex


## -description


The <b>ItemByIndex</b> property retrieves an <a href="https://docs.microsoft.com/windows/desktop/api/certenroll/nn-certenroll-ix509namevaluepair">IX509NameValuePair</a> object from the collection by index number.

This property is read-only.
