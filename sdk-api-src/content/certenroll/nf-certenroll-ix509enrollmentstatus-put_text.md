# IX509EnrollmentStatus::put_Text


## -description


The <b>Text</b> property specifies or retrieves a string that contains a message associated with the status of the enrollment process.

This property is read/write.


## -parameters


## -remarks



You can call the <a href="https://docs.microsoft.com/windows/desktop/api/certenroll/nf-certenroll-ix509enrollmentstatus-appendtext">AppendText</a> method to add information to the message.
