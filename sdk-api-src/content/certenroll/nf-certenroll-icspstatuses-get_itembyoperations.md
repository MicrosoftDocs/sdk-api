# ICspStatuses::get_ItemByOperations


## -description


The <b>ItemByOperations</b> property retrieves an <a href="https://msdn.microsoft.com/en-us/library/Aa376760(v=VS.85).aspx">ICspStatus</a> object that has the same name as the  provider specified on input and the same algorithm but identifies a different cryptographic operation.

This property is read-only.
