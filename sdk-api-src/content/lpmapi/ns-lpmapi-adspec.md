# ADSPEC structure


## -description


The 
<b>ADSPEC</b> structure contains Adspec message information for RSVP.


## -struct-fields




### -field adspec_header

Adspec header, expressed as an <a href="https://msdn.microsoft.com/90a237c0-0e62-4f27-927a-e3f3c1ac629e">RsvpObjHdr</a> structure.


### -field adspec_body

Adspec message body.
