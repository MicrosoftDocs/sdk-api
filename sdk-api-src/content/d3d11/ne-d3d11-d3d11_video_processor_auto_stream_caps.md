# D3D11_VIDEO_PROCESSOR_AUTO_STREAM_CAPS enumeration


## -description

Specifies the automatic image processing capabilities of the video processor.

## -enum-fields

### -field D3D11_VIDEO_PROCESSOR_AUTO_STREAM_CAPS_DENOISE:0x1

Denoise.

### -field D3D11_VIDEO_PROCESSOR_AUTO_STREAM_CAPS_DERINGING:0x2

Deringing.

### -field D3D11_VIDEO_PROCESSOR_AUTO_STREAM_CAPS_EDGE_ENHANCEMENT:0x4

Edge enhancement.

### -field D3D11_VIDEO_PROCESSOR_AUTO_STREAM_CAPS_COLOR_CORRECTION:0x8

Color correction.

### -field D3D11_VIDEO_PROCESSOR_AUTO_STREAM_CAPS_FLESH_TONE_MAPPING:0x10

Flesh-tone mapping.

### -field D3D11_VIDEO_PROCESSOR_AUTO_STREAM_CAPS_IMAGE_STABILIZATION:0x20

Image stabilization.

### -field D3D11_VIDEO_PROCESSOR_AUTO_STREAM_CAPS_SUPER_RESOLUTION:0x40

Enhanced image resolution.

### -field D3D11_VIDEO_PROCESSOR_AUTO_STREAM_CAPS_ANAMORPHIC_SCALING:0x80

Anamorphic scaling.
