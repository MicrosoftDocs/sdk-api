# CD3D11_RECT::CD3D11_RECT(const D3D11_RECT &)


## -description

Instantiates a new instance of a <a href="/windows/win32/api/d3d11/ns-d3d11-cd3d11_rect">CD3D11_RECT</a> structure that is initialized with a **D3D11_RECT** structure.

## -parameters

### -param o

Address of the **D3D11_RECT** structure that initializes the **CD3D11_RECT** structure.
