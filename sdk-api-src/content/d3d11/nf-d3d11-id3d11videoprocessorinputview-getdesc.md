# ID3D11VideoProcessorInputView::GetDesc


## -description

Gets the properties of the video processor input view.

## -parameters

### -param pDesc [out]

A pointer to a <a href="/windows/desktop/api/d3d11/ns-d3d11-d3d11_video_processor_input_view_desc">D3D11_VIDEO_PROCESSOR_INPUT_VIEW_DESC</a> structure. The method fills the structure with the view properties.
