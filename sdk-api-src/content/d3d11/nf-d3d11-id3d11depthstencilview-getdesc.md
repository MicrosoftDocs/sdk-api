# ID3D11DepthStencilView::GetDesc


## -description

Get the depth-stencil view.

Type: <b><a href="/windows/desktop/api/d3d11/ns-d3d11-d3d11_depth_stencil_view_desc">D3D11_DEPTH_STENCIL_VIEW_DESC</a>*</b>
