# tagXHR_CERT structure


## -description


Defines a buffer that points to an encoded certificate.


## -struct-fields




### -field cbCert

 The size, in bytes, of the encoded certificate.


### -field pbCert

A pointer to the buffer that contains the encoded certificate.
