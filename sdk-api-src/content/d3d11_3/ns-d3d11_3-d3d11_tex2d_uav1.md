# D3D11_TEX2D_UAV1 structure


## -description

Describes a unordered-access 2D texture resource.

## -struct-fields

### -field MipSlice

The mipmap slice index.

### -field PlaneSlice

The index (plane slice number) of the plane to use in the texture.
