# ID3D11DeviceContext3::SetHardwareProtectionState


## -description


Sets the hardware protection state.
        


## -parameters




### -param HwProtectionEnable [in]

Type: <b>BOOL</b>

Specifies whether to enable hardware protection.
          


## -returns



This method does not return a value.
