## -description


The <b>IDirectDrawKernel::GetKernelHandle</b> method returns a kernel-mode handle to the DirectDraw object.


## -parameters






#### -param unnamedParam1

Caller-supplied pointer into which the kernel-mode handle of the DirectDraw object is returned.
