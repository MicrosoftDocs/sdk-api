# IP4_ARRAY structure


## -description


The <b>IP4_ARRAY</b> structure stores an array of IPv4 addresses.


## -struct-fields




### -field AddrCount

The number of IPv4 addresses in <b>AddrArray</b>.


### -field AddrArray.size_is

 


### -field AddrArray.size_is.AddrCount

 


### -field AddrArray

An array of <a href="https://docs.microsoft.com/windows/desktop/DNS/dns-data-types">IP4_ADDRESS</a> data types that contains a list of IPv4 address.
