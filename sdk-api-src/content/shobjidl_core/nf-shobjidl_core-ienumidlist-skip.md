# IEnumIDList::Skip


## -description


Skips the specified number of elements in the enumeration sequence.


## -parameters




### -param celt

Type: <b>ULONG</b>

The number of item identifiers to skip.


## -returns



Type: <b>HRESULT</b>

Returns S_OK if successful, or a COM-defined error value otherwise.
