# THUMBBUTTONMASK enumeration


## -description


Used by the <a href="https://docs.microsoft.com/windows/desktop/api/shobjidl_core/ns-shobjidl_core-thumbbutton">THUMBBUTTON</a> structure to specify which members of that structure contain valid data.


## -enum-fields




### -field THB_BITMAP

The <b>iBitmap</b> member contains valid information.


### -field THB_ICON

The <b>hIcon</b> member contains valid information.


### -field THB_TOOLTIP

The <b>szTip</b> member contains valid information.


### -field THB_FLAGS

The <b>dwFlags</b> member contains valid information.
