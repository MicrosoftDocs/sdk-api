# IFileDialogCustomize::RemoveAllControlItems


## -description

Not implemented.

## -parameters

### -param dwIDCtl [in]

Type: <b>DWORD</b>

The ID of the container control from which to remove the items.

## -returns

Type: <b>HRESULT</b>

If this method succeeds, it returns <b>S_OK</b>. Otherwise, it returns an <b>HRESULT</b> error code.
