# IFolderView::GetAutoArrange


## -description


Gets the current state of the folder's Auto Arrange mode.


## -parameters






## -returns



Type: <b>HRESULT</b>

Returns S_OK if the folder is in Auto Arrange mode; S_FALSE if it is not.
