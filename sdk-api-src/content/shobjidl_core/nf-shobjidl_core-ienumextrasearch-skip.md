# IEnumExtraSearch::Skip


## -description

Skip a specified number of objects.

## -parameters

### -param celt

Type: <b>ULONG</b>

The number of objects to skip.

## -returns

Type: <b>HRESULT</b>

Returns S_OK if successful, or a COM-defined error code otherwise.
