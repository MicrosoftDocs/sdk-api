# IFolderViewSettings::GetIconSize


## -description

Gets the folder icon size.

## -parameters

### -param puIconSize [out]

Type: <b>UINT*</b>

A pointer to the icon size.

## -returns

Type: <b>HRESULT</b>

If this method succeeds, it returns <b>S_OK</b>. Otherwise, it returns an <b>HRESULT</b> error code.
