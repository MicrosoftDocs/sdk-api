# IOperationsProgressDialog::PauseTimer


## -description

Pauses progress dialog timer.



## -returns

Type: <b>HRESULT</b>

If this method succeeds, it returns <b>S_OK</b>. Otherwise, it returns an <b>HRESULT</b> error code.
