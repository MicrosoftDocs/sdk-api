# IEnumFullIDList::Reset


## -description

Returns the enumerator to the beginning of the enumeration sequence.



## -returns

Type: <b>HRESULT</b>

If this method succeeds, it returns <b>S_OK</b>. Otherwise, it returns an <b>HRESULT</b> error code.
