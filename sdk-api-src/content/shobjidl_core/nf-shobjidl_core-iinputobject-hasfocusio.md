# IInputObject::HasFocusIO


## -description

Determines if one of the object's windows has the keyboard focus.

## -parameters

## -returns

Type: <b>HRESULT</b>

Returns S_OK if one of the object's windows has the keyboard focus, or S_FALSE otherwise.
