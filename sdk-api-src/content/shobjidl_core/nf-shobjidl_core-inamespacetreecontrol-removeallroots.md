# INameSpaceTreeControl::RemoveAllRoots


## -description

Removes all roots and their children from a tree.



## -returns

Type: <b>HRESULT</b>

If this method succeeds, it returns <b>S_OK</b>. Otherwise, it returns an <b>HRESULT</b> error code.
