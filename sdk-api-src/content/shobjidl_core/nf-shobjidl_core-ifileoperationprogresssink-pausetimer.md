# IFileOperationProgressSink::PauseTimer


## -description

Not supported.



## -returns

Type: <b>HRESULT</b>

If this method succeeds, it returns <b>S_OK</b>. Otherwise, it returns an <b>HRESULT</b> error code.

## -remarks

This method should return S_OK rather than E_NOTIMPL.
