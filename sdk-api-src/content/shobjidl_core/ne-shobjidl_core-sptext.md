# SPTEXT enumeration


## -description

Specifies the type of descriptive text being provided to an <a href="/windows/desktop/api/shobjidl_core/nn-shobjidl_core-iactionprogress">IActionProgress</a> interface.

## -enum-fields

### -field SPTEXT_ACTIONDESCRIPTION:1

The text is a high level, short description.

### -field SPTEXT_ACTIONDETAIL

The text is a detailed description.
