# IFileOperation::SetProgressMessage


## -description

Not implemented.

## -parameters

### -param pszMessage [in]

Type: <b>LPCWSTR</b>

Pointer to the window title. This is a null-terminated, Unicode string.

## -returns

Type: <b>HRESULT</b>

If this method succeeds, it returns <b>S_OK</b>. Otherwise, it returns an <b>HRESULT</b> error code.
