# IEnumResources::Skip


## -description

Skips a specified number of resources.

## -parameters

### -param celt [in]

Type: <b>ULONG</b>

The number of resources to skip.

## -returns

Type: <b>HRESULT</b>

If this method succeeds, it returns <b>S_OK</b>. Otherwise, it returns an <b>HRESULT</b> error code.
