# INameSpaceTreeControl::CollapseAll


## -description

Collapses all of the items in the given tree.



## -returns

Type: <b>HRESULT</b>

If this method succeeds, it returns <b>S_OK</b>. Otherwise, it returns an <b>HRESULT</b> error code.
