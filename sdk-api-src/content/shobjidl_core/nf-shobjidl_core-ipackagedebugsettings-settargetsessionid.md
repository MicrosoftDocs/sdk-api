# IPackageDebugSettings::SetTargetSessionId


## -description

Sets the session identifier.

## -parameters

### -param sessionId [in]

The session identifier.

## -returns

If this method succeeds, it returns <b>S_OK</b>. Otherwise, it returns an <b>HRESULT</b> error code.
