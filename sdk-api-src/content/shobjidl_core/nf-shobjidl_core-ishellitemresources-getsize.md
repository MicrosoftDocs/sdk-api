# IShellItemResources::GetSize


## -description

Gets the source size.

## -parameters

### -param pullSize [out]

Type: <b>ULONGLONG*</b>

A pointer to the source size.

## -returns

Type: <b>HRESULT</b>

If this method succeeds, it returns <b>S_OK</b>. Otherwise, it returns an <b>HRESULT</b> error code.
