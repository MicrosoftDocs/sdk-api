# _SPTEXT enumeration


## -description


Specifies the type of descriptive text being provided to an <a href="https://msdn.microsoft.com/e742e381-0fd2-482a-81a0-7b43d11b073b">IActionProgress</a> interface.


## -enum-fields




### -field SPTEXT_ACTIONDESCRIPTION

The text is a high level, short description.


### -field SPTEXT_ACTIONDETAIL

The text is a detailed description.
