# IAppVisibility::IsLauncherVisible


## -description


Gets a value that indicates whether the Start screen is displayed.


## -parameters




### -param pfVisible [out]

<b>TRUE</b> if the Start screen is displayed; otherwise, <b>FALSE.</b>
