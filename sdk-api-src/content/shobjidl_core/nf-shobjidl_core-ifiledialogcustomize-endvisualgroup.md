# IFileDialogCustomize::EndVisualGroup


## -description

Stops the addition of elements to a visual group in the dialog.



## -returns

Type: <b>HRESULT</b>

If this method succeeds, it returns <b>S_OK</b>. Otherwise, it returns an <b>HRESULT</b> error code.
