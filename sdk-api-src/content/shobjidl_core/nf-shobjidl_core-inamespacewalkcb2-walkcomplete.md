# INamespaceWalkCB2::WalkComplete


## -description

Removes data collected during a namespace walk.

## -parameters

### -param hr [in]

Type: <b>HRESULT</b>

The results of <a href="/windows/desktop/api/shobjidl_core/nf-shobjidl_core-inamespacewalk-walk">Walk</a>.

## -returns

Type: <b>HRESULT</b>

Always returns S_OK.
