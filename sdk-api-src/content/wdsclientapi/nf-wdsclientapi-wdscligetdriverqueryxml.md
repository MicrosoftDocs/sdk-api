# WdsCliGetDriverQueryXml function


## -description


This function generates an XML string which can be used to query a WDS server for driver packages using the <a href="https://docs.microsoft.com/windows/desktop/api/wdsclientapi/nf-wdsclientapi-wdscliobtaindriverpackagesex">WdsCliObtainDriverPackagesEx</a> function.  The target OS information section of the WDS driver query XML is generated if the path to the Windows directory of the applied image is specified. 


## -parameters




### -param pwszWinDirPath [in, optional]

The path to the Windows directory of the applied image. This parameter is optional. If it is specified,  the section of the WDS driver query XML  for the target operating system is generated.


### -param ppwszDriverQuery [out]

A pointer to a pointer to a string that receives the generated WDS driver query XML. The caller has to free this buffer using "delete[](*ppwszDriverQuery)".


## -returns



If the function succeeds, the return is <b>S_OK</b>.
