# FWPM_LAYER_ENUM_TEMPLATE0_ structure


## -description



		The <b>FWPM_LAYER_ENUM_TEMPLATE0</b> structure is used for enumerating layers.


## -struct-fields




### -field reserved

Reserved for system use.


## -remarks



Currently, there is no way to limit the
enumeration — all layers are returned.

<b>FWPM_LAYER_ENUM_TEMPLATE0</b> is a specific implementation of FWPM_LAYER_ENUM_TEMPLATE. See <a href="https://msdn.microsoft.com/FBDF53E5-F7DE-4DEB-AC18-6D2BB59FE670">WFP Version-Independent Names and Targeting Specific Versions of Windows</a>  for more information.
