# WCM_TIME_INTERVAL structure


## -description


The <b>WCM_TIME_INTERVAL</b> structure defines a time interval.


## -struct-fields




### -field wYear

Type: <b>WORD</b>

Years.


### -field wMonth

Type: <b>WORD</b>

Months.


### -field wDay

Type: <b>WORD</b>

Days.


### -field wHour

Type: <b>WORD</b>

Hours.


### -field wMinute

Type: <b>WORD</b>

Minutes.


### -field wSecond

Type: <b>WORD</b>

Seconds.


### -field wMilliseconds

Type: <b>WORD</b>

Milliseconds.
