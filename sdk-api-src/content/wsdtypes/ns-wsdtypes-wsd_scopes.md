# WSD_SCOPES structure


## -description


A collection of scopes used in WS-Discovery messaging.


## -struct-fields




### -field MatchBy

A matching rule used for scopes.


### -field Scopes

Reference to a <a href="https://msdn.microsoft.com/86d77741-39c3-44bd-b072-d2d4eb99e488">WSD_URI_LIST</a> structure that contains a list of scopes.
