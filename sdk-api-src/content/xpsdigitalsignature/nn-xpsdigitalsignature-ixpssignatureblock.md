# IXpsSignatureBlock interface


## -description

Represents a block of signature requests  that are  stored in a SignatureDefinitions part.

## -inheritance

The <b xmlns:loc="http://microsoft.com/wdcml/l10n">IXpsSignatureBlock</b> interface inherits from the <a href="/windows/desktop/api/unknwn/nn-unknwn-iunknown">IUnknown</a> interface. <b>IXpsSignatureBlock</b> also has these types of members:
<ul>
<li><a href="https://docs.microsoft.com/">Methods</a></li>
</ul>

## -remarks

This interface cannot exist independently from the signature manager from which it was instantiated.
