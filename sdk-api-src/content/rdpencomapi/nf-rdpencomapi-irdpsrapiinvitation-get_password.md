# IRDPSRAPIInvitation::get_Password


## -description


The password string.

This property is read-only.


## -parameters


## -remarks



The password is set when calling the <a href="https://msdn.microsoft.com/169d220b-3a2a-490e-9c1c-03a707d59f6c">IRDPSRAPIInvitationManager::CreateInvitation</a> method.
