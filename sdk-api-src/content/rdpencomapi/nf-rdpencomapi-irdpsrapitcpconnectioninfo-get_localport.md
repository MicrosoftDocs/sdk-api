# IRDPSRAPITcpConnectionInfo::get_LocalPort


## -description


Retrieves the local port that is being used by the sharer or by the viewer in communication. For example, the sharer can retrieve its own port that is being used to communicate with the viewer and vice versa.

This property is read-only.
