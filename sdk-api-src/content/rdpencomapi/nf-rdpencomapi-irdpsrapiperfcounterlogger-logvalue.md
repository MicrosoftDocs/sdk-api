# IRDPSRAPIPerfCounterLogger::LogValue


## -description

Logs a value.

## -parameters

### -param lValue [in]

The value to log.

## -returns

If the method succeeds, the return value is <b>S_OK</b>. Otherwise, the return value is an error code.
