# IRDPSRAPIAttendee::get_Flags


## -description

Retrieves the attendee specific flags that are defined in the <a href="/windows/win32/api/rdpencomapi/ne-rdpencomapi-rdpencomapi_attendee_flags">RDPENCOMAPI_ATTENDEE_FLAGS</a> enumeration type.

This property is read-only.
