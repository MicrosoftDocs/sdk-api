# ITocParser::RemoveTocByIndex


## -description


The <b>RemoveTocByIndex</b> method removes a table of contents, specified by an index, from the TOC Parser object.


## -parameters




### -param arg1 [in]

A member of the <a href="https://docs.microsoft.com/windows/desktop/api/wmcodecdsp/ne-wmcodecdsp-toc_pos_type">TOC_POS_TYPE</a> enumeration that specifies the <a href="https://docs.microsoft.com/windows/desktop/medfound/the-position-type-of-a-table-of-contents">position type</a> of the table of contents to be removed.


### -param dwTocIndex [in]

The index of the table of contents to be removed.


## -returns



The method returns an <b>HRESULT</b>. Possible values include, but are not limited to, those in the following table.
