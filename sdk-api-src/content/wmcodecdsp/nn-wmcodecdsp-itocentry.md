# ITocEntry interface


## -description

The <b>ITocEntry</b> interface represents an individual entry in a table of contents. It provides methods for setting and retrieving descriptive information for the entry.

## -inheritance

The <b xmlns:loc="http://microsoft.com/wdcml/l10n">ITocEntry</b> interface inherits from the <a href="/windows/desktop/api/unknwn/nn-unknwn-iunknown">IUnknown</a> interface. <b>ITocEntry</b> also has these types of members:
<ul>
<li><a href="https://docs.microsoft.com/">Methods</a></li>
</ul>
