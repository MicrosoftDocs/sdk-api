# ITfLangBarItemMgr::GetItemNum


## -description

Obtains the number of items in the language bar.

## -parameters




### -param pulCount [out]

Pointer to a <b>ULONG</b> that receives the number of items in the language bar.


## -returns



This method can return one of these values.
