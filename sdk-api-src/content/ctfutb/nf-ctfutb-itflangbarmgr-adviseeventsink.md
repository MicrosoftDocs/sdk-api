# ITfLangBarMgr::AdviseEventSink


## -description


The <b>ITfLangBarMgr::AdviseEventSink</b> method advises a sink about a language bar event.


## -parameters




### -param pSink [in]

Sink object to advise about the event.


### -param hwnd [in]

Reserved; must be <b>NULL</b>.


### -param dwFlags [in]

Reserved; must be 0.


### -param pdwCookie [in]

Pointer to an identifier for the connection.

## -remarks



<i>pdwCookie</i> receives an identifier that should be passed to <a href="https://docs.microsoft.com/windows/desktop/api/ctfutb/nf-ctfutb-itflangbarmgr-unadviseeventsink">ITfLangBarMgr::UnadviseEventSink</a> when the event sink is no longer required.
