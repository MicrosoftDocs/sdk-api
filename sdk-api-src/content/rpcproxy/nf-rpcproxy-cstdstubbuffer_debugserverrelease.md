# CStdStubBuffer_DebugServerRelease function


## -description

<p class="CCE_Message">[CStdStubBuffer_DebugServerRelease is not supported and may be altered or unavailable in the future.]

The <b>CStdStubBuffer_DebugServerRelease</b> function implements the <a href="/windows/desktop/api/objidl/nf-objidl-irpcstubbuffer-debugserverrelease">IRpcStubBuffer::DebugServerRelease</a> method.

## -parameters

### -param This [in]

Pointer to the <b>IRpcStubBuffer</b> object.

### -param pv [in]

Pointer to the interface to release. The interface pointer comes from a call to the <a href="/windows/desktop/api/rpcproxy/nf-rpcproxy-cstdstubbuffer_debugserverqueryinterface">CStdStubBuffer_DebugServerQueryInterface</a> method.

## -remarks

This function is used internally by proxies that are generated by MIDL.
