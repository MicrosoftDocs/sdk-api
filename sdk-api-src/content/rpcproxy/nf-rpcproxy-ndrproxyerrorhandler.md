# NdrProxyErrorHandler function


## -description


The <b>NdrProxyErrorHandler</b> function maps an exception into an HRESULT, with RPC facility code.


## -parameters




### -param dwExceptionCode [in]

Exception code to be mapped.


## -returns



Returns the HRESULT.
