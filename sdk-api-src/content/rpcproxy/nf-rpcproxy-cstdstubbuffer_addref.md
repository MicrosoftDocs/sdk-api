# CStdStubBuffer_AddRef function


## -description


<p class="CCE_Message">[CStdStubBuffer_AddRef is not supported and may be altered or unavailable in the future.]

The <b>CStdStubBuffer_AddRef</b> function  implements the <a href="_com_irpcstubbuffer">IRpcStubBuffer::AddRef</a> method.


## -parameters




### -param This [in]

Pointer to  the <a href="_com_irpcstubbuffer">IRpcStubBuffer</a> object. 


## -returns



Returns an integer from 1 to <i>n</i>, indicating the value of the new reference count.




## -remarks



 The return value is for diagnostic and testing purposes only; the return value may be unstable.

This function is used internally by proxies that are generated by MIDL.
