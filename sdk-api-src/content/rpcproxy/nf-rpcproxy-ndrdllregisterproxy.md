# NdrDllRegisterProxy function


## -description


The <b>NdrDllRegisterProxy</b> function creates a registry entry for the interfaces contained in the proxy DLL.


## -parameters




### -param hDll [in]

Handle to the proxy DLL.


### -param pProxyFileList [in]

Pointer to a <a href="https://msdn.microsoft.com/dbe797da-3ec3-4fe0-83bf-30669127a401">ProxyFileInfo</a> structure generated by MIDL.


### -param pclsid [in]

Pointer to the class identifier of the proxy being registered.


## -returns



Returns S_OK on success.
