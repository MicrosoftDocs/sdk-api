# IITDatabase interface


## -description

Use this interface for opening and closing the database object, and for instantiating objects stored in the database.

## -inheritance

The <b xmlns:loc="http://microsoft.com/wdcml/l10n">IITDatabase</b> interface inherits from the <a href="/windows/desktop/api/unknwn/nn-unknwn-iunknown">IUnknown</a> interface. <b>IITDatabase</b> also has these types of members:

</ul>
