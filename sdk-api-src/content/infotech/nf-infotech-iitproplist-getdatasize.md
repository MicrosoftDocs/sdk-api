# IITPropList::GetDataSize


## -description


Returns the number of bytes needed to save the property data. 


## -parameters




### -param lpvHeader [in]

Pointer to a buffer containing the header. 


### -param dwHdrSize [in]

Size of the header buffer. 


### -param dwDataSize [out, ref]

Size in bytes. 


## -returns
