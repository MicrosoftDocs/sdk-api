# IUPnPRemoteEndpointInfo::GetGuidValue


## -description


The <b>GetGuidValue</b> method currently is not supported.


## -parameters




### -param bstrValueName [in]

Not supported.


### -param pguidValue [out]

Not supported.


## -returns



This method returns E_INVALIDARG, a COM error code defined in WinError.h.
