# IntPtrToShort function


## -description

Converts a value of type <b>INT_PTR</b> to a value of type <b>SHORT</b>.

## -parameters

### -param iOperand [in]

The value to convert.

### -param psResult [out]

The converted value.

## -returns

If this function succeeds, it returns <b>S_OK</b>. Otherwise, it returns an <b>HRESULT</b> error code.
