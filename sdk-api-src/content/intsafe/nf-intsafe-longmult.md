# LongMult function


## -description

Multiplies two values of type <b>LONG</b>.

## -parameters

### -param lMultiplicand [in]

The first value.

### -param lMultiplier [in]

The second value.

### -param plResult [out]

The result.

## -returns

If this function succeeds, it returns <b>S_OK</b>. Otherwise, it returns an <b>HRESULT</b> error code.
