# ByteToInt8 function


## -description

Converts a value of type <b>BYTE</b> to a value of type <b>INT8</b>.

## -parameters

### -param bOperand [in]

The value to convert.

### -param pi8Result [out]

The converted value.

## -returns

If this function succeeds, it returns <b>S_OK</b>. Otherwise, it returns an <b>HRESULT</b> error code.
