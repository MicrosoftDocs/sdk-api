# UInt8Add function


## -description

Adds two values of type <b>UINT8</b>.

## -parameters

### -param u8Augend [in]

The first value.

### -param u8Addend [in]

The second value.

### -param pu8Result [out]

The result.

## -returns

If this function succeeds, it returns <b>S_OK</b>. Otherwise, it returns an <b>HRESULT</b> error code.
