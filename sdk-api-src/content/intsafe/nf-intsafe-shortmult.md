# ShortMult function


## -description

Multiplies two values of type <b>SHORT</b>.

## -parameters

### -param sMultiplicand [in]

The first value.

### -param sMultiplier [in]

The second value.

### -param psResult [out]

The result.

## -returns

If this function succeeds, it returns <b>S_OK</b>. Otherwise, it returns an <b>HRESULT</b> error code.
