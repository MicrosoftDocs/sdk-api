# IntPtrAdd function


## -description

Adds two values of type <b>INT_PTR</b>.

## -parameters

### -param iAugend [in]

The first value.

### -param iAddend [in]

The second value.

### -param piResult [out]

The result.

## -returns

If this function succeeds, it returns <b>S_OK</b>. Otherwise, it returns an <b>HRESULT</b> error code.
