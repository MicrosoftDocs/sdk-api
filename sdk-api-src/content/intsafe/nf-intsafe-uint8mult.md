# UInt8Mult function


## -description

Multiplies two values of type <b>UINT8</b>.

## -parameters

### -param u8Multiplicand [in]

The first value.

### -param u8Multiplier [in]

The second value.

### -param pu8Result [out]

The result.

## -returns

If this function succeeds, it returns <b>S_OK</b>. Otherwise, it returns an <b>HRESULT</b> error code.
