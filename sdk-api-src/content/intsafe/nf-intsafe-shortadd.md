# ShortAdd function


## -description

Adds two values of type <b>SHORT</b>.

## -parameters

### -param sAugend [in]

The first value.

### -param sAddend [in]

The second value.

### -param psResult [out]

The result.

## -returns

If this function succeeds, it returns <b>S_OK</b>. Otherwise, it returns an <b>HRESULT</b> error code.
