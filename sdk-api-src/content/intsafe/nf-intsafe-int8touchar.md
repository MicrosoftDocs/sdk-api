# Int8ToUChar function


## -description

Converts a value of type <b>INT8</b> to a value of type <b>UCHAR</b>.

## -parameters

### -param i8Operand [in]

The value to convert.

### -param pch [out]

The converted value.

## -returns

If this function succeeds, it returns <b>S_OK</b>. Otherwise, it returns an <b>HRESULT</b> error code.
