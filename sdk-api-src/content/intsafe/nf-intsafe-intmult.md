# IntMult function


## -description

Multiplies two values of type <b>INT</b>.

## -parameters

### -param iMultiplicand [in]

The first value.

### -param iMultiplier [in]

The second value.

### -param piResult [out]

The result.

## -returns

If this function succeeds, it returns <b>S_OK</b>. Otherwise, it returns an <b>HRESULT</b> error code.
