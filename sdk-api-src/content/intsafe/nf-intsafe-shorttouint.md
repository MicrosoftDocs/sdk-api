# ShortToUInt function


## -description

Converts a value of type <b>SHORT</b> to a value of type <b>UINT</b>.

## -parameters

### -param sOperand [in]

The value to convert.

### -param puResult [out]

The converted value.

## -returns

If this function succeeds, it returns <b>S_OK</b>. Otherwise, it returns an <b>HRESULT</b> error code.
