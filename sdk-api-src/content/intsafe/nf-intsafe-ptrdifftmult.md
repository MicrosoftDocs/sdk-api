# PtrdiffTMult function


## -description

Multiplies two values of type <b>ptrdiff_t</b>.

## -parameters

### -param Multiplicand [in]

The first value.

### -param Multiplier [in]

The second value.

### -param pResult [out]

The result.

## -returns

If this function succeeds, it returns <b>S_OK</b>. Otherwise, it returns an <b>HRESULT</b> error code.
