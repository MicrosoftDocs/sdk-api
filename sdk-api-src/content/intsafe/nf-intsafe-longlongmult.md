# LongLongMult function


## -description

Multiplies two values of type <b>LONGLONG</b>.

## -parameters

### -param llMultiplicand [in]

The first value.

### -param llMultiplier [in]

The second value.

### -param pllResult [out]

The result.

## -returns

If this function succeeds, it returns <b>S_OK</b>. Otherwise, it returns an <b>HRESULT</b> error code.
