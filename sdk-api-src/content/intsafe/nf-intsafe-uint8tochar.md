# UInt8ToChar function


## -description

Converts a value of type <b>UINT8</b> to a value of type <b>CHAR</b>.

## -parameters

### -param u8Operand [in]

The value to convert.

### -param pch [out]

The converted value.

## -returns

If this function succeeds, it returns <b>S_OK</b>. Otherwise, it returns an <b>HRESULT</b> error code.
