# IKsNodeControl::put_NodeId


## -description



Sets the node identifier for the extension unit.




## -parameters




### -param dwNodeId [in]

Node identifier.


## -returns



If the method succeeds, it returns S_OK. If it fails, it returns an <b>HRESULT</b> error code.
