# IMixerOCX::GetStatus


## -description

The <code>GetStatus</code> method returns the current status of the Overlay Mixer. (Not implemented.)

## -parameters

### -param pdwStatus [out]

Pointer that receives the current status of the Overlay Mixer.

## -returns

Returns E_NOTIMPL.
