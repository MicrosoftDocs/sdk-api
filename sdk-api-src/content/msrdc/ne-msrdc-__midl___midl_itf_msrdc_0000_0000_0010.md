# __MIDL___MIDL_itf_msrdc_0000_0000_0010 enumeration


## -description


Defines the access mode values for RDC file mapping objects.


## -enum-fields




### -field RDCMAPPING_Undefined

The mapping access mode is unknown.


### -field RDCMAPPING_ReadOnly

Specifies read-only access.


### -field RDCMAPPING_ReadWrite

Specifies read/write access.
