# __MIDL___MIDL_itf_msrdc_0000_0000_0003 enumeration


## -description


Defines the set of 
    data chunks used to generate a remote copy.


## -enum-fields




### -field RDCNEED_SOURCE

The chunk is a source chunk.


### -field RDCNEED_TARGET

This value is reserved for future use.


### -field RDCNEED_SEED

The chunk is a seed chunk.


### -field RDCNEED_SEED_MAX

This value is reserved for future use.
