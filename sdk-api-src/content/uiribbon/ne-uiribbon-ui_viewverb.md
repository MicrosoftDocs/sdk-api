# UI_VIEWVERB enumeration


## -description

Specifies values that identify the type of action to complete on a Ribbon framework View.

## -enum-fields

### -field UI_VIEWVERB_CREATE:0

Create a View.

### -field UI_VIEWVERB_DESTROY:1

Destroy a View.

### -field UI_VIEWVERB_SIZE:2

Resize a View.

### -field UI_VIEWVERB_ERROR:3

Unable to complete the action.
