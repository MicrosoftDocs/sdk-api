# IUIImage interface


## -description


The <b>IUIImage</b> interface is implemented by the application and defines the method 
		for retrieving an image to display in the ribbon and context popup UI of the Windows Ribbon framework .


## -inheritance

The <b xmlns:loc="http://microsoft.com/wdcml/l10n">IUIImage</b> interface inherits from the <a href="https://msdn.microsoft.com/33f1d79a-33fc-4ce5-a372-e08bda378332">IUnknown</a> interface. <b>IUIImage</b> also has these types of members:
