# ITAgentHandlerEvent::get_Event


## -description

The 
<b>get_Event</b> method gets the description for the event that has occurred.

## -parameters

Pointer to the 
<a href="/windows/desktop/api/tapi3/ne-tapi3-agenthandler_event">AGENTHANDLER_EVENT</a> descriptor.
