# ITAgentSession::get_AverageTalkTime


## -description


The 
<b>get_AverageTalkTime</b> method gets the average time (in seconds) spent talking per ACD call, during this agent session (by this agent).


## -parameters




### -param plTalkTime [out]

Average talk time.
