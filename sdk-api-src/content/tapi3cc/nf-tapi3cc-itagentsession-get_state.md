# ITAgentSession::get_State


## -description


The 
<b>get_State</b> method gets the current state of this session.

Pointer to an 
<a href="https://docs.microsoft.com/windows/desktop/api/tapi3/ne-tapi3-agent_session_state">AGENT_SESSION_STATE</a>.
