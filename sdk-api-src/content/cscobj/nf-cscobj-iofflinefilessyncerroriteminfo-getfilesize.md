# IOfflineFilesSyncErrorItemInfo::GetFileSize


## -description


Retrieves the size of the item in bytes.


## -parameters




### -param pSize [out]

Receives the item's size in bytes.


## -returns



Returns <b>S_OK</b> if successful, or an error value otherwise.
