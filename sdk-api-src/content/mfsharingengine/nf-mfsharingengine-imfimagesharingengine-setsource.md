# IMFImageSharingEngine::SetSource


## -description

Sets the source stream.

## -parameters

### -param pStream

The source stream.

## -returns

If this method succeeds, it returns <b>S_OK</b>. Otherwise, it returns an <b>HRESULT</b> error code.
