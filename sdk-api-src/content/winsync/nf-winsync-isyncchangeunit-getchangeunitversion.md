# ISyncChangeUnit::GetChangeUnitVersion


## -description


Gets the version for the change unit change.


## -parameters




### -param pbCurrentReplicaId [in]

The ID of the replica that originated the change to retrieve.


### -param pVersion [out]

Returns the version of the change.
