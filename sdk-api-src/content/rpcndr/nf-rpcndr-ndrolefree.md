# NdrOleFree function


## -description

The <b>NdrOleFree</b> function is a wrapper for the <a href="/windows/desktop/api/combaseapi/nf-combaseapi-cotaskmemfree">CoTaskMemFree</a> function.

## -parameters

### -param NodeToFree [in]

Pointer to the memory to be freed.
